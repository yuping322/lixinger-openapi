接口: macro_china_bond_public

目标地址: https://www.chinamoney.com.cn/chinese/xzjfx/

描述: 中国外汇交易中心暨全国银行间同业拆借中心-债券信息披露-新债发行; 近期债券发行数据

限量: 单次返回所有历史数据

| 名称  | 类型  | 描述  |
|-----|-----|-----|
| -   | -   | -   |

|-------|---------|----------|
| 债券全称  | object  | -        |
| 债券类型  | object  | -        |
| 发行日期  | object  | -        |
| 计息方式  | object  | -        |
| 价格    | float64 | 注意单位: 元  |
| 债券期限  | object  | -        |
| 计划发行量 | float64 | 注意单位: 亿元 |
| 债券评级  | object  | -        |

```python
import akshare as ak
```
