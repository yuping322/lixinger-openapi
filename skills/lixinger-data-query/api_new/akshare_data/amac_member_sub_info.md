目标地址: https://gs.amac.org.cn/amac-infodisc/res/pof/member/index.html?primaryInvestType=private

限量: 单次返回当前时刻所有历史数据

| 名称 | 类型 | 描述 |
|----|----|----|
| -  | -  | -  |

|----------|--------|----|
| 机构（会员）名称 | object | -  |
| 会员代表     | object | -  |
| 会员类型     | object | -  |
| 会员编号     | object | -  |
| 入会时间     | object | -  |
| 公司类型     | object | -  |

import akshare as ak
