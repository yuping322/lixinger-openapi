接口: index_component_sw

目标地址: https://www.swsresearch.com//institute_sw/allIndex/releasedIndex/releasedetail?code=801001&name=申万中小

限量: 该接口返回指定 symbol 的数据

|--------|-----|--------------------------------------------------|
| symbol | str | symbol="801001"; 指数代码                            |

|------|---------|-----|
| 序号   | int64   | -   |

| 证券名称 | object  | -   |

| 计入日期 | object  | -   |

```python
import akshare as ak
```
