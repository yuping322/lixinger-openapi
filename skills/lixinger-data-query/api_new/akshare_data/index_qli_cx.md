接口: index_qli_cx

目标地址: https://yun.ccxe.com.cn/indices/qli

描述: 财新指数-高质量因子

限量: 该接口返回所有历史数据

输入参数

| 名称  | 类型  | 描述  |
|-----|-----|-----|
| -   | -   | -   |

输出参数

| 名称      | 类型      | 描述      |
|---------|---------|---------|
| 日期      | object  | -       |
| 高质量因子指数 | float64 | -       |
| 变化幅度    | float64 | 注意单位: % |

接口示例

```python
import akshare as ak

index_qli_cx_df = ak.index_qli_cx()
print(index_qli_cx_df)
```
