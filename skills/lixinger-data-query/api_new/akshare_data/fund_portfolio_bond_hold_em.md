限量: 单次返回指定 symbol 和 date 的所有持仓数据

输入参数

| 名称     | 类型  | 描述                                                       |
|--------|-----|----------------------------------------------------------|
| symbol | str | symbol="000001"; 基金代码, 可以通过调用 **ak.fund_name_em()** 接口获取 |
| date   | str | date="2023"; 指定年份                                        |

输出参数

| 名称    | 类型      | 描述       |
|-------|---------|----------|
| 序号    | int64   | -        |

| 债券名称  | object  | -        |
| 占净值比例 | float64 | 注意单位: %  |
| 持仓市值  | float64 | 注意单位: 万元 |
| 季度    | object  | -        |

接口示例

```python
import akshare as ak
```
