## 交易类型分类

### 纳入（主动增持行为）

| 类型 | 说明 | 信号强度 |
|------|------|---------|
| 二级市场竞价买入 | 通过交易所集合竞价/连续竞价直接买入 | 最强 |
| 大宗交易买入 | 通过大宗交易平台买入 | 强 |
| 集中竞价+大宗交易混合 | 同时使用两种方式 | 强 |

### 排除（非主动行为）

| 类型 | 说明 | 排除原因 |
|------|------|---------|
| 股权激励行权 | 行使股票期权 | 薪酬安排，非主动决策 |
| 限制性股票解锁 | 限制性股票归属 | 预设计划 |
| 定向增发认购 | 参与定增 | 折价获取，信号混杂 |
| 协议转让 | 非公开协议方式 | 可能涉及控制权交易 |
| 可转债转股 | 转债转换为股票 | 机械操作 |
| 司法拍卖/过户 | 非自愿交易 | 被动行为 |
| 继承/赠与 | 非交易性质 | 非投资决策 |

### 特殊情况

- **增持计划公告后实施**：关注实际增持进度是否符合计划。仅公告计划而未实施的不纳入统计
- **回购注销 vs 回购用于激励**：回购并注销是更强的信号（减少流通股本）；回购用于未来股权激励的信号较弱
- **承诺不减持**：控股股东/实控人发布不减持承诺有一定信号价值，但弱于真金白银增持

## 集群增持识别

### 定义

集群增持 = 时间窗口内 ≥ 2名不同内部人主动增持。

### 信号分级

| 级别 | 条件 | 信号强度 |
|------|------|---------|
| 极强 | ≥ 4人增持，含董事长或总经理 | 非常强 |
| 强 | 3人增持，至少1名核心高管 | 强 |
| 中等 | 2人增持 | 中等 |
| 弱 | 仅1人增持（仅供参考） | 弱 |

### 职务权重

| 职务 | 权重 | 原因 |
|------|------|------|
| 董事长/实际控制人 | 5x | 战略全局视野，信息优势最大 |
| 总经理/总裁 | 4x | 经营层最高负责人 |
| 财务总监/CFO | 4x | 最了解财务状况 |
| 副总经理/副总裁 | 3x | 分管业务知情 |
| 董事会秘书 | 2x | 了解公司信息披露全貌 |
| 非独立董事 | 2x | 参与重大决策 |
| 独立董事 | 1.5x | 有一定信息接触但有限 |
| 监事 | 1x | 监督角色，信息优势有限 |
| 持股5%以上股东 | 1x | 可能有非基本面动机 |

### 时间聚集度

时间越集中，信号越强：

- 所有增持在2周内完成：**强聚集** — 可能存在共同的信息触发
- 增持分散在30–60天：**中等聚集** — 持续但不紧迫
- 增持分散在60–90天：**弱聚集** — 可能是巧合

## 显著性评估

### 相对于年薪

| 增持金额 / 年薪 | 评估 |
|----------------|------|
| > 200% | 极高信心 — 个人重大财务决策 |
| 100–200% | 非常显著 |
| 50–100% | 显著 |
| 20–50% | 中等 |
| < 20% | 可能象征性增持 |

### 相对于现有持股

| 增持量 / 现有持股 | 评估 |
|------------------|------|
| > 50% | 大幅加仓，极高信心 |
| 25–50% | 显著加仓 |
| 10–25% | 有意义的增加 |
| < 10% | 小额添加 |

### 绝对金额门槛

| 金额 | 评估 |
|------|------|
| > 1000万元 | 任何情况下都具有实质意义 |
| 500–1000万元 | 对多数高管有意义 |
| 100–500万元 | 对中层管理人员有意义 |
| < 100万元 | 仅在薪酬也较低时有意义 |

## 信号强度评分

### 综合评分公式

```
信号分 = 集群级别 × 职务权重 × 显著性 × 情境系数
```

### 情境系数

| 情境 | 系数 | 原因 |
|------|------|------|
| 股价处于近一年低位 / 跌幅超30% | 1.5x | 逆势增持，信心极强 |
| 行业整体下跌期间增持 | 1.3x | 排除行业因素，个股信心 |
| 业绩预告/快报前增持 | 1.2x | 可能预示业绩向好（注意窗口期合规） |
| 股价上涨趋势中增持 | 0.8x | 可能追涨行为 |
| 业绩公告后增持 | 0.7x | 信息已公开 |
| 新任高管上任后立即增持 | 0.6x | 可能是就职信号/形式行为 |
| 监管"鼓励"增持的大环境下 | 0.5x | 可能非自愿/政策驱动 |

## 风险信号与伪信号

### 信号可能被稀释的情况

- **政策性增持**：市场大幅下跌后，监管层"窗口指导"鼓励大股东增持。此类增持往往金额较小、执行不到位
- **维护质押线**：控股股东股权质押比例较高时，增持可能是为了阻止股价跌破质押平仓线，而非看好前景
- **增持承诺到期**：此前承诺的增持计划到期前集中实施，属于履约行为
- **统一行动**：多名高管同日同金额增持，可能是公司统一安排，而非独立判断
- **增持后减持**：关注增持后6个月内是否存在减持行为（A股规定增持后6个月内不得减持）

### 反向信号

- **同期有其他内部人减持**：部分人增持但另一部分人减持，信号模糊
- **增持金额远低于计划上限**：增持计划承诺1–2亿，实际仅增持几百万，说明信心有限
- **增持后发布减持计划**：增持完成后不久即公告减持计划，否定前期信号

### A股特有风险

- **股权质押风险**：查询控股股东质押比例。质押比例 > 50% 时，增持动机存疑
- **关联交易**：公司存在大量关联交易时，增持可能配合利益输送
- **ST/\*ST公司**：ST公司大股东增持可能为避免退市，而非价值投资
- **限售股解禁在即**：大股东增持可能为解禁期维稳股价，解禁后可能大幅减持

## 数据来源

### 主要数据源

- **巨潮资讯网** (cninfo.com.cn) — 上市公司公告原文（最权威）
- **上海证券交易所** (sse.com.cn) — 沪市公司公告
- **深圳证券交易所** (szse.cn) — 深市公司公告
- **东方财富** (eastmoney.com) — 数据汇总整理
- **同花顺 iFinD** — 专业数据终端
- **Wind万得** — 机构级数据终端

### 关键披露规则

- **增持前**：董监高本人及配偶、父母、子女增减持须提前披露（大股东增持需在首次增持后1个交易日内公告）
- **增持后**：变动达到1%时需公告，2个交易日内披露
- **窗口期限制**：定期报告公告前30日、业绩预告/快报前10日、重大事项敏感期内不得交易
- **短线交易**：董监高及5%以上股东6个月内反向交易收益归公司所有
- **减持规则**：大股东通过竞价减持每3个月不超过1%，大宗交易每3个月不超过2%
