# 方法论：涨停板分析器（A股）

本技能用于**分析涨停板个股的特征、封单强度、开板风险、连板逻辑，用于识别强势股和短线机会**。涨停板是A股市场的特色，是短线交易的核心关注点，也是市场情绪和资金偏好的直接体现。

## 数据口径

### 数据源与字段映射

涨停板分析需要以下数据：
- **涨停板数据**：AKShare (`stock_zt_pool_em`), 东方财富
  - 涨停时间、涨停价、最新价、涨停原因、封单量、封单金额
  - 实时数据，盘中更新
- **连板统计**：AKShare (`stock_zt_pool_previous_em`)
  - 连板天数、首次涨停日期、连板期间涨幅
  - 日频数据
- **开板记录**：AKShare (`stock_zt_pool_dtgc_em`)
  - 开板次数、开板时间、开板原因
  - 实时数据
- **市场数据**：AKShare (`stock_zh_a_hist`)
  - 个股价格、成交量、换手率、流通市值
  - 实时/日频数据

关键字段：
- `涨停时间`：首次涨停的时间（如09:30、10:15）
- `封单量`：涨停价位的买单数量（万手）
- `封单金额`：封单量 * 涨停价（亿元）
- `封单强度`：封单金额 / 流通市值（%）
- `连板天数`：连续涨停的天数（1板、2板、3板...）
- `涨停原因`：首板、板块联动、游资、机构、消息刺激等
- `开板次数`：当日打开涨停板的次数

### 时间窗口与频率

- **盘中实时监控**：9:30-15:00实时监控涨停板动态
- **日频统计**：每日收盘后统计涨停板数量、连板分布、成功率
- **周频分析**：每周分析涨停板特征、市场情绪、强势板块

建议窗口：
- 涨停时间：早盘（9:30-10:00）、午盘（13:00-14:00）、尾盘（14:30-15:00）
- 连板周期：1-5板（短线）、6-10板（中线）、10板以上（妖股）
- 封单强度：实时监控、收盘确认

## 核心指标

### 指标列表与计算公式

1. **涨停强度指标**：
   - `封单强度 = 封单金额 / 流通市值`（%）
   - `封单比 = 封单量 / 流通股本`（%）
   - `涨停速度 = 涨停时间 - 开盘时间`（分钟）
   - `封板时长 = 收盘时间 - 涨停时间`（分钟）

2. **连板指标**：
   - `连板天数 = 连续涨停天数`
   - `连板期间涨幅 = (当前价 - 首板前收盘价) / 首板前收盘价`（%）
   - `连板加速度 = (最近3板涨幅 - 前3板涨幅) / 前3板涨幅`
   - `连板成功率 = 次日继续涨停次数 / 总涨停次数`（%）

3. **开板风险指标**：
   - `开板次数 = 当日打开涨停板次数`
   - `开板时长 = Σ(开板持续时间)`（分钟）
   - `开板幅度 = (涨停价 - 最低价) / 涨停价`（%）
   - `封板成功率 = 最终封板次数 / 总涨停次数`（%）

4. **市场情绪指标**：
   - `涨停家数 = 当日涨停股票数量`
   - `连板家数 = 当日2板及以上股票数量`
   - `炸板率 = 开板股票数量 / 涨停股票数量`（%）
   - `涨停板块集中度 = Top3板块涨停数 / 总涨停数`（%）

5. **资金偏好指标**：
   - `平均市值 = mean(涨停股流通市值)`
   - `小盘股占比 = 流通市值 < 50亿的涨停股数量 / 总涨停数`（%）
   - `ST股占比 = ST股涨停数量 / 总涨停数`（%）
   - `科创板占比 = 科创板涨停数量 / 总涨停数`（%）

### 标准化方法

- 使用**封单强度**标准化不同市值个股的封单力度
- 使用**涨停速度**识别强势股（越早涨停越强）
- 使用**连板天数**分类短线、中线、妖股
- 使用**涨停家数**判断市场情绪（> 100家 = 情绪高涨）

## 信号与阈值

### 可测试规则（Insight Rules）

Rule 1 (早盘秒板 + 强封单 → 次日高开):
IF {涨停时间 <= 9:35 AND 封单强度 >= 10% AND 开板次数 = 0 AND 涨停原因非"ST摘帽"}
THEN {次日高开概率 >= 70%，预期高开幅度 >= 3%；若次日继续涨停，连板成功率 >= 60%}
CONFIDENCE {0.68; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，流通市值 >= 20亿的个股信号更可靠}
FAILURE_MODE {次日大盘大跌（系统性风险）；个股利空（如业绩暴雷）；监管介入（如异常交易核查）；封单虚假（尾盘撤单）；集合竞价大量抛盘}

Rule 2 (连板加速 + 板块共振 → 继续连板):
IF {连板天数 >= 2 AND 连板加速度 > 0 AND 同板块涨停数 >= 3 AND 封单强度 >= 5%}
THEN {次日继续涨停概率 >= 55%；若开板，当日回封概率 >= 40%}
CONFIDENCE {0.58; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，题材股、概念股信号更明显}
FAILURE_MODE {板块分歧（龙头开板）；资金接力失败（无新游资）；题材证伪（利好落空）；监管打压（如问询函）；连板过高（> 10板）风险极大}

Rule 3 (多次开板 + 封板成功 → 分歧转一致):
IF {开板次数 >= 3 AND 最终封板成功 AND 封单强度 >= 3% AND 尾盘封板（14:30后）}
THEN {次日低开高走概率 >= 60%，预期日内涨幅 >= 5%；表明多空分歧后资金重新达成一致}
CONFIDENCE {0.62; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，2-3板个股信号更强}
FAILURE_MODE {次日集合竞价大量抛盘；开盘后继续分歧（再次开板）；封单虚假（尾盘封板但封单弱）；大盘走弱}

Rule 4 (涨停家数暴增 + 炸板率低 → 市场情绪高涨):
IF {涨停家数 >= 100 AND 炸板率 <= 20% AND 连板家数 >= 20}
THEN {未来1-3个交易日，市场情绪持续高涨，涨停板机会多；但需警惕情绪过热后的快速降温}
CONFIDENCE {0.65; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场整体，牛市或反弹行情中信号更强}
FAILURE_MODE {情绪过热后快速降温（次日涨停家数骤降）；监管降温（如窗口指导）；外部冲击（如美股暴跌）；涨停板集中在垃圾股（质量差）}

### 触发/解除/无效化条件

- **触发监控**：
  - 个股涨停（任何时间）
  - 涨停时间 <= 9:35（秒板）
  - 连板天数 >= 2（连板股）
  - 涨停家数 >= 100（情绪高涨）

- **解除监控**：
  - 涨停打开且未回封（炸板）
  - 次日低开 >= 5%（资金撤退）
  - 连板中断（开板未回封）
  - 涨停家数 < 30（情绪降温持续3天）

- **无效化**：
  - 个股停牌（无法交易）
  - ST股或退市股（风险股）
  - 数据缺失或延迟 > 5分钟（盘中）
  - 市场异常（熔断、系统故障）

### 阈值选择依据

- **9:35涨停**：开盘5分钟内涨停，是强势的标志（秒板）
- **10%封单强度**：封单占流通市值10%以上，是强封单
- **3次开板**：多次开板后回封，表明多空分歧后达成一致
- **100家涨停**：涨停家数超过100，表明市场情绪高涨（总共约5000只股票）
- **20%炸板率**：炸板率低于20%，表明封板成功率高

## 边界条件与降级策略

### 数据缺失/异常值处理

- **数据缺失**：
  - 封单数据缺失：使用成交量和换手率推算
  - 涨停时间缺失：标注"时间未知"
  - 连板数据缺失：手动计算历史涨停天数

- **异常值处理**：
  - 封单强度 > 50%：检查是否流通市值错误，若确认则标注"超强封单"
  - 涨停时间 < 9:25：检查是否集合竞价涨停，若确认则标注"集合竞价涨停"
  - 开板次数 > 10：检查是否数据错误，若确认则标注"频繁开板"

### 降级策略

- **AKShare接口不稳定**：
  - 备用数据源1：东方财富网页爬取
  - 备用数据源2：同花顺涨停板
  - 备用数据源3：手动监控（盘中软件）
  - 降级标注："使用备用数据源"

- **实时数据延迟**：
  - 使用延迟数据（标注延迟时间）
  - 使用快照数据（每5分钟更新）
  - 降级标注："数据延迟X分钟"

- **历史数据不足**（< 20天）：
  - 使用可用历史计算成功率
  - 使用市场平均成功率代替
  - 降级标注："历史数据不足"

## A股特殊注意

### 1. T+1交易制度影响

- **当日涨停次日才能卖出**：涨停板买入后被锁定：
  - 次日高开可获利（但可能无法卖出）
  - 次日低开或跌停被套（无法止损）
  - 需要判断次日走势（高开/低开/继续涨停）

### 2. 涨跌停限制

- **主板±10%**：涨停价 = 前收盘价 * 1.10
- **科创板/创业板±20%**：涨停价 = 前收盘价 * 1.20
- **ST股±5%**：涨停价 = 前收盘价 * 1.05
- **新股首日无涨跌停**：上市首日可能涨幅超过100%

### 3. 涨停板交易规则

- **涨停后仍可买入**：只要有卖单，就可以在涨停价买入
- **涨停后无法卖出**：涨停价位只有买单，没有卖单（除非开板）
- **排队买入**：涨停价位买单按时间优先原则排队
- **撤单规则**：14:57后无法撤单（尾盘封单更可靠）

### 4. 涨停原因分类

A股涨停原因包括：
- **首板**：首次涨停，可能是题材启动、消息刺激、板块联动
- **连板**：连续涨停，2板、3板、4板...
- **反包板**：前日跌停或大跌，次日涨停（V型反转）
- **ST摘帽**：ST股摘帽当日涨停（风险较高）
- **重组复牌**：重组复牌后连续涨停（需关注重组进展）

### 5. 封单真实性

- **真封单**：封单量大（> 10%流通盘）、封单持续（> 2小时）、尾盘不撤单
- **假封单**：封单量小（< 3%流通盘）、频繁撤单、尾盘撤单
- **对倒封单**：同一资金在买卖盘对倒，制造虚假封单
- **需要识别**：真封单才有持续性，假封单次日容易低开

### 6. 开板类型

- **瞬间开板**：开板后立即回封（< 1分钟），表明封单强
- **短暂开板**：开板后5-10分钟回封，表明多空分歧
- **长时间开板**：开板后30分钟以上才回封或不回封，表明分歧大
- **尾盘开板**：14:30后开板，通常难以回封，次日容易低开

### 7. 连板逻辑

- **1板（首板）**：题材启动，龙头初现
- **2板（二板）**：确认强势，资金认可
- **3板（三板）**：加速上涨，情绪高涨
- **4-5板**：高位震荡，分歧加大
- **6-10板**：妖股行情，风险极高
- **10板以上**：极端行情，随时可能崩盘

### 8. 板块效应

- **板块联动**：同板块多只股票涨停，表明板块启动
- **龙头带动**：龙头涨停后，跟风股跟涨
- **板块轮动**：不同板块轮流涨停，表明资金轮动
- **需要识别**：龙头股（最先涨停、封单最强、市值最大）

### 9. 游资席位

- **知名游资**：如"中信证券杭州延安路"、"华泰证券深圳益田路"
- **游资风格**：短线操作、追涨杀跌、喜欢连板
- **游资接力**：多个游资轮流买入，推高股价
- **需要关注**：龙虎榜数据（次日晚间公布）

### 10. 监管风险

- **异常交易核查**：连续涨停、高换手率可能被核查
- **停牌核查**：交易所可能停牌核查（如"请说明涨停原因"）
- **问询函**：上市公司可能收到问询函（如"请说明与XX概念的关系"）
- **风险提示**：上市公司可能发布风险提示公告（如"股价严重偏离基本面"）

## 回测说明（最小化）

- **回测设计**：
  - 每日收盘后统计涨停板
  - 次日开盘买入（集合竞价或开盘价）
  - 持有期：1日（T+1卖出）
  - 基准：涨停前收盘价

- **性能指标**：
  - 次日高开概率、平均高开幅度
  - 次日收盘涨跌幅、胜率
  - 夏普比率、最大回撤
  - 分类统计（秒板 vs 尾盘板、首板 vs 连板）

- **证伪条件**：
  - 次日高开概率 <= 50%
  - 平均收盘涨跌幅 <= 0
  - 夏普比率 <= 0.5

- **敏感性分析**：
  - 测试不同涨停时间（秒板 vs 尾盘板）
  - 测试不同封单强度（强封 vs 弱封）
  - 测试不同连板天数（首板 vs 2板 vs 3板）
  - 测试不同市场环境（牛市 vs 熊市）

## 使用示例

### 示例1：早盘秒板

**输入**：2024-02-15 09:32，某股票涨停
**数据**：
- 涨停时间：09:32（开盘2分钟）
- 涨停价：10.00元
- 封单量：50万手
- 封单金额：5亿元
- 流通市值：40亿元
- 封单强度：12.5%
- 开板次数：0
- 涨停原因：AI概念启动

**分析**：
- 早盘秒板（09:32）
- 强封单（12.5%）
- 未开板（0次）
- 题材明确（AI概念）
- **结论**：强势股，次日高开概率高，可关注或买入（需承担T+1风险）

### 示例2：连板加速

**输入**：2024-03-20，某股票3板涨停
**数据**：
- 连板天数：3板
- 连板期间涨幅：+33%（1.1^3 ≈ 1.33）
- 连板加速度：+0.15（加速）
- 同板块涨停数：5只
- 封单强度：8%
- 涨停时间：10:15

**分析**：
- 3板连板（强势）
- 加速上涨（加速度+0.15）
- 板块共振（5只涨停）
- 封单较强（8%）
- **结论**：连板加速，板块共振，次日继续涨停概率较高，但风险增加

### 示例3：多次开板后回封

**输入**：2024-04-10，某股票2板涨停
**数据**：
- 连板天数：2板
- 开板次数：4次
- 开板时长：累计45分钟
- 最终封板时间：14:35（尾盘）
- 封单强度：5%
- 开板幅度：-3%（最低跌到涨停价以下3%）

**分析**：
- 2板连板
- 多次开板（4次）表明分歧大
- 尾盘回封（14:35）表明资金重新达成一致
- 封单中等（5%）
- **结论**：分歧转一致，次日可能低开高走，但风险较高

### 示例4：市场情绪高涨

**输入**：2024-05-15，市场整体
**数据**：
- 涨停家数：135只
- 连板家数：28只
- 炸板率：15%（20只炸板）
- 涨停板块集中度：45%（AI、半导体、新能源占比）
- 平均涨停时间：10:30

**分析**：
- 涨停家数多（135只）
- 连板家数多（28只）
- 炸板率低（15%）
- 板块集中（AI、半导体、新能源）
- **结论**：市场情绪高涨，涨停板机会多，但需警惕情绪过热后的降温
