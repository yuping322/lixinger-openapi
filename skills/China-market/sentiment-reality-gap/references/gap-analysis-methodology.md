## 情绪度量

### 量化指标

| 指标 | 数据来源 | 严重负面信号 |
|------|---------|------------|
| 股价6个月涨跌幅 | 行情数据 | 跌幅 ≥ 30% |
| 相对行业超额跌幅 | 行情数据 | 跑输行业 ≥ 15% |
| 分析师盈利预测变化 | Wind/iFinD一致预期 | 3个月内下调 ≥ 10% |
| 分析师评级变化 | 券商研报 | 2家以上下调评级 |
| 北向资金持仓变化 | 陆股通数据 | 3个月净流出且占流通股比例下降 |
| 融资余额变化 | 交易所数据 | 融资余额3个月下降 ≥ 20% |
| 公募基金持仓 | 基金季报 | 连续2季减持 |
| 大宗交易折价率 | 大宗交易数据 | 持续高折价（折价率 > 8%） |

### 定性指标

| 指标 | 评估方法 | 严重负面信号 |
|------|---------|------------|
| 主流财经媒体报道基调 | 人工评估/舆情监控 | 负面报道密集 |
| 雪球/股吧散户情绪 | 舆情分析 | 恐慌性言论主导 |
| 券商研报覆盖变化 | 研报数量跟踪 | 停止覆盖或研报数量骤减 |
| 行业会议/调研热度 | 机构调研数据 | 调研次数骤降 |

### 情绪时间线构建

按时间轴梳理负面事件序列：

```
[日期1] — [触发事件] — 股价反应：-X%
[日期2] — [后续发展] — 股价反应：-X%
[日期3] — [最新状态] — 股价反应：-X%
累计下跌：-XX%，同期行业指数：-X%，超额下跌：-XX%
```

## 基本面验证

### 两层验证体系

**第一层：财务健康检查**（排除财务风险）

| 检查项 | 通过标准 | 不通过则排除 |
|-------|---------|------------|
| 近4季度扣非净利润 | 至少3个季度为正 | 是 |
| 经营性现金流 | 近12个月为正 | 是 |
| 短期偿债 | 流动比率 > 1 且货币资金覆盖短期有息负债 | 是 |
| 审计意见 | 标准无保留意见 | 是 |
| ST/退市风险 | 不存在 | 是 |

**第二层：竞争力验证**（确认护城河完整）

| 检查项 | 评估方式 | 信号含义 |
|-------|---------|---------|
| 市场份额 | 营收增速 vs 行业增速 | 份额提升 = 护城河完好 |
| 毛利率趋势 | 近8个季度毛利率走势 | 稳定/上升 = 竞争优势持续 |
| 研发投入 | 研发费用增速 | 持续增长 = 创新能力维持 |
| 客户集中度 | 前五大客户占比变化 | 分散化或稳定 = 安全 |
| 核心管理层 | 近1年核心高管是否离职 | 稳定 = 利好 |
| 行业壁垒 | 新进入者情况 | 无显著新竞争者 = 壁垒稳固 |

## 问题分类决策树

```
问题根因是什么？
├── 政策/监管变化
│   ├── 政策是否改变行业基本盈利模式？
│   │   ├── 是 → 结构性（例：教育双减、房地产三道红线）
│   │   └── 否 → 暂时性（例：环保限产、阶段性补贴退坡）
│   └── 政策方向是否已明确？
│       ├── 是 → 可评估影响
│       └── 否 → 不确定 — 等待
├── 行业周期下行
│   ├── 行业是否有历史周期性？
│   │   ├── 是 → 暂时性（在周期底部可能是买入机会）
│   │   └── 否 → 可能结构性
│   └── 需求是否永久性减少？
│       ├── 是 → 结构性
│       └── 否 → 暂时性
├── 公司层面问题
│   ├── 短期业绩不达预期
│   │   ├── 原因是一次性的？ → 暂时性
│   │   └── 反映竞争力下降？ → 可能结构性
│   ├── 管理层丑闻/变动
│   │   ├── 治理机制是否健全？ → 暂时性
│   │   └── 公司治理根本有问题？ → 结构性
│   └── 财务造假/审计问题
│       └── 几乎总是结构性 → 排除
├── 宏观系统性风险
│   ├── 市场整体调整 → 暂时性（尤其是高质量公司被错杀）
│   └── 流动性危机 → 暂时性但恢复时间不确定
└── 外部冲击
    ├── 贸易摩擦/地缘政治
    │   ├── 是否直接影响公司核心收入？
    │   │   ├── 是（如出口占比>50%被制裁） → 半结构性
    │   │   └── 否 → 暂时性
    └── 自然灾害/疫情等 → 暂时性
```

## 偏差评分模型

### 评分公式

```
偏差得分 = 情绪负面程度（0–50分）+ 基本面强度（0–50分）
```

得分越高 = 偏差越大 = 逆向机会越好。

### 情绪负面程度（0–50分）

| 指标 | 权重 | 评分标准 |
|------|------|---------|
| 股价6月跌幅 | 15 | >40%=15, 30–40%=12, 20–30%=8, <20%=4 |
| 相对行业超额跌幅 | 10 | >20%=10, 10–20%=7, 5–10%=4, <5%=2 |
| 分析师下调 | 10 | 盈利下调>15%=10, 10–15%=7, 5–10%=4, <5%=2 |
| 北向资金 | 8 | 大幅减持=8, 小幅减持=5, 持平=2, 增持=0 |
| 融资/基金持仓 | 7 | 大幅下降=7, 小幅下降=4, 持平=2 |

### 基本面强度（0–50分）

| 指标 | 权重 | 评分标准 |
|------|------|---------|
| 营收增长 | 10 | 加速增长=10, 稳定增长=7, 放缓但正=4, 负增长=0 |
| 扣非净利润趋势 | 10 | 加速增长=10, 稳定=7, 放缓但正=4, 亏损=0 |
| 经营性现金流 | 10 | 充裕=10, 健康=7, 偏弱=4, 负值=0 |
| ROE水平 | 10 | >15%=10, 10–15%=7, 5–10%=4, <5%=2 |
| 护城河评估 | 10 | 稳固=10, 良好=7, 一般=4, 削弱=0 |

| 总分 | 偏差等级 | 含义 |
|------|---------|------|
| 80–100 | ★★★★★ 极大偏差 | 情绪极度负面但基本面非常强劲 |
| 65–79 | ★★★★ 显著偏差 | 明显的错误定价可能 |
| 50–64 | ★★★ 中等偏差 | 有一定偏差但不极端 |
| 35–49 | ★★ 轻微偏差 | 偏差有限或基本面有瑕疵 |
| < 35 | ★ 偏差不显著 | 低估可能合理 |

## 历史回溯研究

### A股超跌反弹统计

历史上A股大幅超跌且基本面未变的股票，平均回归路径：

- **6个月内**: 约40%的超跌股（跌幅>30%、基本面稳健）收复50%+跌幅
- **12个月内**: 约55%收复大部分跌幅
- **24个月内**: 约65%回到下跌前水平附近

**不回归的情况（价值陷阱）**：
- 行业发生结构性变化
- 公司治理持续恶化
- 财务问题被逐步暴露
- 技术路线被淘汰

### 催化剂类型及有效性

| 催化剂类型 | 历史有效性 | A股举例 |
|-----------|----------|---------|
| 业绩拐点 | 高 | 季报/半年报扣非利润企稳回升 |
| 政策转向 | 高 | 地产放松、平台经济松绑 |
| 管理层增持 | 中高 | 董事长大额增持 |
| 估值触及历史极值 | 中 | PB跌破历史底部 |
| 行业供给出清 | 中 | 同行业竞争者退出或减产 |
| 北向资金/机构回流 | 中 | 外资重新大量买入 |
| 分红/回购 | 中 | 大幅提高分红比例 |

## 常见陷阱

### 避免的心理偏差

- **锚定效应**：因股价从高点大幅下跌就认为"便宜"——可能之前严重高估
- **损失厌恶的对面**：过度乐观于"已经跌了这么多不可能再跌"
- **叙事偏差**：为负面事件构建"一定是暂时的"的故事，而忽视结构性恶化的证据
- **幸存者偏差**：只记得成功的逆向投资案例（中国平安2022低位），忽视失败案例

### A股特有陷阱

- **ST摘帽预期**：ST公司突击利润保壳，基本面并未实质改善
- **壳价值博弈**：小市值公司因壳价值获得溢价，注册制下此逻辑已弱化
- **概念题材透支**：某些行业（如元宇宙、区块链）曾因概念炒高后回落，基本面从未匹配
- **政策期望过度乐观**：期望政策全面转向（如地产完全放松）但实际可能是"托而不举"
