## 总回报计算

### 分红再投资回报

假设所有现金分红在除权日以收盘价买入：

1. 以初始投资金额和股数为起点
2. 每次分红时计算新增股数：`分红金额 / 除权日收盘价`
3. 累计持有股数
4. 期末价值 = 总持股数 × 当前股价
5. 年化回报 = `(期末价值 / 初始价值)^(1/年数) − 1`

**注意**：A股除权除息后股价自动调整，需使用前复权或后复权价格进行准确计算。

### 非再投资回报

价格回报 + 累计收到的现金分红：
- 总回报 = `(股价变动 + 累计分红) / 初始投资`

### 送股/转增处理

送股和转增会增加股数但不产生现金收入：
- 10送5：每10股增加5股，总股数变为1.5倍
- 10转10：每10股转增10股，总股数变为2倍
- 计算回报时需按除权后的股数和价格调整

## 分红指标

### 股息率

```
股息率 = 近12个月每股现金分红 / 当前股价 × 100%
```

- 使用实际已分配的现金分红，不含送股/转增
- 如公司当年尚未实施分红，使用上一年度分红金额
- 特别分红（如资产处置后的一次性分红）应单独标注

### 分红增长率

```
CAGR = (最新年度每股分红 / N年前每股分红)^(1/N) − 1
```

- 分别计算3年和5年CAGR
- 如有送股/转增导致股本变化，需使用调整后的每股分红
- 特殊分红不纳入增长率计算

### 分红连续性

| 模式 | 评估 |
|------|------|
| 连续5年以上现金分红且金额递增 | 优秀 |
| 连续5年以上现金分红，金额波动但趋势向上 | 良好 |
| 连续分红但偶有年份金额下降 | 合格 |
| 分红不连续，间断年份 | 较差 |
| 多年不分红或以送转代替现金分红 | 不合格 |

### 分红率

**利润口径**：
```
分红率 = 每股现金分红 / 每股收益 × 100%
```

**现金流口径**（更优）：
```
现金流分红率 = 现金分红总额 / 自由现金流 × 100%
```

| 利润分红率 | 评估 |
|-----------|------|
| < 30% | 非常安全，分红增长空间大 |
| 30–50% | 健康，分红与留存平衡 |
| 50–70% | 适中，需关注盈利稳定性 |
| 70–90% | 偏高，仅适用于盈利非常稳定的行业（银行、公用事业） |
| > 90% | 高风险，盈利下滑将直接威胁分红 |

### 自由现金流覆盖

```
FCF覆盖倍数 = 自由现金流 / 现金分红总额
```

| 覆盖倍数 | 评估 |
|---------|------|
| > 2.0倍 | 优秀，分红有充足现金支撑 |
| 1.5–2.0倍 | 良好 |
| 1.2–1.5倍 | 合格，安全边际偏薄 |
| 1.0–1.2倍 | 警示，现金流弱化将威胁分红 |
| < 1.0倍 | 危险，现金分红超过自由现金流 |

## 可持续性评分模型

| 因素 | 权重 | 评分标准 |
|------|------|---------|
| 自由现金流覆盖 | 25% | >2x=25, 1.5–2x=20, 1.2–1.5x=15, 1–1.2x=8, <1x=0 |
| 利润分红率 | 15% | <40%=15, 40–55%=12, 55–70%=8, 70–85%=4, >85%=0 |
| 分红连续性 | 15% | 连续5年递增=15, 连续5年稳定=10, 偶有波动=5, 间断=0 |
| 盈利稳定性 | 15% | 低波动=15, 中等波动=10, 高波动=5, 极不稳定=0 |
| 资产负债表 | 15% | 资产负债率<50%=15, 50–65%=10, 65–75%=5, >75%=0 |
| 行业防御性 | 10% | 公用事业/银行/消费=10, 工业/科技=7, 周期性=4 |
| 分红政策明确性 | 5% | 章程明确规定+历史执行=5, 仅惯例=3, 无承诺=1 |

| 总分 | 评级 |
|------|------|
| 80–100 | ★★★★★ 分红堡垒 |
| 65–79 | ★★★★ 非常可靠 |
| 50–64 | ★★★ 可靠，需持续关注 |
| 35–49 | ★★ 中等风险 |
| < 35 | ★ 分红削减风险较高 |

## 行业差异化标准

| 行业 | 分红率上限调整 | 特殊考量 |
|------|-------------|---------|
| 银行 | 35%（受资本充足率约束） | 关注核心一级资本充足率、拨备覆盖率 |
| 煤炭/石化 | 取决于商品周期 | 用周期均值盈利计算分红率 |
| 公用事业 | 可接受70–80% | 收入稳定，受价格管制保护 |
| 房地产 | 审慎评估 | 行业下行期分红可持续性存疑 |
| 消费 | 50–60% | 关注品牌力和渠道稳定性 |
| 钢铁/有色 | 强周期，分红率需按周期评估 | 高峰期高分红≠可持续 |

## 税后收益计算

A股个人投资者现金分红差异化征税：

| 持有期限 | 税率 | 实际股息率折扣 |
|---------|------|-------------|
| > 1年 | 5%（常见口径） | 股息率 × 0.95 |
| 1个月 – 1年 | 10% | 股息率 × 0.9 |
| < 1个月 | 20% | 股息率 × 0.8 |

**策略建议**：高股息策略应以长期持有（> 1年）为基础框架，以获得更低的税负摩擦（税费政策可能调整，请以当期规则为准）。
