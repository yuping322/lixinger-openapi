# 方法论：市场概览仪表板（A股）

本技能用于**提供A股市场的全景概览，包括指数表现、市场广度、资金流向、板块轮动、情绪指标等关键信息，帮助投资者快速把握市场状态**。市场概览是日常监控的基础，是投资决策的起点。

## 数据口径

### 数据源与字段映射

市场概览仪表板需要以下数据：
- **指数数据**：AKShare (`stock_zh_index_daily`), 东方财富
  - 上证指数、深证成指、创业板指、科创50、沪深300、中证500
  - 日频数据，实时更新
- **市场广度数据**：AKShare (`stock_zh_a_spot_em`), 同花顺
  - 上涨家数、下跌家数、涨停家数、跌停家数、平盘家数
  - 日频数据，实时更新
- **资金流向数据**：AKShare (`stock_market_fund_flow_em`)
  - 主力净流入、大单净流入、中单净流入、小单净流入
  - 日频数据，实时更新
- **北向资金数据**：AKShare (`stock_hsgt_north_net_flow_in_em`)
  - 北向资金净流入、沪股通净流入、深股通净流入
  - 日频数据，T日晚间更新
- **板块数据**：AKShare (`stock_board_industry_name_em`)
  - 申万一级行业涨跌幅、概念板块涨跌幅
  - 日频数据，实时更新
- **情绪指标**：AKShare (`stock_market_activity_legu`)
  - 两市成交额、换手率、市盈率、市净率
  - 日频数据，实时更新

关键字段：
- `指数涨跌幅`：各主要指数的涨跌幅（%）
- `上涨家数/下跌家数`：上涨和下跌的个股数量
- `涨停家数/跌停家数`：涨停和跌停的个股数量
- `主力净流入`：主力资金净流入金额（亿元）
- `北向资金净流入`：北向资金净流入金额（亿元）
- `两市成交额`：沪深两市总成交额（亿元）
- `领涨板块/领跌板块`：涨跌幅最大的行业或概念板块

### 时间窗口与频率

- **实时监控**：盘中实时更新（每分钟或每5分钟）
- **日频总结**：每日收盘后生成当日市场概览
- **周频总结**：每周汇总市场表现、资金流向、板块轮动
- **月频总结**：每月分析市场趋势、估值水平、风险提示

建议窗口：
- 当日表现：实时指数、涨跌家数、资金流向
- 短期趋势：5日、10日、20日指数涨跌幅和资金流向
- 中期趋势：60日、120日指数走势和市场广度
- 长期趋势：252日（年度）指数表现和估值水平

## 核心指标

### 指标列表与计算公式

1. **指数表现指标**：
   - `指数涨跌幅 = (当前价格 - 前收盘价) / 前收盘价` (%)
   - `指数N日涨跌幅 = (当前价格 - N日前价格) / N日前价格` (%)
   - `指数相对强度 = 个股指数涨跌幅 - 基准指数涨跌幅` (百分点)
   - `指数波动率 = std(日收益率)` over 20日 (%)

2. **市场广度指标**：
   - `上涨比例 = 上涨家数 / (上涨家数 + 下跌家数)` (%)
   - `涨停比例 = 涨停家数 / 总股票数` (%)
   - `跌停比例 = 跌停家数 / 总股票数` (%)
   - `涨跌比 = 上涨家数 / 下跌家数`
   - `市场广度指标 = (上涨家数 - 下跌家数) / 总股票数` (%)

3. **资金流向指标**：
   - `主力净流入 = 大单买入 - 大单卖出`（亿元）
   - `主力净流入强度 = 主力净流入 / 两市成交额` (%)
   - `北向资金净流入`（亿元）
   - `北向资金净流入强度 = 北向资金净流入 / 两市成交额` (%)
   - `资金流向一致性 = sign(主力净流入) == sign(北向资金净流入)`

4. **板块轮动指标**：
   - `领涨板块涨幅`：涨幅最大的行业/概念板块（%）
   - `领跌板块跌幅`：跌幅最大的行业/概念板块（%）
   - `板块分化度 = std(所有板块涨跌幅)` (%)
   - `板块轮动速度 = count(板块排名变化 > 5)` / 总板块数 (%)

5. **情绪指标**：
   - `两市成交额`（亿元）
   - `成交额百分位 = percentile_rank(当前成交额, 252日历史)` (%)
   - `平均换手率 = 两市成交额 / 两市流通市值` (%)
   - `市场情绪评分 = f(涨跌比, 涨停家数, 成交额, 北向流入)` (0-100)

6. **估值指标**：
   - `沪深300市盈率PE`
   - `沪深300市净率PB`
   - `PE百分位 = percentile_rank(当前PE, 5年历史)` (%)
   - `PB百分位 = percentile_rank(当前PB, 5年历史)` (%)

### 标准化方法

- 使用**百分位排名**对成交额、估值进行标准化（相对历史）
- 使用**评分**（0-100）量化市场情绪（0=极度悲观，100=极度乐观）
- 使用**比例**（%）衡量市场广度（上涨比例、涨停比例）
- 使用**相对强度**比较不同指数表现（个股指数 vs 基准指数）

## 信号与阈值

### 可测试规则（Insight Rules）

Rule 1 (市场广度强 + 资金流入 → 上涨趋势):
IF {上涨比例 >= 70% AND 主力净流入 > 100亿 AND 北向资金净流入 > 50亿 AND 涨停家数 > 50}
THEN {未来3-5个交易日，市场继续上涨概率 >= 68%，预期涨幅 >= 2%}
CONFIDENCE {0.68; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场整体，市场广度是重要的趋势指标}
FAILURE_MODE {市场已大幅上涨（追高风险）；突发利空（政策、地缘政治）；资金流入放缓；外围市场大跌（美股、港股）}

Rule 2 (市场广度弱 + 资金流出 → 下跌趋势):
IF {上涨比例 <= 30% AND 主力净流出 > 100亿 AND 北向资金净流出 > 50亿 AND 跌停家数 > 30}
THEN {未来3-5个交易日，市场继续下跌概率 >= 70%，预期跌幅 >= 2%}
CONFIDENCE {0.70; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场整体，市场广度弱是下跌信号}
FAILURE_MODE {市场已大幅下跌（超跌反弹）；政策利好（降息、降准）；资金流出放缓；外围市场大涨}

Rule 3 (成交额放量 + 上涨 → 趋势确认):
IF {两市成交额 > 1.5万亿 AND 成交额百分位 > 80 AND 上证指数涨幅 > 1% AND 上涨比例 > 60%}
THEN {未来5-10个交易日，市场上涨趋势确认，预期涨幅 >= 3%}
CONFIDENCE {0.65; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，成交额放量是趋势确认的重要信号}
FAILURE_MODE {放量是恐慌性抛售（虽然当日上涨）；成交额持续萎缩（趋势不可持续）；市场已高位（放量见顶）}

Rule 4 (估值低位 + 市场广度弱 → 底部区域):
IF {PE百分位 < 20 AND PB百分位 < 20 AND 上涨比例 < 40% AND 市场情绪评分 < 30}
THEN {未来1-3个月，市场处于底部区域，中长期配置价值高，预期反弹 >= 10%}
CONFIDENCE {0.63; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，估值低位叠加悲观情绪通常是底部信号}
FAILURE_MODE {基本面继续恶化（经济衰退）；政策不及预期；外围市场持续下跌；市场信心崩溃（流动性危机）}

### 触发/解除/无效化条件

- **触发监控**：
  - 上涨比例 >= 70%或 <= 30%（市场广度极端）
  - 涨停家数 > 50或跌停家数 > 30（情绪极端）
  - 主力净流入/流出 > 100亿（资金大幅流动）
  - 北向资金净流入/流出 > 50亿（外资大幅流动）
  - 两市成交额 > 1.5万亿或 < 0.6万亿（成交额极端）
  - PE/PB百分位 < 20或 > 80（估值极端）

- **解除监控**：
  - 市场广度恢复正常（上涨比例40-60%）
  - 资金流向恢复平衡（净流入/流出 < 50亿）
  - 成交额恢复正常（0.8-1.2万亿）
  - 估值恢复正常（PE/PB百分位30-70）

- **无效化**：
  - 市场休市（节假日、系统故障）
  - 数据异常（交易所数据错误）
  - 重大突发事件（熔断、战争、疫情）

### 阈值选择依据

- **70%上涨比例**：市场广度强，大部分个股上涨，趋势明确
- **30%上涨比例**：市场广度弱，大部分个股下跌，趋势明确
- **100亿主力净流入/流出**：显著的资金流动，约占日成交额的1%
- **50亿北向资金净流入/流出**：显著的外资流动，是重要的边际力量
- **1.5万亿成交额**：放量水平，约为日均成交额的1.5倍
- **0.6万亿成交额**：缩量水平，约为日均成交额的0.6倍
- **20/80百分位**：估值极端水平，历史上通常是底部/顶部区域

## 边界条件与降级策略

### 数据缺失/异常值处理

- **数据缺失**：
  - 指数数据缺失：使用前一交易日数据，标注"数据未更新"
  - 市场广度数据缺失：使用指数涨跌推算，标注"估算值"
  - 资金流向数据缺失：使用成交额推算，标注"估算值"

- **异常值处理**：
  - 涨停家数 > 200：检查是否有重大利好（政策、降息），标注"极端行情"
  - 跌停家数 > 100：检查是否有重大利空（黑天鹅），标注"极端行情"
  - 成交额 > 2万亿：检查是否有特殊事件（IPO、大盘股上市），标注"异常放量"

### 降级策略

- **AKShare接口不稳定**：
  - 备用数据源1：东方财富网页（`http://quote.eastmoney.com/center/`）
  - 备用数据源2：同花顺iFinD
  - 备用数据源3：手动输入（从交易软件获取）
  - 降级标注："使用备用数据源，可能有延迟"

- **市场广度数据缺失**：
  - 使用指数涨跌推算（指数涨 > 1%，上涨比例约60-70%）
  - 使用历史平均（上涨比例约50%）
  - 降级标注："市场广度为估算值"

- **资金流向数据缺失**：
  - 使用成交额变化推算（成交额放量，资金流入）
  - 使用北向资金代替主力资金
  - 降级标注："资金流向为估算值"

## A股特殊注意

### 1. T+1交易制度影响

- **当日买入次日才能卖出**：
  - 市场广度强时，次日可能继续上涨（锁定效应）
  - 市场广度弱时，次日可能继续下跌（无法止损）
  - 需要关注次日开盘情况

### 2. 涨跌停限制影响

- **涨停板限制买入**：
  - 涨停家数多时，市场情绪极度乐观
  - 但涨停股无法买入（追不上）
  - 次日可能继续涨停或高开回落

- **跌停板限制卖出**：
  - 跌停家数多时，市场情绪极度悲观
  - 但跌停股无法卖出（卖不出）
  - 次日可能继续跌停或低开反弹

### 3. 停牌影响

- **停牌股不计入市场广度**：
  - 停牌股较多时，市场广度失真
  - 需要剔除停牌股计算
  - 关注停牌原因（重组、风险）

### 4. 板块轮动特点

- **A股板块轮动快**：
  - 题材炒作频繁（概念板块）
  - 板块持续性差（1-2周）
  - 需要及时跟踪板块变化

- **龙头效应明显**：
  - 板块上涨通常由龙头股带动
  - 龙头股涨停，板块跟涨
  - 需要关注龙头股表现

### 5. 北向资金重要性

- **北向资金是增量资金**：
  - 外资持续流入，市场上涨
  - 外资持续流出，市场下跌
  - 北向资金是重要的边际力量

- **北向资金领先性**：
  - 北向资金流向通常领先市场
  - 北向资金逆势流入，可能是底部信号
  - 北向资金逆势流出，可能是顶部信号

### 6. 成交额与市场情绪

- **成交额放量**：
  - 上涨放量：趋势确认（资金追涨）
  - 下跌放量：恐慌性抛售（资金出逃）
  - 高位放量：可能见顶（获利了结）
  - 低位放量：可能见底（抄底资金）

- **成交额缩量**：
  - 上涨缩量：趋势不可持续（资金观望）
  - 下跌缩量：可能见底（抛压减轻）
  - 持续缩量：市场观望（等待方向）

### 7. 估值水平参考

- **PE/PB百分位**：
  - < 20%：低估值，底部区域
  - 20-40%：合理偏低
  - 40-60%：合理区间
  - 60-80%：合理偏高
  - > 80%：高估值，顶部区域

- **行业差异**：
  - 周期性行业：PE波动大，PB更有参考价值
  - 成长性行业：PE较高，关注PEG（PE/增长率）
  - 金融行业：PB更有参考价值（< 1倍PB为低估）

### 8. 政策敏感性

- **A股是政策市**：
  - 降息降准：利好，市场上涨
  - 监管趋严：利空，市场下跌
  - 需要关注政策动向

- **重要会议**：
  - 两会（3月）：政策预期
  - 中央经济工作会议（12月）：定调次年
  - 政治局会议（季度）：政策微调

### 9. 外围市场影响

- **美股影响**：
  - 美股大涨，A股通常高开
  - 美股大跌，A股通常低开
  - 需要关注美股走势（尤其是纳斯达克）

- **港股影响**：
  - 港股与A股联动性强（AH股）
  - 港股走势领先A股（外资主导）
  - 需要关注恒生指数、恒生科技指数

### 10. 季节性规律

- **春季躁动**（1-2月）：
  - 年初资金充裕，市场上涨概率高
  - 两会前政策预期，市场活跃

- **五穷六绝七翻身**：
  - 5-6月市场通常弱势（资金紧张）
  - 7月可能反弹（中报预期）

- **年末行情**（11-12月）：
  - 年末资金紧张，市场波动大
  - 次年预期，可能有跨年行情

## 回测说明（最小化）

- **回测设计**：
  - 每日收盘后生成市场概览
  - 根据市场广度、资金流向、情绪指标生成交易信号
  - 次日开盘买入/卖出
  - 持有期：3日/5日/10日
  - 基准：沪深300

- **性能指标**：
  - 信号胜率（信号后市场走势符合预期的概率）
  - 平均超额收益（信号后N日收益 - 基准收益）
  - 夏普比率、最大回撤

- **证伪条件**：
  - 信号胜率 <= 55%（不显著优于随机）
  - 平均超额收益 <= 0（不优于基准）

- **敏感性分析**：
  - 测试不同阈值（70% vs 75%上涨比例）
  - 测试不同市场环境（牛市 vs 熊市 vs 震荡市）
  - 测试不同持有期（3日 vs 5日 vs 10日）

## 使用示例

### 示例1：强势市场

**输入**：2024-02-15（周四）收盘
**数据**：
- 上证指数：+1.8%，深证成指：+2.3%，创业板指：+2.8%
- 上涨家数：3850，下跌家数：1150，上涨比例：77%
- 涨停家数：85，跌停家数：5
- 主力净流入：+180亿，北向资金净流入：+75亿
- 两市成交额：1.65万亿（放量）
- 领涨板块：科技（+4.5%）、新能源（+3.8%）

**输出**：
- 市场状态：强势上涨
- 市场广度：强（77%上涨比例）
- 资金流向：大幅流入（主力+北向）
- 市场情绪：乐观（85个涨停）
- **结论**：市场趋势向上，建议持股或加仓，关注科技、新能源板块

### 示例2：弱势市场

**输入**：2024-03-20（周三）收盘
**数据**：
- 上证指数：-1.5%，深证成指：-2.1%，创业板指：-2.8%
- 上涨家数：1200，下跌家数：3800，上涨比例：24%
- 涨停家数：8，跌停家数：45
- 主力净流出：-150亿，北向资金净流出：-60亿
- 两市成交额：0.75万亿（缩量）
- 领跌板块：房地产（-5.2%）、金融（-3.5%）

**输出**：
- 市场状态：弱势下跌
- 市场广度：弱（24%上涨比例）
- 资金流向：大幅流出（主力+北向）
- 市场情绪：悲观（45个跌停）
- **结论**：市场趋势向下，建议减仓或观望，规避房地产、金融板块

### 示例3：底部区域

**输入**：2024-04-10（周三）收盘
**数据**：
- 上证指数：-0.5%，PE百分位：18%，PB百分位：15%
- 上涨比例：35%
- 主力净流出：-50亿，北向资金净流入：+20亿（逆势）
- 两市成交额：0.65万亿（极度缩量）
- 市场情绪评分：25（悲观）

**输出**：
- 市场状态：底部区域
- 估值水平：低估（PE/PB百分位 < 20%）
- 市场情绪：悲观（情绪评分25）
- 北向资金：逆势流入（底部信号）
- **结论**：市场处于底部区域，中长期配置价值高，可逢低布局优质股

### 示例4：震荡市场

**输入**：2024-05-15（周三）收盘
**数据**：
- 上证指数：+0.2%，深证成指：-0.1%，创业板指：+0.3%
- 上涨比例：52%
