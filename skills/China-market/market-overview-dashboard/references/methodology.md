# 方法论（Draft）

本文件用于沉淀该技能的计算口径、阈值建议、边界条件与常见陷阱。后续完善时优先补齐：数据字段映射、阈值来源与样本期。

## 数据口径

- **数据源与字段映射**：
    - **成交额**：两市总成交额（沪市 + 深市）。
    - **北向**：北向资金净流入。
    - **涨跌分布**：红盘率。
    - **主线板块**：涨幅 Top 3 行业/概念。
- **时间窗口与频率**：
    - **实时**：每分钟刷新。
    - **日终**：盘后总结。

## 核心指标

- **万亿成交**：
    - 两市成交额 > 1 万亿人民币。系牛市或活跃市场的门槛。若 < 6000 亿，为流动性枯竭（地量）。

- **赚钱效应 (Money Effect)**：
    - (涨停家数 + 涨幅>5%家数) / (跌停家数 + 跌幅>5%家数)。
    - 直观反映今日是否容易赚钱。

## 信号与阈值

- **信号定义**：
    - **放量滞涨**：成交额创新高但指数收十字星或阴线。资金分歧巨大，见顶风险。
    - **地量见地价**：成交额萎缩至年内低点，且指数不再创新低。

- **阈值选择依据**：
    - **换手率**：全A换手率 < 0.6% 通常是底部区域。

## 边界条件与降级策略

- **数据缺失/异常值处理**：
    - 若北向通道关闭（假期），在仪表盘中置灰并提示。
- **若某接口不稳定**：
    - 仅展示指数涨跌幅和成交额。

## A股特殊注意

- **各种“茅”权重**：
    - 上证 50 / 沪深 300 很大程度上受“茅指数”（核心资产）影响。看盘时需区分“黄线”（小票）和“白线”（权重）的剪刀差。
- **期指交割日**：
    - 每月第三个周五是股指期货交割日，尾盘常有剧烈波动（日内资金平仓/移仓）。仪表盘需高亮提示该日期。
