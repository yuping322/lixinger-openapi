# 方法论：股东风险检查（A股）

本技能用于**监控A股上市公司股东结构变化、大股东减持、股权质押、一致行动人变化等股东风险**。股东风险是A股特有的重要风险，股东行为往往影响公司治理和股价表现。

## 数据口径

### 数据源与字段映射

股东风险检查需要以下数据：
- **股东结构**：AKShare (`stock_individual_fund_flow`), 巨潮资讯
  - 前十大股东、前十大流通股东
  - 持股数量、持股比例、持股变化
  - 季频数据，财报后更新
- **减持计划**：AKShare (`stock_share_change_cninfo`), 交易所公告
  - 减持主体、减持数量、减持比例、减持时间
  - 实时数据，公告后更新
- **股权质押**：AKShare (`stock_pledge_stat`), 交易所
  - 质押数量、质押比例、质押方、质权人
  - 周频数据，每周更新
- **股东关系**：公司公告、天眼查
  - 一致行动人、关联方、实际控制人
  - 不定期更新，重大变化时公告

关键字段：
- `大股东持股比例`：第一大股东持股比例 (%)
- `前十大股东持股比例`：前十大股东合计持股比例 (%)
- `减持比例`：计划减持数量 / 总股本 (%)
- `质押比例`：质押股份 / 持股总数 (%)
- `控制权稳定性`：实际控制人是否变化

### 时间窗口与频率

- **季频监控**：每季度财报后更新股东结构
- **周频监控**：每周监控减持公告、质押变化
- **日频预警**：每日监控减持实施、质押平仓风险
- **实时预警**：重大股东变化、控制权变更实时预警

建议窗口：
- 减持窗口：公告后3-6个月（减持实施期）
- 质押风险：实时监控（股价接近平仓线）
- 股东变化：季度对比（股东结构变化）

## 核心指标

### 指标列表与计算公式

1. **股东集中度指标**：
   - `第一大股东持股比例 = 第一大股东持股 / 总股本` (%)
   - `前三大股东持股比例 = 前三大股东合计持股 / 总股本` (%)
   - `前十大股东持股比例 = 前十大股东合计持股 / 总股本` (%)
   - `股权集中度 = 1 - Σ(持股比例^2)`（赫芬达尔指数）

2. **减持风险指标**：
   - `减持比例 = 计划减持数量 / 总股本` (%)
   - `减持金额 = 减持数量 × 当前股价`（亿元）
   - `减持进度 = 已减持数量 / 计划减持数量` (%)
   - `减持压力 = 未减持数量 / 日均成交量`（天）

3. **质押风险指标**：
   - `质押比例 = 质押股份 / 持股总数` (%)
   - `质押市值 = 质押股份 × 当前股价`（亿元）
   - `平仓线距离 = (当前股价 - 平仓线) / 当前股价` (%)
   - `质押风险等级 = f(质押比例, 平仓线距离)`

4. **控制权稳定性指标**：
   - `控制权集中度 = 实际控制人持股比例` (%)
   - `控制权变化 = 1{实际控制人变化}`
   - `一致行动人数量 = count(一致行动人)`
   - `控制权争夺风险 = 1{第一大股东持股 < 30% AND 前三大股东持股接近}`

5. **股东变化指标**：
   - `新进股东数量 = count(新进前十大股东)`
   - `退出股东数量 = count(退出前十大股东)`
   - `股东持股变化率 = (当前持股 - 上期持股) / 上期持股` (%)
   - `机构持股比例 = 机构投资者持股 / 总股本` (%)

### 标准化方法

- 使用**分位数**对股东集中度进行标准化（相对全市场）
- 使用**风险等级**对质押风险进行分类（低/中/高风险）
- 使用**变化率**计算股东持股变化（相对上期）
- 使用**压力测试**评估减持对股价的影响

## 信号与阈值

### 可测试规则（Insight Rules）

Rule 1 (大股东大比例减持 → 股价压力):
IF {减持比例 >= 2% AND 减持主体为第一大股东 AND 减持压力 >= 20天}
THEN {减持期间（3-6个月），股价下跌概率 >= 68%，预期跌幅 >= 10%}
CONFIDENCE {0.68; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股个股，流动性一般的股票（日成交额 < 2亿）风险更大}
FAILURE_MODE {减持是财务需求（非看空）；市场整体向好（系统性支撑）；公司基本面改善（对冲减持）；减持价格设置下限（保护股价）}

Rule 2 (高比例质押 + 股价下跌 → 平仓风险):
IF {质押比例 >= 80% AND 平仓线距离 <= 15% AND 股价20日跌幅 >= 20%}
THEN {未来5-10个交易日，股价继续下跌概率 >= 70%，可能触发平仓}
CONFIDENCE {0.70; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股个股，质押比例高的股票}
FAILURE_MODE {大股东补充质押（降低平仓风险）；股价反弹（远离平仓线）；质权人不强平（协商展期）；公司回购（稳定股价）}

Rule 3 (控制权变更 → 不确定性增加):
IF {实际控制人变化 OR 第一大股东变化 AND 新控制人持股 < 30%}
THEN {变更后3-6个月，股价波动加大，不确定性增加}
CONFIDENCE {0.60; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股个股，民营企业风险更大}
FAILURE_MODE {新控制人实力强（改善预期）；控制权变更是友好收购；公司基本面稳定；新控制人承诺不改变主业}

Rule 4 (机构持股增加 → 正面信号):
IF {机构持股比例季度增加 >= 5百分点 AND 新进机构 >= 3家 AND 机构为知名机构}
THEN {未来1-2个季度，股价上涨概率 >= 65%，预期涨幅 >= 15%}
CONFIDENCE {0.65; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股个股，中小盘股信号更强}
FAILURE_MODE {机构持股是被动配置（指数调整）；机构误判（基本面转差）；市场整体转弱；机构短期持有（快进快出）}

### 触发/解除/无效化条件

- **触发监控**：
  - 减持公告（减持比例 >= 1%）
  - 质押比例 >= 60%或平仓线距离 <= 20%
  - 控制权变更公告
  - 机构持股变化 >= 3百分点

- **解除监控**：
  - 减持完成或减持计划取消
  - 质押比例降至 < 50%或平仓线距离 > 30%
  - 控制权稳定（新控制人持股 >= 30%）
  - 股东结构稳定（连续2季度无重大变化）

- **无效化**：
  - 公司退市或被收购
  - 数据缺失或不可靠
  - 公司重大重组（股东结构重置）

### 阈值选择依据

- **2%减持比例**：减持2%以上对股价有显著影响
- **80%质押比例**：质押比例80%以上为高风险
- **15%平仓线距离**：股价距平仓线15%以内为危险区域
- **30%控制权**：持股30%以上通常能稳定控制权
- **5百分点机构持股增加**：机构持股增加5百分点以上为显著增持

## 边界条件与降级策略

### 数据缺失/异常值处理

- **数据缺失**：
  - 股东数据缺失：使用上一季度数据，标注"数据延迟"
  - 质押数据缺失：使用交易所官网数据，标注"手动更新"
  - 减持公告缺失：使用龙虎榜数据推算，标注"推算数据"

- **异常值处理**：
  - 持股比例 > 100%：检查是否数据错误
  - 质押比例 > 100%：检查是否重复质押
  - 减持比例 > 持股比例：检查是否数据错误

### 降级策略

- **股东数据不全**：
  - 仅监控前三大股东（忽略其他股东）
  - 使用公开信息推算股东变化
  - 降级标注："股东数据不全，仅提供部分分析"

- **质押数据缺失**：
  - 使用历史质押数据推算
  - 使用行业平均质押比例估算
  - 降级标注："质押数据缺失，使用估算值"

- **减持公告延迟**：
  - 使用龙虎榜数据补充
  - 使用大宗交易数据补充
  - 降级标注："减持数据可能延迟"

## A股特殊注意

### 1. T+1交易制度影响

- **减持需要T+1**：大股东减持当日买入次日才能卖出
- **影响较小**：减持通常是计划性的，T+1影响不大

### 2. 涨跌停限制影响

- **跌停无法卖出**：股价跌停时大股东无法减持
- **质押平仓延后**：跌停时无法强平，风险累积
- **连续跌停风险**：质押股票连续跌停，平仓风险激增

### 3. 停牌影响

- **停牌期间无法减持**：重大事项停牌期间大股东无法减持
- **复牌后集中减持**：复牌后可能集中减持，股价压力大
- **质押风险累积**：停牌期间质押风险累积，复牌后可能爆发

### 4. 减持新规

- **减持比例限制**：大股东减持有比例限制（3个月不超过1%）
- **减持方式限制**：集中竞价、大宗交易、协议转让
- **减持公告要求**：提前15个交易日公告
- **需要关注新规变化**：减持新规不断调整，影响减持行为

### 5. 质押新规

- **质押比例限制**：单一股票质押比例不超过50%（部分券商）
- **质押用途限制**：质押资金用途受限（不得炒股）
- **质押风险管理**：券商加强质押风险管理，平仓线提高
- **需要关注质押新规**：质押新规影响质押行为和风险

### 6. 控制权争夺

- **举牌制度**：持股5%需要公告，每增加5%再公告
- **收购要约**：持股30%需要发起全面要约收购
- **一致行动人**：一致行动人合计持股计算
- **需要关注举牌公告**：举牌往往引发控制权争夺

### 7. 股东减持动机

- **财务需求**：大股东资金需求（还债、投资）
- **看空公司**：大股东不看好公司前景
- **股权激励**：高管行权后减持套现
- **需要区分减持动机**：不同动机对股价影响不同

### 8. 机构投资者行为

- **机构调研**：机构调研后可能增持
- **机构重仓**：机构重仓股通常质地较好
- **机构减持**：机构减持可能是风格切换，非看空
- **需要关注机构行为**：机构行为往往领先市场

### 9. 股东协议

- **一致行动协议**：股东之间签订一致行动协议
- **表决权委托**：股东委托表决权给其他股东
- **股份锁定协议**：股东承诺一定期限内不减持
- **需要关注股东协议**：协议影响控制权稳定性

### 10. 国资背景

- **国资股东**：国资股东通常不减持，控制权稳定
- **国资改革**：国资改革可能引入战略投资者
- **国资纾困**：国资纾困民企，接盘质押股份
- **需要区分国资和民资**：国资背景公司股东风险较低

## 回测说明（最小化）

- **回测设计**：
  - 减持公告后买入/卖出
  - 持有期：减持期间（3-6个月）
  - 基准：个股前期走势、行业指数

- **性能指标**：
  - 信号胜率（减持后股价下跌概率）
  - 平均跌幅（减持期间平均跌幅）
  - 最大回撤

- **证伪条件**：
  - 信号胜率 <= 50%（不优于随机）
  - 平均跌幅 <= 0（减持不影响股价）

## 使用示例

### 示例1：大股东大比例减持

**输入**：个股A，2024-02-15
**数据**：
- 第一大股东计划减持3%
- 减持金额：约5亿元
- 减持压力：25天（日均成交2000万）
- 减持原因：财务需求
**输出**：
- 信号：大股东大比例减持（股价压力）
- **结论**：减持比例大，减持压力重，短期股价承压，建议回避

### 示例2：高比例质押风险

**输入**：个股B，2024-03-20
**数据**：
- 质押比例：85%
- 平仓线距离：12%
- 20日跌幅：-25%
- 当前股价：8.5元，平仓线：7.5元
**输出**：
- 信号：高比例质押 + 股价下跌（平仓风险）
- **结论**：质押比例高，股价接近平仓线，平仓风险大，建议回避

### 示例3：控制权变更

**输入**：个股C，2024-04-10
**数据**：
- 实际控制人变化：A集团 → B集团
- 新控制人持股：28%
- 原控制人持股：25%（减持至15%）
- 控制权争夺：是
**输出**：
- 信号：控制权变更（不确定性增加）
- **结论**：控制权变更，新控制人持股不足30%，不确定性大，观望

### 示例4：机构持股增加

**输入**：个股D，2024Q1
**数据**：
- 机构持股比例：从15%升至22%（+7百分点）
- 新进机构：5家（含知名公募基金）
- 机构类型：公募基金、社保基金
- 公司基本面：业绩增速30%
**输出**：
- 信号：机构持股增加（正面信号）
- **结论**：知名机构增持，公司基本面好，中长期看好
