# 方法论：商誉风险监控（A股）

本技能用于**监控A股上市公司商誉规模、商誉减值风险、并购标的业绩承诺完成情况**。商誉减值是A股重要的业绩地雷，商誉风险监控是风险管理的关键环节。

## 数据口径

### 数据源与字段映射

商誉风险监控需要以下数据：
- **商誉数据**：AKShare (`stock_financial_abstract`), 财报
  - 商誉余额、商誉占净资产比例、商誉占市值比例
  - 季频数据，财报后更新
- **并购数据**：公司公告、巨潮资讯
  - 并购标的、并购价格、业绩承诺、承诺期
  - 不定期更新，并购时公告
- **业绩承诺**：公司公告
  - 承诺净利润、实际净利润、完成率、补偿条款
  - 年频数据，年报后更新
- **减值测试**：公司公告、财报附注
  - 减值测试方法、关键假设、减值准备
  - 年频数据，年报后更新

关键字段：
- `商誉余额`：商誉账面价值（亿元）
- `商誉占净资产比例`：商誉 / 净资产 (%)
- `商誉占市值比例`：商誉 / 市值 (%)
- `业绩承诺完成率`：实际净利润 / 承诺净利润 (%)
- `商誉减值风险等级`：低/中/高风险

### 时间窗口与频率

- **年频监控**：每年年报后更新商誉数据
- **季频监控**：每季度监控并购标的业绩
- **实时预警**：业绩预告、商誉减值公告实时预警

建议窗口：
- 业绩承诺期：通常3年
- 减值测试：每年年报
- 风险预警：承诺期最后一年、业绩大幅下滑时

## 核心指标

### 指标列表与计算公式

1. **商誉规模指标**：
   - `商誉余额 = 并购成本 - 可辨认净资产公允价值`（亿元）
   - `商誉占净资产比例 = 商誉 / 净资产` (%)
   - `商誉占市值比例 = 商誉 / 市值` (%)
   - `商誉占总资产比例 = 商誉 / 总资产` (%)

2. **业绩承诺指标**：
   - `业绩承诺完成率 = 实际净利润 / 承诺净利润` (%)
   - `累计完成率 = Σ(实际净利润) / Σ(承诺净利润)` (%)
   - `业绩缺口 = 承诺净利润 - 实际净利润`（亿元）
   - `补偿金额 = f(业绩缺口, 补偿条款)`（亿元）

3. **商誉减值风险指标**：
   - `减值风险等级 = f(商誉占比, 业绩完成率, 行业景气度)`
   - `潜在减值金额 = 商誉 × 减值比例估计`（亿元）
   - `减值对EPS影响 = 潜在减值 / 总股本`（元/股）
   - `减值对净资产影响 = 潜在减值 / 净资产` (%)

4. **并购标的质量指标**：
   - `并购溢价率 = (并购价格 - 净资产) / 净资产` (%)
   - `并购PE = 并购价格 / 承诺净利润`（倍）
   - `标的盈利能力 = 标的净利润 / 标的净资产` (%)
   - `标的业绩增速 = (当期净利润 - 去年同期) / 去年同期` (%)

5. **商誉变化指标**：
   - `商誉增加额 = 当期商誉 - 上期商誉`（亿元）
   - `商誉减值额 = 当期商誉减值损失`（亿元）
   - `商誉减值比例 = 商誉减值额 / 商誉余额` (%)
   - `历史减值次数 = count(商誉减值年份)`

### 标准化方法

- 使用**占比指标**衡量商誉风险（商誉/净资产、商誉/市值）
- 使用**完成率**衡量业绩承诺风险（实际/承诺）
- 使用**风险等级**分类商誉风险（低/中/高）
- 使用**情景分析**估算潜在减值影响

## 信号与阈值

### 可测试规则（Insight Rules）

Rule 1 (高商誉占比 + 业绩承诺未完成 → 减值风险):
IF {商誉占净资产 >= 50% AND 业绩承诺完成率 <= 80% AND 承诺期最后一年}
THEN {年报后商誉减值概率 >= 70%，预期减值金额 >= 商誉余额 × 30%}
CONFIDENCE {0.70; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股个股，并购较多的公司（传媒、计算机、医药）}
FAILURE_MODE {标的业绩反转（完成承诺）；公司调整减值测试假设（延后减值）；监管放松（减值压力小）；公司盈利能力强（吸收减值）}

Rule 2 (商誉占市值高 + 股价下跌 → 减值压力):
IF {商誉占市值 >= 30% AND 股价年跌幅 >= 30% AND 标的业绩下滑 >= 20%}
THEN {年报后商誉减值概率 >= 65%，股价可能继续下跌 >= 15%}
CONFIDENCE {0.65; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股个股，市值较小的公司（< 100亿）风险更大}
FAILURE_MODE {股价反弹（市值上升）；标的业绩改善；公司其他业务盈利强（对冲减值）；减值测试通过（无需减值）}

Rule 3 (业绩承诺刚完成 + 承诺期结束 → 业绩变脸风险):
IF {业绩承诺完成率 = 100-105% AND 承诺期刚结束 AND 并购溢价率 >= 300%}
THEN {承诺期后1-2年，标的业绩下滑概率 >= 60%，可能触发商誉减值}
CONFIDENCE {0.60; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股个股，高溢价并购的公司}
FAILURE_MODE {标的业绩持续增长（非业绩承诺驱动）；标的有核心竞争力；母公司支持力度大；行业景气度上升}

Rule 4 (历史多次减值 → 并购质量差):
IF {历史减值次数 >= 2 AND 累计减值金额 >= 商誉余额 × 50% AND 公司继续并购}
THEN {新并购标的减值风险高，投资者应谨慎}
CONFIDENCE {0.65; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股个股，并购频繁的公司}
FAILURE_MODE {公司吸取教训（提高并购质量）；新并购标的质地好；行业景气度上升；公司管理能力提升}

### 触发/解除/无效化条件

- **触发监控**：
  - 商誉占净资产 >= 30%
  - 业绩承诺完成率 <= 90%
  - 承诺期最后一年
  - 标的业绩大幅下滑（>= 20%）

- **解除监控**：
  - 商誉减值完成（商誉余额大幅下降）
  - 业绩承诺期结束且标的业绩稳定
  - 商誉占比降至 < 20%（减值或净资产增加）

- **无效化**：
  - 公司出售并购标的（商誉消失）
  - 公司重大重组（商誉重置）
  - 会计准则变化（商誉处理方式变化）

### 阈值选择依据

- **50%商誉占净资产**：商誉占净资产50%以上为高风险
- **30%商誉占市值**：商誉占市值30%以上，减值对股价影响大
- **80%业绩承诺完成率**：完成率80%以下，减值风险显著上升
- **300%并购溢价率**：溢价率300%以上，商誉规模大，减值风险高
- **2次历史减值**：历史减值2次以上，并购质量差

## 边界条件与降级策略

### 数据缺失/异常值处理

- **数据缺失**：
  - 商誉数据缺失：使用上一期数据，标注"数据延迟"
  - 业绩承诺数据缺失：使用公告数据补充，标注"手动更新"
  - 减值测试数据缺失：使用行业平均减值比例估算

- **异常值处理**：
  - 商誉占比 > 100%：检查是否净资产为负
  - 业绩完成率 > 200%：检查是否数据错误或业绩爆发
  - 减值金额 > 商誉余额：检查是否数据错误

### 降级策略

- **商誉数据不全**：
  - 使用公开信息推算商誉规模
  - 使用行业平均商誉占比估算
  - 降级标注："商誉数据不全，使用估算值"

- **业绩承诺数据缺失**：
  - 使用标的历史业绩推算
  - 使用行业平均业绩增速估算
  - 降级标注："业绩承诺数据缺失，使用估算值"

- **减值测试数据缺失**：
  - 使用历史减值比例推算
  - 使用行业平均减值比例估算
  - 降级标注："减值测试数据缺失，使用行业平均"

## A股特殊注意

### 1. T+1交易制度影响

- **商誉减值公告后无法当日卖出**：减值公告通常在盘后，次日开盘可能跌停
- **需要提前预警**：在年报前识别减值风险，提前规避

### 2. 涨跌停限制影响

- **减值公告后连续跌停**：商誉大额减值可能导致连续跌停，无法卖出
- **风险极大**：需要提前识别风险，避免被套

### 3. 停牌影响

- **重大事项停牌**：商誉减值可能导致停牌，复牌后补跌
- **停牌期间风险累积**：停牌期间无法交易，风险累积

### 4. 商誉减值会计处理

- **商誉不摊销**：A股商誉不摊销，仅在减值时一次性计提
- **减值不可转回**：商誉减值后不可转回，永久损失
- **减值时点集中**：商誉减值通常集中在年报，四季度业绩预告时预警

### 5. 业绩承诺制度

- **业绩承诺期通常3年**：并购时承诺3年业绩，承诺期结束后风险上升
- **业绩承诺刚完成风险大**：承诺期内勉强完成，承诺期后可能变脸
- **补偿条款**：业绩未完成时原股东补偿，但补偿往往不足

### 6. 监管趋严

- **减值测试要求提高**：监管要求更严格的减值测试，减值压力增加
- **商誉减值问询**：商誉减值时监管问询，需要充分说明
- **需要关注监管动态**：监管政策变化影响商誉减值

### 7. 行业差异

- **并购较多的行业**：传媒、计算机、医药等行业并购多，商誉风险大
- **周期性行业**：周期性行业景气度下行时，商誉减值风险上升
- **需要区分行业**：不同行业商誉风险不同

### 8. 并购动机

- **外延式扩张**：通过并购快速扩张，商誉规模大
- **跨界并购**：跨界并购整合难度大，商誉减值风险高
- **需要区分并购动机**：不同动机并购质量不同

### 9. 商誉减值影响

- **业绩大幅下滑**：商誉减值导致业绩大幅下滑，甚至亏损
- **净资产减少**：商誉减值减少净资产，可能导致ST
- **股价大跌**：商誉减值公告后股价通常大跌
- **需要提前规避**：商誉减值影响大，需要提前识别风险

### 10. 商誉减值时点

- **集中在年报**：商誉减值通常在年报时计提
- **四季度业绩预告**：四季度业绩预告时可能预警商誉减值
- **需要关注年报前**：年报前是商誉减值高发期，需要重点关注

## 回测说明（最小化）

- **回测设计**：
  - 年报前识别高商誉风险公司
  - 年报后统计商誉减值情况
  - 计算减值预测准确率

- **性能指标**：
  - 减值预测准确率
  - 减值金额预测误差
  - 规避减值后的超额收益

- **证伪条件**：
  - 预测准确率 <= 50%（不优于随机）
  - 规避减值后收益 <= 0（无效）

## 使用示例

### 示例1：高商誉占比 + 业绩承诺未完成

**输入**：个股A，2024年报前
**数据**：
- 商誉余额：15亿元
- 净资产：25亿元，商誉占比：60%
- 业绩承诺完成率：75%（承诺期最后一年）
- 标的业绩：承诺5亿，实际3.75亿
**输出**：
- 信号：高商誉占比 + 业绩承诺未完成（减值风险）
- **结论**：商誉减值风险高，预计减值4-6亿元，建议回避

### 示例2：商誉占市值高 + 股价下跌

**输入**：个股B，2024年报前
**数据**：
- 商誉余额：8亿元
- 市值：20亿元，商誉占市值：40%
- 股价年跌幅：-35%
- 标的业绩下滑：-25%
**输出**：
- 信号：商誉占市值高 + 股价下跌（减值压力）
- **结论**：商誉减值压力大，减值后股价可能继续下跌，建议回避

### 示例3：业绩承诺刚完成

**输入**：个股C，2024年（承诺期刚结束）
**数据**：
- 业绩承诺完成率：102%（勉强完成）
- 承诺期：2021-2023年，刚结束
- 并购溢价率：400%
- 标的2024年业绩：预计下滑30%
**输出**：
- 信号：业绩承诺刚完成（业绩变脸风险）
- **结论**：承诺期结束后业绩可能变脸，未来1-2年商誉减值风险高

### 示例4：历史多次减值

**输入**：个股D，2024年
**数据**：
- 历史减值次数：3次（2019、2021、2023年）
- 累计减值金额：12亿元
- 原商誉余额：20亿元，累计减值60%
- 公司2024年新并购：5亿元
**输出**：
- 信号：历史多次减值（并购质量差）
- **结论**：公司并购质量差，新并购减值风险高，建议回避
