# 方法论：投资备忘录生成器（A股）

本技能用于**生成结构化的投资备忘录（Investment Memo），总结投资逻辑、关键假设、风险因素、估值分析、投资建议等核心内容**。投资备忘录是机构投资的标准输出，用于投资决策、内部沟通、合规存档。

## 数据口径

### 数据源与字段映射

投资备忘录生成需要以下数据：
- **公司基本信息**：AKShare (`stock_individual_info_em`), 公司公告
  - 公司名称、股票代码、所属行业、主营业务、实际控制人
  - 不定期更新（公司变更时）
- **财务数据**：AKShare (`stock_financial_abstract_em`), Wind
  - 营业收入、净利润、ROE、毛利率、资产负债率
  - 季度/年度更新（财报披露时）
- **估值数据**：AKShare (`stock_a_lg_indicator`)
  - PE、PB、PS、市值、股价
  - 日频更新
- **行业数据**：AKShare (`stock_board_industry_name_em`), 券商研报
  - 行业规模、行业增速、竞争格局、行业趋势
  - 不定期更新
- **公司公告**：AKShare (`stock_notice_report`)
  - 业绩预告、重组公告、风险提示、股东变更
  - 实时更新
- **券商研报**：Wind、同花顺
  - 评级、目标价、盈利预测、投资逻辑
  - 不定期更新

关键字段：
- `公司名称`、`股票代码`、`所属行业`、`主营业务`
- `营业收入`、`净利润`、`ROE`、`毛利率`、`净利率`
- `PE`、`PB`、`市值`、`股价`
- `行业规模`、`行业增速`、`市场份额`
- `投资逻辑`、`关键假设`、`风险因素`、`估值方法`、`目标价`

### 时间窗口与频率

- **初次生成**：首次研究公司时，生成完整的投资备忘录
- **季度更新**：每季度财报披露后，更新财务数据和估值
- **重大事件更新**：重大公告（重组、业绩预告、风险提示）时更新
- **年度复盘**：每年年报披露后，全面复盘投资逻辑和假设

建议窗口：
- 财务分析：最近4个季度、最近3年年报
- 估值分析：当前估值 vs 历史估值（5年）、行业估值
- 盈利预测：未来2-3年EPS、营收、净利润
- 风险监控：持续跟踪风险因素变化

## 核心指标

### 指标列表与计算公式

1. **投资逻辑指标**：
   - `核心竞争力`：技术壁垒、品牌优势、渠道优势、成本优势
   - `成长驱动力`：市场扩容、份额提升、产品升级、并购整合
   - `盈利能力`：ROE、毛利率、净利率、ROIC
   - `财务质量`：经营现金流/净利润、资产负债率、应收账款周转

2. **关键假设指标**：
   - `营收增长假设`：未来3年营收CAGR（%）
   - `利润率假设`：未来3年净利率（%）
   - `估值假设`：目标PE或PB
   - `行业假设`：行业增速、竞争格局、政策环境

3. **估值指标**：
   - `PE估值 = 目标EPS × 目标PE`
   - `PB估值 = 目标每股净资产 × 目标PB`
   - `DCF估值 = Σ(未来现金流 / (1+WACC)^t) + 终值`
   - `相对估值 = 可比公司平均PE × 公司EPS`

4. **风险指标**：
   - `行业风险`：政策风险、竞争风险、技术风险、周期风险
   - `公司风险`：经营风险、财务风险、治理风险、合规风险
   - `估值风险`：估值过高、盈利预测过于乐观、市场风格切换
   - `流动性风险`：成交量、换手率、大股东减持

5. **投资建议指标**：
   - `目标价 = 估值方法计算的合理价格`
   - `预期收益 = (目标价 - 当前价) / 当前价` (%)
   - `投资评级`：强烈推荐/推荐/中性/减持（基于预期收益）
   - `持有期限`：短期（< 6个月）/中期（6-12个月）/长期（> 12个月）

### 标准化方法

- 使用**SWOT分析**（优势、劣势、机会、威胁）结构化投资逻辑
- 使用**情景分析**（乐观/中性/悲观）量化关键假设
- 使用**多种估值方法**（PE、PB、DCF、相对估值）交叉验证
- 使用**风险矩阵**（概率 × 影响）量化风险因素

## 信号与阈值

### 可测试规则（Insight Rules）

Rule 1 (强投资逻辑 + 低估值 → 强烈推荐):
IF {核心竞争力强 AND 成长驱动力清晰 AND PE < 行业PE × 0.8 AND 预期收益 >= 30%}
THEN {投资评级：强烈推荐，目标价上调，持有期限：12个月}
CONFIDENCE {0.70; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，投资逻辑清晰且估值合理的公司}
FAILURE_MODE {关键假设不成立（行业增速低于预期、份额提升不及预期）；估值中枢下移（市场风格切换）；突发风险事件（政策变化、竞争加剧）}

Rule 2 (关键假设变化 → 更新备忘录):
IF {实际营收增速 < 假设增速 - 5% OR 实际净利率 < 假设净利率 - 2% OR 行业政策重大变化}
THEN {立即更新投资备忘录，重新评估投资逻辑和目标价}
CONFIDENCE {0.85; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，关键假设是投资逻辑的基础}
FAILURE_MODE {短期波动（季节性、一次性因素）；假设本身过于保守（实际表现更好）}

Rule 3 (风险因素实现 → 降低评级):
IF {识别的风险因素实现（如政策利空、竞争加剧、财务恶化）AND 预期收益 < 10%}
THEN {投资评级下调至中性或减持，目标价下调，建议减仓或清仓}
CONFIDENCE {0.80; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，风险管理是投资的重要环节}
FAILURE_MODE {风险因素影响小于预期；公司成功化解风险；市场整体上涨掩盖风险}

Rule 4 (达到目标价 → 兑现收益):
IF {股价 >= 目标价 × 0.95 AND 持有期 >= 6个月}
THEN {建议兑现收益，卖出或减仓，更新投资备忘录}
CONFIDENCE {0.75; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，及时兑现收益是投资纪律}
FAILURE_MODE {公司基本面超预期（上调目标价）；市场情绪高涨（继续上涨）；长期持有逻辑（不急于卖出）}

### 触发/解除/无效化条件

- **触发生成备忘录**：
  - 首次研究公司（建仓前）
  - 重大投资决策（加仓、减仓、清仓）
  - 季度财报披露（更新财务数据）
  - 重大公告（重组、业绩预告、风险提示）
  - 投资逻辑变化（行业趋势、竞争格局）

- **触发更新备忘录**：
  - 关键假设变化（营收增速、利润率、估值）
  - 风险因素实现（政策变化、竞争加剧）
  - 股价达到目标价（兑现收益）
  - 持有期满（复盘投资逻辑）

- **无效化**：
  - 公司退市（投资失败）
  - 公司被并购（投资逻辑改变）
  - 清仓（不再持有）

### 阈值选择依据

- **30%预期收益**：强烈推荐的收益阈值，足以覆盖风险和机会成本
- **10%预期收益**：中性评级的收益阈值，低于10%不值得投资
- **0.8倍行业PE**：低估值的阈值，低于行业PE的80%通常有安全边际
- **5%营收增速差异**：关键假设变化的阈值，超过5%需要重新评估
- **2%净利率差异**：关键假设变化的阈值，超过2%需要重新评估
- **95%目标价**：兑现收益的阈值，接近目标价时考虑卖出

## 边界条件与降级策略

### 数据缺失/异常值处理

- **数据缺失**：
  - 财报未披露：使用业绩预告或券商预测，标注"预估数据"
  - 行业数据缺失：使用券商研报或行业协会数据，标注"第三方数据"
  - 公司公告缺失：手动查询巨潮资讯，标注"手动获取"

- **异常值处理**：
  - 财务数据异常（如ROE > 50%）：检查是否有非经常性损益，标注"数据异常"
  - 估值数据异常（如PE < 0）：使用其他估值方法（PB、PS），标注"PE不适用"

### 降级策略

- **AKShare接口不稳定**：
  - 备用数据源1：Wind金融终端
  - 备用数据源2：同花顺iFinD
  - 备用数据源3：手动输入（从公司公告、券商研报获取）
  - 降级标注："使用备用数据源，可能有延迟"

- **行业数据缺失**：
  - 使用券商行业研报
  - 使用行业协会数据（如中汽协、钢铁协会）
  - 使用可比公司数据推算
  - 降级标注："行业数据为估算值"

- **盈利预测缺失**：
  - 使用券商一致预期
  - 使用历史增速外推
  - 自行建模预测
  - 降级标注："盈利预测为自行估算"

## A股特殊注意

### 1. T+1交易制度影响

- **投资决策需更谨慎**：
  - 买入后次日才能卖出（无法当日止损）
  - 需要更充分的研究和更高的确定性
  - 避免冲动交易（追涨杀跌）

### 2. 涨跌停限制影响

- **目标价设定需考虑涨跌停**：
  - 业绩超预期可能连续涨停（难以买入）
  - 业绩暴雷可能连续跌停（难以卖出）
  - 需要提前布局或及时止损

### 3. 停牌影响

- **流动性风险**：
  - 重大事项停牌（重组、控制权变更）
  - 停牌期间无法交易（流动性归零）
  - 复牌后可能大涨或大跌（不确定性）
  - 需要在备忘录中明确标注停牌风险

### 4. 信息披露特点

- **公告是主要信息来源**：
  - 财报、业绩预告、重组公告、风险提示
  - 需要及时跟踪公告，更新备忘录
  - 关注"补充披露"、"更正公告"（可能是风险信号）

### 5. 非经常性损益

- **需要区分主营业务和非经常性损益**：
  - 资产出售、政府补助、投资收益
  - 关注扣非净利润（更能反映主营业务）
  - 在备忘录中明确标注非经常性损益影响

### 6. 会计调整

- **需要关注会计政策变更**：
  - 收入确认、费用资本化、减值准备
  - 关注审计意见（标准无保留/保留/否定）
  - 在备忘录中明确标注会计风险

### 7. 行业政策影响

- **A股是政策市**：
  - 行业政策（如新能源补贴、房地产调控）
  - 货币政策（降息降准、流动性）
  - 监管政策（如减持新规、退市新规）
  - 需要在备忘录中明确标注政策风险和政策假设

### 8. 市场风格影响

- **估值中枢随市场风格变化**：
  - 牛市：高估值被接受（成长股PE > 50）
  - 熊市：低估值受青睐（价值股PE < 10）
  - 震荡市：估值回归合理（PE 15-25）
  - 需要在备忘录中明确标注估值假设和市场风格假设

### 9. 大股东减持

- **需要关注大股东减持风险**：
  - 减持计划公告（提前3个月）
  - 减持实施公告（实时）
  - 大股东减持通常是利空（信心不足）
  - 需要在备忘录中明确标注减持风险

### 10. 合规要求

- **机构投资需要合规存档**：
  - 投资备忘录是合规要求（证明投资决策过程）
  - 需要保存完整的研究记录（财务数据、估值模型、风险分析）
  - 需要定期更新（季度、年度、重大事件）
  - 需要复盘（投资逻辑是否成立、假设是否准确）

## 回测说明（最小化）

- **回测设计**：
  - 根据投资备忘录的投资建议（强烈推荐/推荐）买入
  - 持有至目标价或持有期满
  - 跟踪关键假设变化，及时更新备忘录
  - 基准：沪深300、中证500

- **性能指标**：
  - 投资成功率（达到目标价的比例）
  - 平均收益率（实际收益 vs 预期收益）
  - 平均持有期（实际持有期 vs 预期持有期）
  - 关键假设准确率（实际 vs 假设）

- **证伪条件**：
  - 投资成功率 < 60%（大部分投资失败）
  - 平均收益率 < 预期收益率 × 0.5（收益远低于预期）
  - 关键假设准确率 < 70%（假设不准确）

- **敏感性分析**：
  - 测试不同估值方法（PE vs PB vs DCF）
  - 测试不同关键假设（乐观 vs 中性 vs 悲观）
  - 测试不同市场环境（牛市 vs 熊市）
  - 测试不同行业（周期 vs 成长 vs 价值）

## 使用示例

### 示例1：成长股投资备忘录

**公司**：某新能源汽车公司（市值1000亿）

**投资逻辑**：
- 核心竞争力：技术领先（电池技术）、品牌优势（高端定位）
- 成长驱动力：新能源汽车渗透率提升（2024年30% → 2027年50%）、公司份额提升（2024年8% → 2027年12%）
- 盈利能力：ROE 18%，毛利率25%，净利率12%

**关键假设**：
- 营收增长：2024-2027年CAGR 35%
- 净利率：维持12%
- 目标PE：30（行业平均35）

**估值分析**：
- 2027年EPS预测：5元
- 目标价：5元 × 30 = 150元
- 当前价：100元
- 预期收益：50%

**风险因素**：
- 行业风险：补贴退坡、竞争加剧
- 公司风险：产能爬坡不及预期、技术迭代风险
- 估值风险：市场风格切换（成长股估值下降）

**投资建议**：
- 评级：强烈推荐
- 目标价：150元
- 预期收益：50%
- 持有期限：24个月

### 示例2：价值股投资备忘录

**公司**：某银行股（市值5000亿）

**投资逻辑**：
- 核心竞争力：区域优势（长三角）、资产质量好（不良率1.0%）
- 成长驱动力：经济复苏、息差企稳
- 盈利能力：ROE 11%，稳定

**关键假设**：
- 净利润增长：2024-2027年CAGR 5%
- ROE：维持11%
- 目标PB：0.8（行业平均0.7）

**估值分析**：
- 2027年每股净资产预测：15元
- 目标价：15元 × 0.8 = 12元
- 当前价：8元
- 预期收益：50%（含股息）

**风险因素**：
- 行业风险：经济下行、资产质量恶化
- 公司风险：息差收窄、不良率上升
- 估值风险：利率上升（债券收益率上升，股息吸引力下降）

**投资建议**：
- 评级：推荐
- 目标价：12元
- 预期收益：50%（含股息）
- 持有期限：36个月

### 示例3：备忘录更新（关键假设变化）

**公司**：某消费股（市值500亿）

**原关键假设**（2024年1月）：
- 营收增长：2024-2027年CAGR 25%
- 净利率：维持20%
- 目标PE：35

**实际情况**（2024年Q3）：
- 营收增长：2024年前三季度增长15%（低于假设25%）
- 净利率：18%（低于假设20%）
- 原因：行业竞争加剧、促销力度加大

**更新后关键假设**：
- 营收增长：下调至2024-2027年CAGR 18%
- 净利率：下调至18%
- 目标PE：下调至30（竞争加剧，估值中枢下移）

**更新后估值**：
- 2027年EPS预测：3元（原4元）
- 目标价：3元 × 30 = 90元（原140元）
- 当前价：100元
- 预期收益：-10%

**更新后投资建议**：
- 评级：下调至中性（原强烈推荐）
- 目标价：下调至90元（原140元）
- 建议：减仓或清仓

### 示例4：备忘录复盘（投资结束）

**公司**：某科技股（市值200亿）

**投资期间**：2023年1月 - 2024年12月（24个月）

**原投资逻辑**：
- 核心竞争力：技术领先（AI芯片）
- 成长驱动力：AI应用爆发
- 关键假设：营收CAGR 40%，净利率15%，目标PE 40

**实际表现**：
- 营收CAGR：45%（超预期）
- 净利率：18%（超预期）
- 股价：从50元涨至150元（+200%）

**复盘总结**：
- 投资逻辑：成立（AI应用确实爆发）
- 关键假设：保守（实际表现更好）
- 风险管理：成功（及时兑现收益）
- 经验教训：对高成长行业可以更乐观；及时跟踪行业趋势
