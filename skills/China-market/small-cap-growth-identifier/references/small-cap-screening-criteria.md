## 核心筛选标准

### 市值范围

| 分类 | 市值区间 | 特征 |
|------|---------|------|
| 中小盘 | 100–200亿 | 流动性尚可，部分已有机构关注 |
| 小盘 | 50–100亿 | 机构关注开始减少 |
| 微盘 | 20–50亿 | 多数机构不覆盖，信息不对称最大 |
| 极微盘 | < 20亿 | 流动性极差，风险极高，建议排除 |

### 增长指标

**营收增长**：
- 3年CAGR > 20%（硬性门槛）
- 或最近2年 > 25%（如上市时间短）
- 增长应为内生增长（排除并购驱动的增长）

**验证增长真实性**：
- 营收增速 vs 行业增速 → 确认市场份额在提升
- 营收增速 vs 应收账款增速 → 应收增速不应显著快于营收
- 合同负债/预收账款趋势 → 上升说明订单储备良好
- 前五大客户集中度 → 过高（>60%）的集中度使增长脆弱

**利润率趋势**：
- 毛利率：稳定或扩大（3年）
- 营业利润率：改善趋势（允许前期投入导致的暂时压缩，但需有明确拐点预期）
- 扣非净利润率：正值且趋势向好

### 资产负债表

| 指标 | 健康标准 | 警示标准 |
|------|---------|---------|
| 资产负债率 | < 50% | > 60% |
| 有息负债率 | < 30% | > 40% |
| 流动比率 | > 1.5 | < 1.0 |
| 货币资金 / 短期借款 | > 1.5 | < 1.0 |
| 商誉 / 净资产 | < 15% | > 30% |
| 经营性现金流 / 净利润 | > 80% | < 50% |

### 实控人/管理层持股

| 持股比例 | 评估 |
|---------|------|
| > 40% | 优秀 — 利益高度绑定 |
| 25–40% | 良好 |
| 15–25% | 合格 |
| < 15% | 需关注 — 管理层利益绑定弱 |

### 机构持仓

| 公募持仓占流通股 | 机构发现阶段 |
|----------------|------------|
| < 3% | 未被发现 — 信息不对称最大 |
| 3–5% | 初始发现 — 少数基金经理开始关注 |
| 5–10% | 早期共识 — 部分机构已认可 |
| > 10% | 已被广泛发现 — "被忽视"溢价消失 |

## 质量评分模型

### 成长质量评分（0–100）

| 因素 | 权重 | 评分标准 |
|------|------|---------|
| 营收增速 | 20% | >40%=20, 30–40%=16, 20–30%=12, <20%=0 |
| 利润率趋势 | 15% | 持续扩大=15, 稳定=10, 轻微收缩=5, 持续收缩=0 |
| 现金流质量 | 15% | 收现比>100%=15, 80–100%=10, 60–80%=5, <60%=0 |
| 资产负债表 | 10% | 强=10, 良=7, 中=4, 弱=0 |
| 实控人持股 | 10% | >40%=10, 25–40%=7, 15–25%=4, <15%=2 |
| 行业壁垒 | 10% | 高壁垒=10, 中等=6, 低=2 |
| 研发能力 | 10% | 强=10, 中=6, 弱=2 |
| 估值吸引力 | 10% | PEG<0.8=10, 0.8–1.2=7, 1.2–1.5=4, >1.5=2 |

| 总分 | 评级 |
|------|------|
| 80–100 | ★★★★★ 极具吸引力 |
| 65–79 | ★★★★ 非常有潜力 |
| 50–64 | ★★★ 值得关注 |
| 35–49 | ★★ 有瑕疵 |
| < 35 | ★ 不推荐 |

## 实控人与管理层分析

### 实控人评估

| 评估维度 | 正面信号 | 负面信号 |
|---------|---------|---------|
| 持股比例 | 高且稳定，近期有增持 | 低且持续减持 |
| 质押比例 | 无质押或低质押 | 高比例质押（>50%） |
| 行业经验 | 深耕行业10年以上 | 频繁跨行业、非专业背景 |
| 过往记录 | 无违规记录 | 有处罚或诚信问题 |
| 管理风格 | 专注主业，审慎扩张 | 盲目多元化、资本运作导向 |
| 分红回购 | 有合理分红或回购 | 从不分红 |

### 关键人物风险

小公司往往高度依赖实控人/创始人：

| 风险等级 | 条件 |
|---------|------|
| 低 | 管理团队完整，有清晰的继任计划 |
| 中 | 实控人是核心，但团队有一定独立能力 |
| 高 | 公司完全依赖实控人的个人关系/技术/决策 |

## 机构发现潜力评估

### 发现催化剂

| 催化剂 | 信号 | 时间预期 |
|-------|------|---------|
| 公募基金首次出现在前十大 | 季报披露 | 立即 |
| 知名分析师首次覆盖 | 研报发布 | 1–3个月 |
| 入选指数（中证500/1000等） | 指数调整公告 | 按指数调整周期 |
| 入选专精特新名单 | 工信部公告 | 不定期 |
| 北交所转板创业板 | 交易所公告 | 中期 |
| 业绩超预期引发关注 | 业绩公告 | 季度 |
| 行业/公司被媒体/KOL报道 | 舆论关注 | 不确定 |

### 被发现后的估值重估空间

小盘股从"未被发现"到"机构共识"的估值提升路径：

```
阶段1（未发现）: 估值折价20–40% vs 行业中位数
阶段2（初始发现）: 估值折价缩小至10–20%
阶段3（广泛共识）: 估值向行业中位数靠拢
阶段4（成长明星）: 享受成长溢价10–30%
```

## 被忽视原因分类

| 原因类型 | 细分 | 投资含义 |
|---------|------|---------|
| **规模因素** | 市值太小不满足基金最低配置要求 | 随公司成长将自然解决 |
| **流动性** | 日成交量不足以容纳机构资金 | 成交量提升是前置条件 |
| **覆盖缺失** | 没有或极少卖方分析师覆盖 | 首次覆盖是催化剂 |
| **行业冷门** | 所属行业不是当前市场热点 | 行业轮动到来时可能爆发 |
| **地域偏远** | 公司注册地在非一线城市，调研不便 | 线上调研可部分弥补 |
| **历史偏见** | 公司曾经业绩差/出过问题，市场记忆长 | 需时间和业绩证明转变 |
| **板块偏见** | 北交所整体关注度低 | 北交所生态逐步完善 |
| **叙事缺失** | 公司没有"好故事"（如AI、新能源） | 实际价值不依赖叙事 |

## 小盘股特有风险

### 风险矩阵

| 风险类型 | 描述 | 缓释 |
|---------|------|------|
| **流动性风险** | 日成交额低，买卖冲击成本高 | 关注日均成交额>3000万的标的 |
| **信息风险** | 信息披露质量差、透明度低 | 深入阅读年报，交叉验证数据 |
| **治理风险** | 实控人一言堂、关联交易多 | 关注独立董事比例、审计意见 |
| **退市风险** | 注册制下退市标准趋严 | 排除面值退市、财务退市风险标的 |
| **资金链风险** | 小公司抗风险能力弱 | 关注现金储备和融资能力 |
| **大客户依赖** | 前五大客户占比过高 | 客户集中度 < 50% |
| **解禁风险** | 创始人/早期投资者减持 | 关注解禁时间表和减持意愿 |

### 小盘股投资纪律

1. **分散化**：单只小盘股仓位不超过组合的5%
2. **设止损**：个股下跌超20%重新评估
3. **关注成交量**：日均成交额 < 1000万的标的不碰
4. **财务验证**：仔细阅读年报附注，关注异常科目
5. **实地调研**：尽可能参加业绩说明会或查阅调研纪要
