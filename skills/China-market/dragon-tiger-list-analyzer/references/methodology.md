# 方法论：龙虎榜分析器（A股）

本技能用于**分析龙虎榜数据（机构席位、营业部、买卖动向、个股上榜统计）以识别资金偏好与异常成交风险**。龙虎榜是A股特色的信息披露制度，揭示了游资、机构等大资金的交易行为，是短线交易和风险监控的重要工具。

## 数据口径

### 数据源与字段映射

龙虎榜分析需要以下数据：
- **龙虎榜明细**：AKShare (`stock_lhb_detail_em`), 东方财富
  - 上榜日期、股票代码、上榜原因、买入席位、卖出席位
  - 日频数据，T日晚间更新（通常20:00后）
- **席位类型**：机构专用、游资营业部、普通营业部
  - 机构专用：基金、保险、QFII等机构席位
  - 游资营业部：知名游资席位（如"中信证券杭州延安路"）
  - 普通营业部：散户为主的营业部
- **个股上榜统计**：AKShare (`stock_lhb_stock_statistic_em`)
  - 上榜次数、累计买入额、累计卖出额、净买入额
  - 历史累计数据
- **市场数据**：AKShare (`stock_zh_a_hist`)
  - 个股价格、成交量、涨跌幅
  - 日频数据，实时更新

关键字段：
- `上榜原因`：涨幅偏离值达7%、连续三日涨幅偏离值达20%、振幅达15%、换手率达20%等
- `买入席位`：买入金额前5的席位名称和金额
- `卖出席位`：卖出金额前5的席位名称和金额
- `净买入额`：买入总额 - 卖出总额（席位或个股维度）
- `机构席位数`：买入/卖出席位中机构专用的数量
- `游资席位数`：买入/卖出席位中知名游资的数量

### 时间窗口与频率

- **日频监控**：每日收盘后更新，T日晚间可获取T日龙虎榜
- **周频总结**：每周汇总上榜个股、热门席位、机构动向
- **月频趋势**：每月分析上榜频率、席位偏好、成功率

建议窗口：
- 短期跟踪：5日内上榜个股的后续表现
- 中期统计：20日内席位的胜率和收益
- 长期分析：60日内机构席位的配置方向

## 核心指标

### 指标列表与计算公式

1. **上榜强度指标**：
   - `上榜频率 = N日内上榜次数`
   - `净买入强度 = 净买入额 / 流通市值` (%)
   - `换手率 = 龙虎榜成交额 / 流通市值` (%)
   - `集中度 = Top1席位买入额 / 总买入额` (%)

2. **席位类型指标**：
   - `机构参与度 = 机构席位数 / 总席位数` (%)
   - `游资参与度 = 游资席位数 / 总席位数` (%)
   - `机构净买入 = 机构买入额 - 机构卖出额` (亿元)
   - `游资净买入 = 游资买入额 - 游资卖出额` (亿元)

3. **席位胜率指标**：
   - `N日胜率 = 上榜后N日上涨次数 / 上榜总次数` (%)
   - `平均收益 = mean(上榜后N日涨跌幅)` (%)
   - `夏普比率 = 平均收益 / 收益标准差`

4. **异常交易指标**：
   - `对倒嫌疑 = 1{买入席位与卖出席位重叠 >= 2个}`
   - `接力嫌疑 = 1{连续3日上榜 AND 每日不同游资买入}`
   - `出货嫌疑 = 1{机构卖出 >= 3席 AND 游资买入 >= 2席}`

### 标准化方法

- 使用**席位历史胜率**排名识别强势席位（Top20%）
- 使用**净买入强度**标准化不同市值个股的资金流入
- 使用**上榜频率**识别异常活跃个股（> 5次/月）
- 使用**机构参与度**区分机构主导和游资主导

## 信号与阈值

### 可测试规则（Insight Rules）

Rule 1 (机构集中买入 → 中期上涨):
IF {机构席位数 >= 3 AND 机构净买入 > 0 AND 游资席位数 <= 1 AND 上榜原因非"跌幅偏离"}
THEN {未来10-20个交易日，个股上涨概率 >= 60%，预期涨幅 >= 5%}
CONFIDENCE {0.62; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，流通市值 >= 50亿的个股信号更可靠}
FAILURE_MODE {机构买入后基本面恶化（业绩暴雷）；市场系统性下跌；机构买入是被动配置（指数调整）；上榜后连续涨停无法买入}

Rule 2 (游资接力 + 连续上榜 → 短期爆发但风险高):
IF {连续3日上榜 AND 每日游资席位数 >= 3 AND 每日不同游资 AND 上榜原因为"涨幅偏离"}
THEN {未来3-5个交易日，个股可能继续上涨（涨幅 >= 10%），但随后回调风险极高（回撤 >= 15%）}
CONFIDENCE {0.55; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，小市值股票（< 100亿）、题材股、ST股信号更明显}
FAILURE_MODE {游资分歧（买入和卖出席位都是游资）；监管介入（异常交易核查）；涨停板打开后集中抛售；散户接盘不足}

Rule 3 (机构卖出 + 游资买入 → 出货信号):
IF {机构卖出席位数 >= 3 AND 游资买入席位数 >= 2 AND 机构净卖出 > 游资净买入}
THEN {未来5-10个交易日，个股下跌概率 >= 65%，预期跌幅 >= 5%}
CONFIDENCE {0.68; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，前期大幅上涨（60日涨幅 > 50%）的个股信号更强}
FAILURE_MODE {游资强势接盘（后续继续上涨）；机构卖出是止盈（非看空）；基本面超预期（业绩大增）}

Rule 4 (高频上榜 + 对倒嫌疑 → 操纵风险):
IF {20日内上榜次数 >= 5 AND 对倒嫌疑次数 >= 2 AND 换手率 >= 30%}
THEN {个股存在操纵嫌疑，监管介入风险高，建议回避或减仓}
CONFIDENCE {0.70; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，小市值股票（< 50亿）、ST股、壳资源股风险更高}
FAILURE_MODE {高频上榜是正常交易（如重组、并购）；监管未介入；操纵成功（继续拉升）}

### 触发/解除/无效化条件

- **触发监控**：
  - 个股上榜（任何原因）
  - 机构席位数 >= 3（机构集中）
  - 连续3日上榜（异常活跃）
  - 20日内上榜次数 >= 5（高频上榜）

- **解除监控**：
  - 上榜后10日内未再上榜（恢复正常）
  - 机构/游资席位消失（资金退出）
  - 股价回归正常波动（涨跌幅 < 3%持续5日）

- **无效化**：
  - 个股停牌（无法交易）
  - 个股ST或退市（风险股）
  - 数据缺失或延迟 > 2天
  - 龙虎榜规则变化（如阈值调整）

### 阈值选择依据

- **机构席位数 >= 3**：买入或卖出席位中有3个及以上机构专用，表明机构集中行动
- **连续3日上榜**：短期内多次上榜，表明异常活跃或操纵嫌疑
- **20日内上榜5次**：月度高频上榜，表明持续异常交易
- **净买入强度 >= 5%**：龙虎榜净买入占流通市值5%以上，是显著的资金流入
- **换手率 >= 30%**：龙虎榜成交占流通市值30%以上，表明高度活跃

## 边界条件与降级策略

### 数据缺失/异常值处理

- **数据缺失**：
  - 节假日：无龙虎榜数据（正常）
  - 系统故障：使用东方财富网页数据作为备份
  - 席位名称缺失：标注"未知席位"，无法判断类型

- **异常值处理**：
  - 单席位买入 > 10亿：检查是否数据错误，若确认则标注"超大单"
  - 净买入强度 > 20%：检查是否流通市值错误，若确认则标注"极端流入"
  - 上榜原因异常：检查是否新增上榜原因，若确认则标注"新规则"

### 降级策略

- **AKShare接口不稳定**：
  - 备用数据源1：东方财富网页爬取（`http://data.eastmoney.com/stock/lhb.html`）
  - 备用数据源2：同花顺龙虎榜（`http://data.10jqka.com.cn/market/longhu/`）
  - 备用数据源3：手动输入（从交易所官网获取）
  - 降级标注："使用备用数据源，可能有延迟"

- **席位类型无法识别**：
  - 使用席位名称关键词匹配（"机构专用"、"证券杭州"等）
  - 使用历史席位库匹配（维护知名游资席位列表）
  - 无法匹配时标注"未知类型"
  - 降级标注："席位类型基于关键词匹配，可能不准确"

- **历史数据不足**（< 60天）：
  - 使用可用历史计算胜率（标注样本期）
  - 使用市场平均胜率代替席位胜率
  - 降级标注："历史数据不足，胜率可能不准确"

## A股特殊注意

### 1. T+1交易制度影响

- **上榜当日无法卖出**：龙虎榜席位当日买入的股票次日才能卖出，因此：
  - 上榜次日开盘是关键观察点（是否继续买入或集中卖出）
  - 连续上榜比单日上榜更有持续性（锁定效应）
  - 游资接力需要次日新游资接盘

### 2. 涨跌停限制影响

- **涨停板上榜**：个股涨停时上榜，次日走势关键：
  - 次日继续涨停：游资接力成功，短期强势
  - 次日高开回落：游资分歧，可能见顶
  - 次日低开：游资出逃，风险信号

- **跌停板上榜**：个股跌停时上榜，通常是恐慌性抛售：
  - 机构买入：可能是抄底，关注反弹
  - 游资买入：可能是博反弹，风险较高
  - 机构卖出：可能是止损，继续看空

### 3. 停牌影响

- **上榜后停牌**：个股上榜后停牌（如重组、核查），导致：
  - 龙虎榜席位被动锁定（无法卖出）
  - 复牌后可能大幅波动（补涨或补跌）
  - 需要关注停牌原因和复牌时间

### 4. 上榜原因分类

A股龙虎榜上榜原因包括：
- **涨幅偏离值达7%**：单日涨幅与指数偏离 >= 7%
- **连续三日涨幅偏离值达20%**：3日累计涨幅与指数偏离 >= 20%
- **跌幅偏离值达7%**：单日跌幅与指数偏离 >= 7%
- **振幅达15%**：单日最高价与最低价差 >= 15%
- **换手率达20%**：单日换手率 >= 20%（科创板/创业板为30%）
- **连续三日内收盘价格涨幅偏离值累计达到±20%**

不同上榜原因的信号强度不同：
- 涨幅偏离：强势信号，关注持续性
- 跌幅偏离：弱势信号，关注反弹
- 振幅/换手率：活跃信号，关注方向

### 5. 席位类型识别

- **机构专用席位**：
  - 名称包含"机构专用"
  - 通常是基金、保险、QFII、社保等
  - 买入通常是中长期配置，卖出通常是止盈或调仓

- **知名游资席位**：
  - 如"中信证券杭州延安路"、"华泰证券深圳益田路荣超商务中心"等
  - 通常是短线操作，追涨杀跌
  - 需要维护知名游资席位库（定期更新）

- **普通营业部**：
  - 其他营业部席位
  - 通常是散户为主，参考价值较低

### 6. 对倒和操纵识别

- **对倒**：同一资金在买入和卖出席位都出现，制造虚假成交：
  - 买入席位和卖出席位有重叠（同一营业部或关联营业部）
  - 买入额和卖出额接近（如都是5000万）
  - 需要人工判断或使用关联营业部库

- **接力**：多个游资轮流买入，推高股价：
  - 连续多日上榜，每日不同游资买入
  - 前日买入的游资次日卖出，新游资接盘
  - 接力成功：股价持续上涨；接力失败：股价暴跌

- **出货**：机构或游资利用散户接盘出货：
  - 机构或知名游资大量卖出
  - 普通营业部或散户买入
  - 通常发生在股价高位或前期大涨后

### 7. 监管风险

- **异常交易核查**：高频上榜、对倒嫌疑可能被监管核查：
  - 交易所问询函
  - 证监会立案调查
  - 可能导致停牌、处罚、甚至退市

- **内幕交易**：上榜前后有重大事项公告，可能涉及内幕交易：
  - 如上榜后公告重组、业绩大增
  - 需要关注上榜时间与公告时间的关系

### 8. 数据更新时间

- **T日晚间更新**：龙虎榜数据在T日收盘后更新（通常20:00-21:00）：
  - 盘中无法获取当日龙虎榜
  - 次日早盘前可获取前日完整数据
  - 需要在晚间或次日早盘分析

### 9. 科创板/创业板差异

- **上榜阈值不同**：
  - 主板：涨跌幅 ±10%，换手率 20%
  - 科创板/创业板：涨跌幅 ±20%，换手率 30%
  - 科创板/创业板上榜难度更高，信号更强

### 10. 龙虎榜局限性

- **仅披露前5席位**：只能看到买入/卖出金额最大的5个席位，其他席位不可见
- **无法看到持仓**：只能看到当日买卖，无法看到席位的总持仓
- **滞后性**：T日晚间才能看到T日数据，无法实时跟踪
- **可能误导**：游资可能故意制造龙虎榜信号（如对倒）

## 回测说明（最小化）

- **回测设计**：
  - 每日收盘后获取龙虎榜（T日晚间）
  - 次日开盘买入/卖出（T+1日）
  - 持有期：3日/5日/10日
  - 基准：个股上榜前收盘价

- **性能指标**：
  - 信号胜率（上榜后N日上涨概率）
  - 平均收益（上榜后N日涨跌幅）
  - 夏普比率、最大回撤
  - 分类统计（机构主导 vs 游资主导）

- **证伪条件**：
  - 信号胜率 <= 50%（不优于随机）
  - 平均收益 <= 0（不优于持有）
  - 夏普比率 <= 0.5（风险调整后收益不佳）

- **敏感性分析**：
  - 测试不同席位类型（机构 vs 游资）
  - 测试不同上榜原因（涨幅偏离 vs 换手率）
  - 测试不同市值（大盘 vs 小盘）
  - 测试不同市场环境（牛市 vs 熊市）

## 使用示例

### 示例1：机构集中买入

**输入**：2024-02-15，某股票上榜
**龙虎榜数据**：
- 上榜原因：涨幅偏离值达7%（当日涨9.5%）
- 买入席位：机构专用4席，合计买入2.5亿
- 卖出席位：普通营业部5席，合计卖出1.8亿
- 机构净买入：+2.5亿
- 流通市值：80亿
- 净买入强度：3.1%

**分析**：
- 机构集中买入（4席），散户卖出
- 净买入强度适中（3.1%）
- 上榜原因是涨幅偏离（强势）
- **结论**：机构看好，中期上涨概率高，建议关注或买入

### 示例2：游资接力

**输入**：2024-03-18-20，某股票连续3日上榜
**龙虎榜数据**：
- 3月18日：游资A、B、C买入，合计1.2亿
- 3月19日：游资D、E、F买入，游资A、B卖出，净买入0.5亿
- 3月20日：游资G、H买入，游资C、D卖出，净买入0.3亿
- 上榜原因：连续涨幅偏离（3日累计涨25%）

**分析**：
- 连续3日上榜，每日不同游资
- 前日买入的游资次日卖出（接力）
- 净买入逐日递减（接力减弱）
- **结论**：游资接力，短期可能继续上涨，但风险极高，建议谨慎或回避

### 示例3：机构出货

**输入**：2024-04-10，某股票上榜
**龙虎榜数据**：
- 上榜原因：换手率达20%（当日涨5%）
- 买入席位：游资2席 + 普通营业部3席，合计买入3.5亿
- 卖出席位：机构专用3席，合计卖出4.2亿
- 机构净卖出：-4.2亿
- 60日涨幅：+65%

**分析**：
- 机构集中卖出（3席），游资和散户买入
- 前期大幅上涨（+65%）
- 机构净卖出 > 游资净买入
- **结论**：机构出货，游资接盘，短期可能见顶，建议减仓或卖出
