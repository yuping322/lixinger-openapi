# 方法论：周度市场简报生成器（A股）

本技能用于**生成A股市场的周度简报，总结市场表现、热点板块、资金流向和下周展望**。周度简报是投资者了解市场动态的重要工具。

## 数据口径

### 数据源与字段映射

周度简报生成需要以下数据：
- **指数数据**：AKShare (`stock_zh_index_daily`)
  - 主要指数周涨跌幅、成交额
  - 日频数据，周度汇总
- **板块数据**：AKShare (`stock_board_industry_name_em`)
  - 行业板块周涨跌幅、资金流向
  - 日频数据，周度汇总
- **个股数据**：AKShare (`stock_zh_a_spot_em`)
  - 涨跌幅榜、成交额榜、换手率榜
  - 日频数据，周度汇总
- **资金流向数据**：AKShare (`stock_individual_fund_flow`)
  - 北向资金、主力资金流向
  - 日频数据，周度汇总
- **市场情绪数据**：AKShare (`stock_zt_pool_em`)
  - 涨跌停数量、新高新低数量
  - 日频数据，周度汇总

关键字段：
- `周涨跌幅 = (周五收盘价 - 周一开盘价) / 周一开盘价`
- `周成交额 = Σ(日成交额)`
- `周资金流向 = Σ(日资金流向)`

### 时间窗口与频率

- **周度汇总**：每周五收盘后生成
- **对比周期**：与上周、上月、年初至今对比
- **历史分位**：与历史数据对比分位数

## 核心指标

### 指标列表与计算公式

1. **市场表现指标**：
   - `周涨跌幅 = (周五收盘 - 周一开盘) / 周一开盘 × 100%` (%)
   - `周成交额 = Σ(日成交额)` (亿元)
   - `周换手率 = 周成交额 / 流通市值 × 100%` (%)
   - `周振幅 = (周最高 - 周最低) / 周一开盘 × 100%` (%)

2. **板块表现指标**：
   - `板块周涨跌幅` (%)
   - `板块周成交额` (亿元)
   - `板块周资金流向` (亿元)
   - `板块周涨幅排名` (排名)

3. **个股表现指标**：
   - `周涨幅前10` (股票列表)
   - `周跌幅前10` (股票列表)
   - `周成交额前10` (股票列表)
   - `周换手率前10` (股票列表)

4. **资金流向指标**：
   - `北向资金周净流入 = Σ(日净流入)` (亿元)
   - `主力资金周净流入 = Σ(日净流入)` (亿元)
   - `融资余额变化 = 周五融资余额 - 周一融资余额` (亿元)
   - `资金流向板块分布` (板块列表)

5. **市场情绪指标**：
   - `周涨停股数 = Σ(日涨停股数)` (只)
   - `周跌停股数 = Σ(日跌停股数)` (只)
   - `周新高股数 = 周五创新高股数` (只)
   - `周新低股数 = 周五创新低股数` (只)

### 标准化方法

- 使用**周度汇总**统计市场表现
- 使用**排名**识别热点板块和个股
- 使用**同比环比**对比历史表现
- 使用**分位数**评估市场位置

Rule 1 (连续上涨 + 成交放大 → 趋势延续):

THEN {下周继续上涨概率 >= 62%，趋势延续}

Rule 2 (板块轮动加速 → 结构性行情):
IF {周涨幅前5板块涨幅 >= 5% AND 周跌幅前5板块跌幅 <= -3% AND 板块分化度 >= 8%}
THEN {下周板块轮动继续，结构性行情}
CONFIDENCE {0.58; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，板块轮动}
FAILURE_MODE {市场风格转换（普涨或普跌）；政策利好（板块共振）；市场情绪逆转}

Rule 3 (成交萎缩 + 窄幅震荡 → 变盘在即):
IF {周成交额 < 上周 × 0.8 AND 周振幅 < 3% AND 连续震荡周数 >= 3周}
THEN {下周可能变盘，方向待定}
CONFIDENCE {0.55; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，震荡行情}
FAILURE_MODE {震荡延续（继续缩量）；外部因素（突发事件）；政策催化（方向明确）}

Rule 4 (北向资金大幅流出 → 短期承压):
IF {北向资金周净流出 >= 100亿 AND 连续流出周数 >= 2周 AND 主力资金同步流出}
THEN {下周市场承压，可能回调}
CONFIDENCE {0.60; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，资金面}
FAILURE_MODE {政策利好（资金回流）；市场超跌（抄底资金）；外资流出是技术性调整}

### 触发/解除/无效化条件

- **触发生成**：
  - 每周五收盘后自动生成
  - 用户手动触发生成

- **内容要素**：
  - 市场表现（指数涨跌、成交额）
  - 热点板块（涨幅榜、资金流向）
  - 个股表现（涨跌幅榜、成交额榜）
  - 资金流向（北向资金、主力资金）
  - 市场情绪（涨跌停、新高新低）
  - 下周展望（趋势判断、关注点）

- **无效化**：
  - 数据缺失（无法生成完整简报）
  - 市场异常（停市、熔断）

### 阈值选择依据

- **3周连续上涨**：连续上涨3周以上说明趋势形成
- **20%成交额增长**：成交额增长20%以上说明市场活跃
- **5%板块涨幅**：板块周涨幅 >= 5%属于强势板块
- **100亿北向资金流出**：北向资金周净流出 >= 100亿属于大幅流出
- **3%周振幅**：周振幅 < 3%说明市场窄幅震荡

## 边界条件与降级策略

### 数据缺失/异常值处理

- **数据缺失**：
  - 部分数据缺失：使用可用数据生成，标注"数据不完整"
  - 关键数据缺失：无法生成简报，提示用户
  - 历史数据缺失：无法对比，标注"无历史对比"

- **异常值处理**：
  - 周涨跌幅 > 20%或 < -20%：检查是否极端行情
  - 周成交额异常：检查是否数据错误
  - 资金流向异常：检查是否计算错误

### 降级策略

- **数据不完整**（缺失数据 > 20%）：
  - 使用可用数据生成
  - 标注"数据不完整，结论可能有偏"
  - 降级标注："部分数据缺失"

- **市场异常**（停牌股 > 30%）：
  - 剔除停牌股统计
  - 标注"剔除停牌股"
  - 降级标注："停牌股多，统计可能失真"

- **历史数据不足**（< 1年）：
  - 无法计算历史分位数
  - 标注"历史数据不足"
  - 降级标注："无法进行历史对比"

## A股特殊注意

### 1. T+1交易制度影响

- **周五买入周一卖出**：周五买入需周一才能卖出
- **周末风险**：周末可能有政策或事件影响
- **需要关注周末消息**：周度简报需关注周末消息面

### 2. 涨跌停限制影响

- **涨跌停影响统计**：涨跌停股票影响涨跌幅统计
- **连续涨跌停**：连续涨跌停股票周涨跌幅可能很大
- **需要标注涨跌停**：简报中标注涨跌停股票

### 3. 停牌影响

- **停牌股不参与**：停牌股票不参与统计
- **复牌股影响**：复牌股票可能大幅波动
- **需要剔除停牌股**：统计时剔除停牌股

### 4. 板块轮动

- **A股板块轮动快**：A股板块轮动快，热点变化快
- **需要跟踪热点**：简报需跟踪热点板块变化
- **需要分析轮动逻辑**：分析板块轮动背后的逻辑

### 5. 政策影响

- **政策市特征**：A股政策市特征明显
- **周末政策多**：周末常有政策发布
- **需要关注政策**：简报需关注政策动向

### 6. 北向资金

- **北向资金重要**：北向资金是A股重要增量资金
- **需要跟踪北向资金**：简报需跟踪北向资金流向
- **需要分析流向板块**：分析北向资金流向哪些板块

### 7. 市场情绪

- **散户情绪化**：散户情绪化交易，市场情绪波动大
- **需要跟踪情绪指标**：简报需跟踪涨跌停、新高新低等情绪指标
- **需要判断情绪拐点**：判断市场情绪是否到达拐点

### 8. 新股影响

- **新股上市影响**：新股上市影响市场资金和情绪
- **需要关注新股**：简报需关注新股上市情况
- **需要分析新股表现**：分析新股表现反映市场情绪

### 9. 季节性因素

- **季节性规律**：A股有季节性规律（如春季躁动、五穷六绝）
- **需要关注季节性**：简报需关注季节性因素
- **需要历史对比**：对比历史同期表现

### 10. 外部环境

- **外部环境影响**：外部环境（美股、港股、汇率）影响A股
- **需要关注外部环境**：简报需关注外部环境变化
- **需要分析联动性**：分析A股与外部市场联动性

## 回测说明（最小化）

- **回测设计**：
  - 生成历史周度简报
  - 验证简报中的判断
  - 统计判断准确率

- **性能指标**：
  - 趋势判断准确率
  - 热点板块预测准确率
  - 下周展望准确率

- **证伪条件**：
  - 判断准确率 < 55%（不优于随机）
  - 热点预测准确率 < 50%

## 使用示例

### 示例1：趋势延续

**输入**：2024-03-15周度数据
**数据**：
- 连续上涨周数：4周
- 周成交额增长：+25%
- 北向资金周净流入：+85亿
- 主要指数周涨幅：+3.2%
**输出**：
- 市场表现：连续上涨，成交放大
- 资金流向：北向资金持续流入
- 下周展望：趋势延续，继续看好
- **结论**：市场趋势向好，下周继续上涨概率高

### 示例2：板块轮动

**输入**：2024-04-20周度数据
**数据**：
- 周涨幅前5板块：新能源+6.5%，医药+5.8%，消费+5.2%
- 周跌幅前5板块：地产-4.2%，金融-3.8%，周期-3.5%
- 板块分化度：9.2%
**输出**：
- 市场表现：结构性行情，板块分化
- 热点板块：新能源、医药、消费
- 下周展望：板块轮动继续，关注轮动节奏
- **结论**：结构性行情，需要精选板块和个股

### 示例3：变盘在即

**输入**：2024-05-10周度数据
**数据**：
- 周成交额：较上周-22%
- 周振幅：2.5%
- 连续震荡周数：4周
- 北向资金：小幅流出-15亿
**输出**：
- 市场表现：缩量震荡，方向不明
- 资金流向：资金观望，小幅流出
- 下周展望：变盘在即，关注方向选择
- **结论**：市场缩量震荡，变盘在即，方向待定

### 示例4：资金流出承压

**输入**：2024-06-15周度数据
**数据**：
- 北向资金周净流出：-125亿（连续2周）
- 主力资金周净流出：-85亿
- 主要指数周跌幅：-2.8%
- 市场情绪：跌停股数增加
**输出**：
- 市场表现：下跌，资金流出
- 资金流向：北向资金和主力资金同步流出
- 下周展望：市场承压，可能继续回调
- **结论**：资金面承压，下周市场可能继续调整
