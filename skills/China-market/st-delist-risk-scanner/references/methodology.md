# 方法论：ST退市风险扫描器（A股）

本技能用于**识别A股上市公司被ST（特别处理）或退市的风险，监控财务指标、违规行为、交易异常等风险因素**。ST和退市是A股特有的风险机制，一旦触发将导致股价大幅下跌、流动性枯竭，是投资决策的重要风险指标。

## 数据口径

### 数据源与字段映射

ST退市风险扫描需要以下数据：
- **财务数据**：AKShare (`stock_financial_abstract_em`), 同花顺、Wind
  - 净利润、扣非净利润、营业收入、净资产、审计意见
  - 季度/年度更新（财报披露时）
- **ST状态数据**：AKShare (`stock_info_a_code_name`), 交易所公告
  - 股票代码、股票名称、ST标识、退市风险警示标识
  - 实时更新
- **违规处罚数据**：巨潮资讯、交易所公告
  - 证监会处罚、交易所处罚、立案调查
  - 不定期更新（公告披露时）
- **交易数据**：AKShare (`stock_zh_a_hist`)
  - 收盘价、成交量、成交额、换手率
  - 日频数据，实时更新
- **审计数据**：公司年报
  - 审计意见类型（标准无保留、保留、无法表示、否定）
  - 年度更新

关键字段：
- `净利润`：归属于母公司股东的净利润（亿元）
- `扣非净利润`：扣除非经常性损益后的净利润（亿元）
- `净资产`：归属于母公司股东的净资产（亿元）
- `营业收入`：营业总收入（亿元）
- `审计意见`：标准无保留/保留/无法表示/否定
- `连续亏损年数`：连续净利润为负的年数
- `净资产为负`：净资产 < 0（资不抵债）

### 时间窗口与频率

- **季度监控**：每季度财报披露后（4月、8月、10月、次年4月）更新风险评估
- **年度深度分析**：每年年报披露后（次年4月）进行全面风险扫描
- **实时监控**：监控违规处罚公告、立案调查公告、审计意见公告

建议窗口：
- ST风险：最近2年净利润、最近1年净资产、最近1年审计意见
- 退市风险：最近3年净利润、最近1年净资产、最近1年营业收入
- 预警期：年报披露前3个月（1-3月）、三季报披露后（10-12月）

## 核心指标

### 指标列表与计算公式

1. **ST风险指标**：
   - `连续亏损年数 = count(年度净利润 < 0)` over 最近N年
   - `净资产为负 = 1{净资产 < 0}`
   - `非标审计意见 = 1{审计意见 != 标准无保留}`
   - `ST风险评分 = f(连续亏损年数, 净资产, 审计意见)` (0-100)

2. **退市风险指标**：
   - `连续3年亏损 = 1{连续亏损年数 >= 3}`
   - `连续2年净资产为负 = 1{连续2年净资产 < 0}`
   - `营业收入 < 1亿 = 1{年度营业收入 < 1亿}`
   - `退市风险评分 = f(连续亏损, 净资产, 营业收入, 违规)` (0-100)

3. **财务恶化指标**：
   - `净利润恶化率 = (本期净利润 - 上期净利润) / |上期净利润|` (%)
   - `扣非净利润占比 = 扣非净利润 / 净利润` (%)
   - `净资产收益率 = 净利润 / 净资产` (%)
   - `营收下滑率 = (本期营收 - 上期营收) / 上期营收` (%)

4. **违规风险指标**：
   - `违规次数 = 近3年证监会处罚 + 交易所处罚次数`
   - `立案调查 = 1{存在未结案的立案调查}`
   - `信息披露违规 = 1{存在信息披露违规处罚}`
   - `财务造假风险 = 1{审计意见为否定或无法表示}`

5. **交易异常指标**：
   - `连续跌停天数 = 连续跌停的交易日数`
   - `成交额萎缩 = 1{日均成交额 < 1000万}` over 20日
   - `换手率极低 = 1{日均换手率 < 0.5%}` over 20日
   - `股价 < 1元 = 1{收盘价 < 1元}` (面值退市风险)

### 标准化方法

- 使用**风险评分**（0-100）量化ST/退市风险（0=无风险，100=极高风险）
- 使用**等级分类**（低/中/高/极高）对风险进行分级
- 使用**二元标志**（0/1）标识是否触发具体风险条件
- 使用**行业相对**比较财务指标（个股 vs 行业中位数）

## 信号与阈值

### 可测试规则（Insight Rules）

Rule 1 (连续亏损 + 净资产接近为负 → ST风险):
IF {连续亏损年数 >= 2 AND 净资产 > 0 BUT < 1亿 AND 本年预计继续亏损}
THEN {未来3-6个月，公司可能被ST，股价预期下跌 >= 20%，流动性枯竭}
CONFIDENCE {0.75; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，尤其是中小市值股票（< 50亿），ST风险更显著}
FAILURE_MODE {公司扭亏为盈（资产出售、政府补助）；重组成功（引入战略投资者）；会计调整（调增利润）；监管豁免（特殊情况）}

Rule 2 (连续3年亏损 → 退市风险):
IF {连续亏损年数 >= 3 AND 营业收入 < 1亿 AND 无重组预期}
THEN {未来6-12个月，公司面临退市风险，股价预期下跌 >= 50%，建议清仓}
CONFIDENCE {0.82; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，*ST股票（已被退市风险警示）}
FAILURE_MODE {重组成功（借壳上市）；扭亏为盈（业绩反转）；监管政策调整（放宽退市标准）}

Rule 3 (非标审计意见 + 财务恶化 → 高风险):
IF {审计意见 = 保留/无法表示/否定 AND 净利润恶化率 <= -50% AND 扣非净利润 < 0}
THEN {未来6-12个月，公司可能被ST或退市，股价预期下跌 >= 30%}
CONFIDENCE {0.70; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，审计意见是重要的风险信号}
FAILURE_MODE {审计意见改善（更换审计师）；财务状况改善（业绩反转）；重组成功}

Rule 4 (股价 < 1元 + 成交萎缩 → 面值退市风险):
IF {收盘价 < 1元 AND 连续20日收盘价 < 1元 AND 日均成交额 < 1000万}
THEN {未来1-3个月，公司面临面值退市风险，股价可能继续下跌或停牌}
CONFIDENCE {0.85; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，2020年后面值退市成为重要退市方式}
FAILURE_MODE {股价反弹至1元以上（市场炒作、重组预期）；公司回购托底；停牌规避（筹划重组）}

### 触发/解除/无效化条件

- **触发监控**：
  - 连续亏损年数 >= 2（ST风险）
  - 净资产 < 1亿或为负（ST/退市风险）
  - 审计意见为非标（保留/无法表示/否定）
  - 营业收入 < 1亿（退市风险）
  - 股价 < 1元连续10日（面值退市风险）
  - 证监会立案调查或重大违规处罚

- **解除监控**：
  - 扭亏为盈（年度净利润 > 0）
  - 净资产转正且 > 2亿
  - 审计意见恢复标准无保留
  - 营业收入 > 2亿
  - 股价稳定在1元以上连续20日
  - 重组成功（新实控人入主、业务转型）

- **无效化**：
  - 公司已被ST或*ST（风险已实现）
  - 公司已进入退市程序
  - 公司停牌（重组、破产重整）
  - 公司被借壳（壳资源价值）

### 阈值选择依据

- **连续2年亏损**：触发ST的法定条件（《股票上市规则》）
- **连续3年亏损**：触发退市的法定条件（2020年后）
- **净资产为负**：触发ST的法定条件
- **连续2年净资产为负**：触发退市的法定条件
- **营业收入 < 1亿**：触发退市的法定条件（2020年后新增）
- **股价 < 1元连续20日**：触发面值退市的法定条件（2020年后）
- **1000万日均成交额**：流动性枯竭的经验阈值

## 边界条件与降级策略

### 数据缺失/异常值处理

- **数据缺失**：
  - 财报未披露：使用业绩预告或业绩快报，标注"预估数据"
  - 审计意见未披露：使用上一年度审计意见，标注"使用历史数据"
  - 违规数据缺失：手动查询巨潮资讯、交易所网站

- **异常值处理**：
  - 净利润突然转正（> 1亿）：检查是否有非经常性损益（资产出售、政府补助），若是则标注"扣非后仍亏损"
  - 净资产突然转正：检查是否有债务重组、资产注入，若是则标注"需关注可持续性"
  - 营业收入突然暴增：检查是否有业务并购、会计调整，若是则标注"需关注真实性"

### 降级策略

- **AKShare接口不稳定**：
  - 备用数据源1：同花顺iFinD
  - 备用数据源2：Wind金融终端
  - 备用数据源3：手动查询公司公告（巨潮资讯）
  - 降级标注："使用备用数据源，可能有延迟"

- **财报数据延迟**：
  - 使用业绩预告（通常提前1-2个月）
  - 使用业绩快报（通常提前2-4周）
  - 使用券商研报预测
  - 降级标注："使用预估数据，以正式财报为准"

- **审计意见未披露**：
  - 使用上一年度审计意见
  - 关注审计师变更（可能是风险信号）
  - 关注审计费用变化（大幅下降可能是风险信号）
  - 降级标注："审计意见未披露，使用历史数据"

## A股特殊注意

### 1. ST制度演变

- **ST标准**（特别处理）：
  - 连续2年亏损 → 被ST
  - 最近1年净资产为负 → 被ST
  - 最近1年审计意见为否定或无法表示 → 被ST
  - 财务会计报告存在重大会计差错或虚假记载 → 被ST

- **\*ST标准**（退市风险警示）：
  - 连续3年亏损 → 被\*ST
  - 连续2年净资产为负 → 被\*ST
  - 最近1年营业收入 < 1亿 → 被\*ST（2020年后新增）
  - 审计意见为否定或无法表示 → 被\*ST

- **退市标准**（2020年后）：
  - 连续4年亏损 → 退市（旧规则）
  - 连续3年亏损 + 营业收入 < 1亿 → 退市（新规则，更严格）
  - 连续2年净资产为负 → 退市
  - 股价 < 1元连续20日 → 面值退市（新规则）
  - 重大违法 → 强制退市（欺诈发行、重大信息披露违法）

### 2. T+1交易制度影响

- **ST股票跌停后难以卖出**：ST股票涨跌幅限制为±5%，跌停后：
  - T+1才能卖出（无法当日止损）
  - 连续跌停时被锁死（流动性枯竭）
  - 退市前可能连续跌停（损失惨重）

### 3. 涨跌停限制影响

- **ST股票涨跌幅±5%**：比普通股票（±10%）更严格：
  - 下跌时损失相对较小（单日最多-5%）
  - 但连续跌停时累计损失巨大
  - 退市前通常连续跌停（10-20个跌停）

- **\*ST股票涨跌幅±5%**：与ST相同：
  - 退市风险更高，流动性更差
  - 摘帽预期时可能连续涨停（炒作）

### 4. 停牌影响

- **ST前停牌**：公司被ST前通常停牌1-2天：
  - 无法提前卖出（被动持有）
  - 复牌后通常跌停（积累的卖压）

- **退市前停牌**：公司退市前可能长期停牌：
  - 重组、破产重整（数月甚至数年）
  - 停牌期间无法交易（流动性归零）
  - 复牌后可能直接退市（损失100%）

### 5. 摘帽炒作

- **ST摘帽**：公司扭亏为盈后可申请摘帽（撤销ST）：
  - 摘帽前通常有炒作（预期驱动）
  - 摘帽后可能大涨（涨跌幅恢复±10%）
  - 但需警惕"假摘帽"（非经常性损益、会计调整）

- **摘帽条件**：
  - 最近1年净利润 > 0
  - 最近1年净资产 > 0
  - 最近1年审计意见为标准无保留
  - 无其他异常情况

### 6. 保壳行为

- **年末突击保壳**：公司为避免ST/退市，常见保壳手段：
  - 出售资产（变卖子公司、房产）
  - 政府补助（地方政府支持）
  - 债务重组（债转股、债务豁免）
  - 会计调整（调增利润、调减费用）
  - 需要识别保壳行为，判断可持续性

### 7. 重组预期

- **ST股重组**：ST股票常成为"壳资源"，吸引重组方：
  - 借壳上市（注入优质资产）
  - 业务转型（剥离亏损业务）
  - 重组成功后股价可能暴涨（数倍）
  - 但重组失败风险高（继续退市）

### 8. 退市整理期

- **退市整理期交易**：公司退市前有30个交易日的整理期：
  - 股票代码前加"退市"标识
  - 涨跌幅限制±10%（不再是±5%）
  - 流动性极差（日均成交额可能 < 100万）
  - 整理期结束后进入老三板或彻底退市

### 9. 老三板交易

- **退市后进入老三板**：部分退市公司进入老三板（代办股份转让系统）：
  - 交易频率低（每周1次或3次或5次）
  - 流动性极差（可能数月无成交）
  - 股价极低（可能 < 0.1元）
  - 投资者难以退出（事实上的损失100%）

### 10. 监管政策变化

- **2020年退市新规**：退市标准更严格、退市速度更快：
  - 新增营业收入 < 1亿退市标准
  - 新增面值退市标准（股价 < 1元连续20日）
  - 取消暂停上市环节（直接退市）
  - 重大违法强制退市（欺诈发行、财务造假）
  - 需要关注政策变化，及时调整风险评估

## 回测说明（最小化）

- **回测设计**：
  - 每季度财报披露后计算风险评分
  - 触发高风险信号时（ST风险评分 > 70）做空或清仓
  - 触发摘帽预期时（扭亏为盈）做多或加仓
  - 持有期：60日/120日
  - 基准：中证500、创业板指

- **性能指标**：
  - 风险识别率（成功识别ST/退市事件的比例）
  - 误报率（错误预警的比例）
  - 平均超额收益（规避风险后的超额收益）
  - 夏普比率、最大回撤

- **证伪条件**：
  - 风险识别率 < 70%（漏报过多）
  - 误报率 > 30%（错误预警过多）
  - 平均超额收益 <= 0（规避无效）

- **敏感性分析**：
  - 测试不同阈值（连续2年 vs 1年亏损）
  - 测试不同市场环境（牛市 vs 熊市）
  - 测试不同行业（周期 vs 成长）
  - 测试保壳行为识别（扣非净利润 vs 净利润）

## 使用示例

### 示例1：ST风险预警

**输入**：某制造业公司（市值20亿）
**数据**：
- 2023年净利润：-1.5亿
- 2024年净利润：-2.3亿
- 2024年净资产：0.8亿
- 2024年审计意见：保留意见
- 2025年Q1预计继续亏损

**输出**：
- ST风险评分：85（极高）
- 预计被ST时间：2025年4月（年报披露后）
- **结论**：公司面临严重ST风险，建议清仓规避，股价可能下跌20%+

### 示例2：退市风险预警

**输入**：某*ST公司（市值5亿）
**数据**：
- 2022-2024年连续3年亏损
- 2024年营业收入：0.6亿（< 1亿）
- 2024年净资产：-0.5亿（为负）
- 无重组预期

**输出**：
- 退市风险评分：95（极高）
- 预计退市时间：2025年6月
- **结论**：公司面临退市风险，建议立即清仓，避免损失100%

### 示例3：面值退市风险

**输入**：某ST公司（市值3亿）
**数据**：
- 当前股价：0.95元
- 连续15日收盘价 < 1元
- 日均成交额：800万
- 无重组预期

**输出**：
- 面值退市风险：高
- 距离退市：5个交易日（连续20日 < 1元）
- **结论**：公司面临面值退市风险，建议立即清仓或停牌前卖出

### 示例4：摘帽预期

**输入**：某ST公司（市值15亿）
**数据**：
- 2024年净利润：0.5亿（扭亏）
- 2024年扣非净利润：0.3亿（真实盈利）
- 2024年净资产：3亿
- 2024年审计意见：标准无保留

**输出**：
- 摘帽概率：90%
- 预计摘帽时间：2025年5月
- **结论**：公司有望摘帽，股价可能上涨，可考虑逢低布局（但需警惕炒作风险）
