# 方法论：政策敏感性简报（A股）

本技能用于**分析A股市场对政策的敏感性，识别政策敏感板块和个股**。A股是典型的政策市，政策对市场影响巨大。

## 数据口径

### 数据源与字段映射

政策敏感性分析需要以下数据：
- **政策数据**：政府网站、新闻、AKShare
  - 政策类型、发布时间、政策内容
  - 实时数据，政策发布后更新
- **股价数据**：AKShare (`stock_zh_a_hist`)
  - 政策前后股价变化
  - 日频数据，实时更新
- **板块数据**：AKShare (`stock_board_industry_name_em`)
  - 板块涨跌幅、资金流向
  - 日频数据，实时更新

关键字段：
- `政策类型`：货币政策、财政政策、产业政策、监管政策
- `政策影响`：利好、利空、中性
- `政策敏感度 = 政策后涨跌幅 / 市场涨跌幅`

### 时间窗口与频率

- **政策发布日**：T日
- **短期窗口**：T至T+5日
- **中期窗口**：T至T+20日

## 核心指标

### 指标列表与计算公式

1. **政策影响指标**：
   - `政策后涨跌幅 = (T+5收盘 - T开盘) / T开盘 × 100%` (%)
   - `市场涨跌幅 = (指数T+5收盘 - 指数T开盘) / 指数T开盘 × 100%` (%)
   - `超额收益 = 政策后涨跌幅 - 市场涨跌幅` (%)
   - `政策敏感度 = 超额收益 / |政策强度|` (倍)

2. **板块敏感度指标**：
   - `板块政策敏感度 = mean(板块超额收益) / std(板块超额收益)` (倍)
   - `板块政策响应速度 = 达到最大涨幅的天数` (天)
   - `板块政策持续性 = CAR[0, +20] / CAR[0, +5]` (倍)

3. **个股敏感度指标**：
   - `个股政策敏感度 = 个股超额收益 / 板块超额收益` (倍)
   - `个股政策弹性 = ∂个股涨跌幅 / ∂政策强度` (倍)

4. **政策类型指标**：
   - `货币政策敏感度`：对降息降准的敏感度
   - `财政政策敏感度`：对财政刺激的敏感度
   - `产业政策敏感度`：对产业扶持的敏感度
   - `监管政策敏感度`：对监管政策的敏感度

5. **历史敏感度指标**：
   - `历史平均敏感度 = mean(历史政策超额收益)` (%)
   - `敏感度稳定性 = 1 - std(历史敏感度) / mean(历史敏感度)` (0-1)

### 标准化方法

- 使用**超额收益**评估政策影响
- 使用**历史敏感度**识别政策敏感板块
- 使用**事件研究法**量化政策影响

THEN {政策后20个交易日，龙头股超额收益 >= 行业平均 + 5%}

FAILURE_MODE {龙头股估值过高；政策红利分散；行业竞争加剧}

### 触发/解除/无效化条件

- **触发监控**：
  - 重大政策发布
  - 政策力度 >= 7分
  - 政策影响板块明确

- **解除监控**：
  - 政策影响消化（超额收益回归0）
  - 政策执行完成
  - 新政策覆盖旧政策

- **无效化**：
  - 政策取消或延期
  - 政策执行不及预期
  - 市场异常波动

### 阈值选择依据

- **1.5倍敏感度**：板块历史敏感度 >= 1.5属于高敏感板块
- **8分政策力度**：政策力度 >= 8分属于重大政策
- **8%超额收益**：政策后超额收益 >= 8%属于显著影响
- **10bp利率下降**：利率下降 >= 10bp属于显著宽松
- **7分政策力度**：政策力度 >= 7分属于重要政策

## 边界条件与降级策略

### 数据缺失/异常值处理

- **数据缺失**：
  - 政策数据缺失：使用新闻补充，标注"手动补充"
  - 股价数据缺失：使用前后交易日插值，标注"数据插值"

- **异常值处理**：
  - 超额收益 > 50%：检查是否极端政策或数据错误
  - 政策敏感度 > 5：检查是否计算错误

### 降级策略

- **政策信息不完整**：
  - 使用部分信息分析
  - 降级标注："政策信息不完整"

- **历史数据不足**（< 3次政策）：
  - 无法计算历史敏感度
  - 降级标注："历史数据不足"

## A股特殊注意

### 1. 政策市特征

- **政策主导**：A股是典型的政策市
- **政策影响大**：政策对市场影响远大于基本面

### 2. 政策类型

- **货币政策**：降息降准、MLF、LPR
- **财政政策**：减税降费、基建投资
- **产业政策**：新能源、半导体、医药
- **监管政策**：房地产、教育、互联网

### 3. 政策传导

- **传导快**：A股对政策反应快
- **预期交易**：政策预期比政策本身更重要

### 4. 政策敏感板块

- **高敏感**：新能源、半导体、医药、基建
- **低敏感**：消费、金融

### 5. 政策周期

- **政策周期明显**：宽松-收紧-宽松
- **需要判断周期**：判断当前处于政策周期哪个阶段

### 6. 政策预期

- **预期充分**：政策预期充分时，政策落地可能利好出尽
- **预期不足**：政策超预期时，市场反应强烈

### 7. 政策执行

- **执行力度**：政策执行力度影响市场反应
- **执行进度**：政策执行进度影响持续性

### 8. 政策组合

- **政策组合拳**：多项政策同时出台，影响更大
- **政策协同**：货币+财政+产业政策协同

### 9. 政策反转

- **政策反转风险**：政策可能反转，需要关注政策基调
- **政策退出**：政策退出时市场可能回调

### 10. 地方政策

- **地方政策**：地方政策对区域板块影响大
- **需要关注地方政策**：关注地方政府政策

## 回测说明（最小化）

- **回测设计**：
  - 识别历史重大政策
  - 计算政策后超额收益
  - 统计政策敏感度

- **性能指标**：
  - 政策后平均超额收益
  - 政策敏感度
  - 政策影响持续时间

- **证伪条件**：
  - 平均超额收益不显著
  - 政策敏感度 < 1
