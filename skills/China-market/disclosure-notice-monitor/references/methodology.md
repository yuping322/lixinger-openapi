# 方法论（Draft）

本文件用于沉淀该技能的计算口径、阈值建议、边界条件与常见陷阱。后续完善时优先补齐：数据字段映射、阈值来源与样本期。

## 数据口径

- **数据源与字段映射**：
    - **A股公告**：使用 `cn/company/announcement`。关注 `title`（标题）和 `type`（公告类别，如“临时公告”、“定期报告”）。
    - **港股公告**：使用 `hk/company/announcement`。
    - **监管信息**：使用 `cn/company/regulatory-info`（获取问询函、监管函、立案调查通知）。
- **时间窗口与频率**：
    - **实时监控**：公告发布通常集中在收盘后（15:30-20:00）或午间休市。
    - **财报季**：1月/4月/8月/10月为密集披露期。

## 核心指标

- **负面事件标签**：
    - **立案调查**：关键词“立案”、“调查”、“违法违规”。
    - **违约风险**：关键词“逾期”、“诉讼”、“仲裁”、“冻结”。
    - **减持**：关键词“减持计划”、“被动减持”。
    - **质押**：关键词“补充质押”、“质押延期”（尤其关注控股股东）。

- **正面/中性事件标签**：
    - **增持/回购**：关键词“增持”、“回购”、“注销”。
    - **股权激励**：关键词“草案”、“授予”。
    - **并购重组**：关键词“收购”、“重组”、“定增”。

## 信号与阈值

- **信号定义**：
    - **黑天鹅预警**：标题包含“立案调查”或“实施退市风险警示(ST)”。此为最高优先级警报。
    - **业绩暴雷**：业绩预告大幅“修正”（下修），或年报被出具“非标”审计意见。
    - **流动性危机**：控股股东质押比例 > 80% 且发布“补充质押”公告；或涉及“债务逾期”。

- **阈值选择依据**：
    - **金额阈值**：诉讼/担保金额占净资产比例 > 10% 通常被视为重大。
    - **减持比例**：大股东减持 > 1% 或清仓式减持，对情绪面打击较大。

## 边界条件与降级策略

- **数据缺失/异常值处理**：
    - 公告内容非结构化数据，若 API 仅提供 PDF 链接，需提示用户“人工阅读详情”。本技能暂基于“标题关键词”进行初筛。
- **若某接口不稳定**：
    - 优先确保“监管信息”接口畅通，因为监管函通常意味着实质性利空。

## A股特殊注意

- **信息不对称**：
    - 虽然 A 股要求公平披露，但股价往往在公告前有异动（“内幕消息”）。若股价在利好公告前大涨，公告后可能是“利好兑现”卖点。
- **特定时间窗口**：
    - **业绩预告**：1月31日前必须披露（若预亏/扭亏/大幅变动）。这是年度最大的“雷区”和“彩票期”。
    - **年报问询**：5-6月是交易所发年报问询函的高峰期，回复不佳可能导致 ST 或暴跌。
