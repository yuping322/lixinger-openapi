# 方法论：公告披露监控（A股）

本技能用于**实时监控A股上市公司公告，识别重大事项、风险提示、业绩变动等关键信息，并评估对股价的潜在影响**。公告是A股信息披露的主要渠道，及时捕捉和解读公告是投资决策的重要环节。

## 数据口径

### 数据源与字段映射

公告披露监控需要以下数据：
- **公告数据**：AKShare (`stock_notice_report`), 巨潮资讯、东方财富
  - 公告标题、公告类型、披露时间、公告内容、附件
  - 实时更新，T日晚间可获取T日公告
- **公告分类**：根据标题和内容自动分类
  - 业绩类（业绩预告、业绩快报、年报、季报）
  - 重组类（并购重组、资产注入、股权转让）
  - 风险类（风险提示、诉讼仲裁、违规处罚）
  - 股权类（股东减持、股权质押、解禁）
  - 其他类（停牌复牌、分红派息、股东大会）
- **股价数据**：AKShare (`stock_zh_a_hist`)
  - 收盘价、涨跌幅、成交量、换手率
  - 日频数据，实时更新
- **历史公告影响**：统计历史同类公告对股价的影响
  - 公告后1日/3日/5日涨跌幅
  - 公告后成交量变化
  - 公告后市场关注度（研报、新闻）

关键字段：
- `公告标题`：公告的标题（包含关键信息）
- `公告类型`：业绩/重组/风险/股权/其他
- `披露时间`：公告披露的日期和时间
- `重要性等级`：高/中/低（根据内容判断）
- `情绪倾向`：利好/中性/利空
- `预期影响`：对股价的预期影响（涨跌幅%）

### 时间窗口与频率

- **实时监控**：每日盘后（16:00-24:00）监控新公告，重大公告实时推送
- **日频总结**：每日汇总当日公告，按重要性排序
- **周频总结**：每周汇总重要公告，分析市场热点

建议窗口：
- 公告发布后1小时内：快速解读，识别关键信息
- 公告发布后1日：观察市场反应（股价、成交量）
- 公告发布后3-5日：评估持续影响
- 公告发布后1个月：跟踪事项进展（如重组、诉讼）

## 核心指标

### 指标列表与计算公式

1. **公告重要性指标**：
   - `重要性评分 = f(公告类型, 关键词, 历史影响)` (0-100)
   - `关键词匹配度 = count(重大关键词) / 总关键词数` (%)
   - `历史平均影响 = mean(同类公告后N日涨跌幅)` (%)
   - `公告频率 = 近N日公告数量 / N` (条/日)

2. **情绪倾向指标**：
   - `利好词占比 = count(利好词) / 总词数` (%)
   - `利空词占比 = count(利空词) / 总词数` (%)
   - `情绪得分 = 利好词占比 - 利空词占比` (-1 ~ +1)
   - `情绪强度 = |情绪得分|` (0 ~ 1)

3. **市场反应指标**：
   - `公告后涨跌幅 = (T+N日收盘价 - T日收盘价) / T日收盘价` (%)
   - `公告后成交量变化 = (T+1日成交量 - T日成交量) / T日成交量` (%)
   - `公告后换手率变化 = T+1日换手率 - T日换手率` (百分点)
   - `公告后关注度 = 公告后N日研报数量 + 新闻数量`

4. **异常公告指标**：
   - `补充披露 = 1{公告标题包含"补充"、"更正"、"致歉"}`
   - `延期披露 = 1{公告标题包含"延期"、"推迟"}`
   - `风险提示频率 = 近N日风险提示公告数量`
   - `违规处罚 = 1{公告类型为证监会处罚或交易所处罚}`

5. **公告时机指标**：
   - `盘后披露 = 1{披露时间在15:00-24:00}`
   - `周五披露 = 1{披露日为周五}`
   - `节假日前披露 = 1{披露日为节假日前1-2天}`
   - `时机异常评分 = f(盘后, 周五, 节假日前)` (0-100)

### 标准化方法

- 使用**重要性评分**（0-100）量化公告重要性（0=不重要，100=极重要）
- 使用**情绪得分**（-1 ~ +1）量化公告情绪（-1=极利空，+1=极利好）
- 使用**等级分类**（高/中/低）对重要性进行分级
- 使用**历史统计**计算同类公告的平均影响

## 信号与阈值

### 可测试规则（Insight Rules）

Rule 1 (业绩大幅预增 + 盘后披露 → 次日高开):
IF {公告类型=业绩预告 AND 预增幅度 >= 50% AND 盘后披露 AND 历史业绩稳定}
THEN {次日开盘高开 >= 3%，盘中可能涨停，3日累计涨幅 >= 8%}
CONFIDENCE {0.68; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，中小市值股票（< 100亿）反应更强烈}
FAILURE_MODE {业绩预增依赖非经常性损益（不可持续）；市场整体下跌（系统性风险）；公司存在其他负面（如违规、诉讼）；业绩预增已被市场提前预期}

Rule 2 (重大重组 + 停牌 → 复牌后大涨):
IF {公告类型=重组 AND 标的资产优质 AND 停牌时间 < 3个月 AND 重组方案合理}
THEN {复牌后3日累计涨幅 >= 15%，可能连续涨停}
CONFIDENCE {0.65; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，壳资源价值高的小市值股票（< 50亿）}
FAILURE_MODE {重组方案被否决（股东大会或监管）；标的资产估值过高（利益输送）；市场整体下跌；重组后业绩不及预期}

Rule 3 (风险提示频繁 + 补充披露 → 利空):
IF {近10日风险提示公告 >= 3次 AND 存在补充披露 AND 公告时机异常（周五/节假日前）}
THEN {未来5-10日股价下跌 >= 5%，市场信心受损}
CONFIDENCE {0.72; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，频繁风险提示通常是重大问题的前兆}
FAILURE_MODE {风险提示是例行披露（如股价异常波动）；公司及时化解风险；市场整体上涨掩盖利空}

Rule 4 (大股东减持 + 高位 → 股价承压):
IF {公告类型=股东减持 AND 减持比例 >= 2% AND 股价处于历史高位（60日涨幅 >= 30%）}
THEN {未来10-20日股价下跌 >= 8%，成交量放大}
CONFIDENCE {0.70; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，大股东减持是重要的利空信号}
FAILURE_MODE {减持是被动减持（质押平仓、离婚分割）；公司基本面改善（业绩超预期）；市场整体上涨；减持后引入战略投资者}

### 触发/解除/无效化条件

- **触发监控**：
  - 业绩预告（预增/预减 >= 30%）
  - 重大重组（并购、资产注入、控制权变更）
  - 风险提示（诉讼、违规、ST风险）
  - 股东减持（减持比例 >= 1%）
  - 股权质押（质押比例 >= 50%）
  - 违规处罚（证监会、交易所）
  - 停牌复牌（重大事项）

- **解除监控**：
  - 事项完成（重组完成、诉讼结案）
  - 风险消除（业绩改善、违规整改）
  - 市场反应充分（股价已反映公告影响）

- **无效化**：
  - 公告被撤回或更正（信息错误）
  - 公司停牌（无法交易）
  - 公司退市（风险已实现）

### 阈值选择依据

- **50%业绩预增**：显著的业绩改善，市场通常有强烈反应
- **2%减持比例**：较大规模的减持，对股价有明显影响
- **3次风险提示**：频繁风险提示表明问题严重
- **3个月停牌**：停牌时间过长可能导致市场关注度下降
- **30%历史高位**：60日涨幅30%以上，减持压力更大
- **重要性评分70**：高重要性公告，需要重点关注

## 边界条件与降级策略

### 数据缺失/异常值处理

- **数据缺失**：
  - 公告内容缺失：仅使用标题分析，标注"内容未获取"
  - 公告附件缺失：无法深度分析，标注"附件未下载"
  - 历史数据缺失：无法计算历史影响，使用行业平均

- **异常值处理**：
  - 公告时间异常（凌晨披露）：标注"披露时间异常"
  - 公告标题过长（> 100字）：截取关键部分
  - 公告内容过长（> 10万字）：提取摘要和关键段落

### 降级策略

- **AKShare接口不稳定**：
  - 备用数据源1：巨潮资讯网页爬取（`http://www.cninfo.com.cn/`）
  - 备用数据源2：东方财富公告查询（`http://data.eastmoney.com/notices/`）
  - 备用数据源3：交易所官网（上交所、深交所）
  - 降级标注："使用备用数据源，可能有延迟"

- **公告内容解析失败**：
  - 使用标题关键词分析
  - 使用公告类型（业绩/重组/风险等）
  - 人工复核重要公告
  - 降级标注："内容解析失败，仅基于标题分析"

- **情绪分析失败**：
  - 使用公告类型默认情绪（业绩预增=利好，减持=利空）
  - 使用历史同类公告平均情绪
  - 降级标注："情绪分析失败，使用默认值"

## A股特殊注意

### 1. T+1交易制度影响

- **盘后公告次日才能交易**：公告在盘后披露（15:00-24:00），投资者次日才能交易：
  - 利好公告：次日高开，盘中可能涨停
  - 利空公告：次日低开，盘中可能跌停
  - 无法当日反应（信息滞后）

### 2. 涨跌停限制影响

- **涨停限制买入**：利好公告后股价涨停，投资者无法买入：
  - 可能连续涨停（追不上）
  - 涨停打开后可能已大涨（追高风险）

- **跌停限制卖出**：利空公告后股价跌停，投资者无法卖出：
  - 可能连续跌停（卖不出）
  - 损失扩大（无法止损）

### 3. 停牌影响

- **重大事项停牌**：公司披露重大事项（重组、控制权变更）时停牌：
  - 停牌期间无法交易（流动性归零）
  - 复牌后可能大涨或大跌（不确定性）
  - 停牌时间可能很长（数月甚至数年）

- **临时停牌**：股价异常波动时临时停牌（通常1小时）：
  - 停牌后披露澄清公告或风险提示
  - 复牌后可能继续波动

### 4. 公告披露时间规律

- **盘后披露**：大部分公告在盘后披露（15:00-24:00）：
  - 避免盘中剧烈波动
  - 给投资者消化时间
  - 次日开盘反应

- **周五披露**：部分利空公告在周五披露：
  - 周末消化（降低冲击）
  - 周一开盘反应可能减弱
  - 需要警惕"周五埋雷"

- **节假日前披露**：部分利空公告在节假日前披露：
  - 假期消化（降低冲击）
  - 复盘后反应可能减弱
  - 需要警惕"假期埋雷"

### 5. 公告类型特点

- **业绩预告**：提前1-2个月披露业绩预期：
  - 预增：利好，股价通常上涨
  - 预减：利空，股价通常下跌
  - 需要关注扣非净利润（剔除非经常性损益）

- **业绩快报**：提前2-4周披露业绩快报：
  - 比业绩预告更准确
  - 市场反应更强烈

- **年报/季报**：正式财报：
  - 最准确的业绩数据
  - 可能与业绩预告/快报有差异（需要关注）

### 6. 补充披露与更正

- **补充披露**：公告披露后补充信息：
  - 可能是遗漏重要信息（风险信号）
  - 可能是监管要求（信息披露不充分）
  - 需要重点关注补充内容

- **更正公告**：公告披露后更正错误：
  - 可能是数据错误（计算失误）
  - 可能是信息披露违规（故意隐瞒）
  - 需要评估更正影响

### 7. 信息披露违规

- **延期披露**：财报延期披露：
  - 可能是审计问题（财务风险）
  - 可能是内部控制问题
  - 通常是利空信号

- **虚假披露**：信息披露虚假、误导：
  - 证监会处罚（罚款、警告）
  - 股价大跌（信任危机）
  - 可能被ST或退市

### 8. 公告与股价背离

- **利好公告股价下跌**：
  - 利好已被提前炒作（兑现利空）
  - 利好不及预期（失望）
  - 市场整体下跌（系统性风险）

- **利空公告股价上涨**：
  - 利空已被提前消化（利空出尽）
  - 利空不及预期（松一口气）
  - 市场整体上涨（掩盖利空）

### 9. 公告与市场情绪

- **牛市**：利好公告反应强烈，利空公告反应弱：
  - 业绩预增可能连续涨停
  - 减持公告可能被忽视

- **熊市**：利空公告反应强烈，利好公告反应弱：
  - 风险提示可能连续跌停
  - 业绩预增可能被忽视

### 10. 公告与监管政策

- **监管趋严**：2020年后信息披露监管趋严：
  - 延期披露处罚更严
  - 虚假披露可能强制退市
  - 需要关注监管政策变化

- **公告格式规范**：交易所要求公告格式规范：
  - 标题必须包含关键信息
  - 内容必须完整、准确
  - 附件必须齐全

## 回测说明（最小化）

- **回测设计**：
  - 每日盘后监控公告
  - 根据公告类型和内容生成交易信号
  - 次日开盘买入/卖出
  - 持有期：1日/3日/5日
  - 基准：沪深300、中证500

- **性能指标**：
  - 信号胜率（公告后股价符合预期的概率）
  - 平均超额收益（公告后N日收益 - 基准收益）
  - 信息捕捉率（成功捕捉重要公告的比例）
  - 夏普比率、最大回撤

- **证伪条件**：
  - 信号胜率 <= 55%（不显著优于随机）
  - 平均超额收益 <= 0（不优于基准）
  - 信息捕捉率 < 70%（漏报过多）

- **敏感性分析**：
  - 测试不同公告类型（业绩 vs 重组 vs 风险）
  - 测试不同市场环境（牛市 vs 熊市）
  - 测试不同市值（大盘 vs 中小盘）
  - 测试不同披露时间（盘后 vs 周五 vs 节假日前）

## 使用示例

### 示例1：业绩预增

**输入**：某科技公司（市值60亿）
**公告**：2024年度业绩预告，预计净利润5-6亿，同比增长80-100%
**披露时间**：2025年1月15日 20:00（盘后）

**输出**：
- 公告类型：业绩预告
- 重要性评分：85（高）
- 情绪得分：+0.8（利好）
- 预期影响：次日高开5%+，3日累计涨幅10%+
- **结论**：重大利好，建议次日开盘买入，关注是否涨停

### 示例2：大股东减持

**输入**：某消费公司（市值150亿）
**公告**：控股股东计划减持3%股份，减持价格不低于当前价
**披露时间**：2025年2月7日 21:00（周五盘后）

**输出**：
- 公告类型：股东减持
- 重要性评分：75（高）
- 情绪得分：-0.6（利空）
- 时机异常评分：70（周五披露）
- 预期影响：次周一低开3%+，10日累计跌幅8%+
- **结论**：利空，建议减仓或观望，警惕周一低开

### 示例3：风险提示

**输入**：某制造业公司（市值30亿）
**公告**：风险提示公告（第3次），公司涉及重大诉讼，可能赔偿2亿
**披露时间**：2025年3月20日 22:00（盘后）

**输出**：
- 公告类型：风险提示
- 重要性评分：80（高）
- 情绪得分：-0.7（利空）
- 风险提示频率：3次/10日（频繁）
- 预期影响：次日低开5%+，可能跌停，5日累计跌幅10%+
- **结论**：严重利空，建议清仓规避，关注后续进展

### 示例4：重组预案

**输入**：某ST公司（市值10亿）
**公告**：重大资产重组预案，拟注入优质资产，预计增厚EPS 50%
**披露时间**：2025年4月10日（复牌）

**输出**：
- 公告类型：重组
- 重要性评分：90（极高）
- 情绪得分：+0.9（极利好）
- 预期影响：复牌后连续涨停，3日累计涨幅20%+
- **结论**：重大利好，建议复牌后买入，关注重组进展和股东大会表决
