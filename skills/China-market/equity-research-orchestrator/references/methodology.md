# 方法论：股票研究编排器（A股）

本技能用于**编排和协调股票研究流程，整合基本面分析、技术面分析、估值分析、风险分析等多个维度，生成全面的股票研究报告**。股票研究是投资决策的核心环节，需要系统化、结构化的分析框架。

## 数据口径

### 数据源与字段映射

股票研究编排需要以下数据：
- **基本面数据**：AKShare (`stock_financial_abstract_em`), Wind、同花顺
  - 财务报表（资产负债表、利润表、现金流量表）
  - 财务指标（ROE、毛利率、净利率、资产负债率）
  - 季度/年度更新（财报披露时）
- **估值数据**：AKShare (`stock_a_lg_indicator`)
  - PE、PB、PS、EV/EBITDA、股息率
  - 日频更新
- **技术面数据**：AKShare (`stock_zh_a_hist`)
  - 股价、成交量、换手率、涨跌幅
  - 日频更新
- **行业数据**：AKShare (`stock_board_industry_name_em`)
  - 行业分类、行业排名、行业估值
  - 日频更新
- **公司公告**：AKShare (`stock_notice_report`)
  - 业绩预告、重组公告、风险提示
  - 实时更新
- **券商研报**：Wind、同花顺
  - 评级（买入/增持/中性/减持/卖出）
  - 目标价、盈利预测
  - 不定期更新

关键字段：
- `营业收入`、`净利润`、`扣非净利润`、`经营现金流`
- `ROE`、`毛利率`、`净利率`、`资产负债率`
- `PE`、`PB`、`PS`、`股息率`
- `股价`、`涨跌幅`、`成交量`、`换手率`
- `行业排名`、`行业PE中位数`
- `券商评级`、`目标价`、`一致预期EPS`

### 时间窗口与频率

- **季度深度研究**：每季度财报披露后（4月、8月、10月、次年4月）
- **月度跟踪**：每月更新股价、估值、行业排名
- **周度监控**：每周关注公告、研报、股价异动
- **日度监控**：每日关注股价、成交量、资金流向

建议窗口：
- 财务分析：最近4个季度、最近3年年报
- 估值分析：当前估值 vs 历史估值（5年）、行业估值
- 技术分析：20日、60日、120日、252日均线和趋势
- 盈利预测：未来2-3年EPS、营收、净利润增长率

## 核心指标

### 指标列表与计算公式

1. **盈利能力指标**：
   - `ROE = 净利润 / 净资产` (%)
   - `毛利率 = (营业收入 - 营业成本) / 营业收入` (%)
   - `净利率 = 净利润 / 营业收入` (%)
   - `ROIC = EBIT × (1 - 税率) / (净资产 + 有息负债)` (%)

2. **成长性指标**：
   - `营收增长率 = (本期营收 - 上期营收) / 上期营收` (%)
   - `净利润增长率 = (本期净利润 - 上期净利润) / 上期净利润` (%)
   - `扣非净利润增长率`：剔除非经常性损益后的增长率 (%)
   - `3年复合增长率CAGR = (最新值 / 3年前值)^(1/3) - 1` (%)

3. **财务质量指标**：
   - `资产负债率 = 总负债 / 总资产` (%)
   - `流动比率 = 流动资产 / 流动负债`
   - `经营现金流 / 净利润`：> 1为优质（现金流好）
   - `应收账款周转天数 = 应收账款 / (营收 / 365)`（天）

4. **估值指标**：
   - `PE = 股价 / EPS`（市盈率）
   - `PB = 股价 / 每股净资产`（市净率）
   - `PS = 市值 / 营业收入`（市销率）
   - `PEG = PE / 净利润增长率`（< 1为低估）
   - `股息率 = 每股股息 / 股价` (%)

5. **相对估值指标**：
   - `PE百分位 = percentile_rank(当前PE, 5年历史)` (%)
   - `PB百分位 = percentile_rank(当前PB, 5年历史)` (%)
   - `相对PE = 个股PE / 行业PE中位数`
   - `相对PB = 个股PB / 行业PB中位数`

6. **技术面指标**：
   - `20日/60日/120日均线`：趋势判断
   - `相对强度RS = 个股涨跌幅 - 基准指数涨跌幅` (百分点)
   - `换手率`：流动性和活跃度 (%)
   - `资金流向`：主力净流入/流出（亿元）

### 标准化方法

- 使用**百分位排名**对估值进行标准化（相对历史和行业）
- 使用**z-score**检测财务指标异常（> 2倍标准差）
- 使用**相对比较**（个股 vs 行业 vs 市场）
- 使用**评分**（0-100）综合评估（基本面、估值、技术面）

## 信号与阈值

### 可测试规则（Insight Rules）

Rule 1 (高ROE + 低估值 + 成长 → 优质成长股):
IF {ROE >= 15% AND PE < 行业PE中位数 AND 净利润增长率 >= 20% AND 经营现金流/净利润 > 0.8}
THEN {未来6-12个月，股价上涨概率 >= 65%，预期涨幅 >= 20%}
CONFIDENCE {0.65; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，尤其是消费、医药、科技等成长性行业}
FAILURE_MODE {行业景气度下降；公司业绩不及预期；估值中枢下移（市场风格切换）；竞争加剧（市场份额下降）}

Rule 2 (低PB + 高股息 + 稳定盈利 → 价值股):
IF {PB < 1.5 AND 股息率 >= 3% AND ROE >= 10% AND 净利润增长率 >= 0% AND 资产负债率 < 60%}
THEN {未来6-12个月，股价稳健上涨概率 >= 60%，预期涨幅 >= 10%，下跌风险有限}
CONFIDENCE {0.60; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，尤其是银行、公用事业、高速公路等价值股}
FAILURE_MODE {利率上升（债券收益率上升，股息吸引力下降）；行业衰退；公司基本面恶化；市场风格偏向成长（价值股被忽视）}

Rule 3 (业绩预增 + 估值合理 + 技术突破 → 短期机会):
IF {业绩预增 >= 30% AND PE百分位 < 60 AND 股价突破60日均线 AND 成交量放大 > 50%}
THEN {未来1-3个月，股价上涨概率 >= 68%，预期涨幅 >= 15%}
CONFIDENCE {0.68; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，业绩预增是重要的催化剂}
FAILURE_MODE {业绩预增依赖非经常性损益；市场整体下跌；技术突破失败（假突破）；业绩预增已被提前炒作}

Rule 4 (财务恶化 + 高估值 + 技术破位 → 规避):
IF {净利润增长率 < -20% AND PE > 行业PE中位数 × 1.5 AND 股价跌破60日均线 AND 主力净流出 > 1亿}
THEN {未来3-6个月，股价下跌概率 >= 70%，预期跌幅 >= 20%，建议规避}
CONFIDENCE {0.70; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场，财务恶化叠加高估值是重要的风险信号}
FAILURE_MODE {公司业绩反转（新业务、新产品）；行业景气度回升；估值中枢上移（市场风格切换）；技术破位后快速反弹}

### 触发/解除/无效化条件

- **触发深度研究**：
  - 季度财报披露（业绩超预期或低于预期）
  - 重大公告（重组、业绩预告、风险提示）
  - 股价异动（单日涨跌幅 > 5%或连续上涨/下跌）
  - 券商研报（评级上调/下调、目标价大幅调整）
  - 行业变化（政策利好/利空、行业景气度变化）

- **解除监控**：
  - 投资逻辑失效（基本面恶化、估值过高）
  - 达到目标价（预期收益兑现）
  - 风险事件（ST、退市、重大违规）

- **无效化**：
  - 公司退市
  - 公司被并购（控制权变更）
  - 行业发生颠覆性变化（技术革命、政策禁止）

### 阈值选择依据

- **15% ROE**：优质公司的盈利能力标准，长期ROE > 15%的公司通常是行业龙头
- **20%净利润增长率**：成长股的增长标准，低于20%难以支撑高估值
- **1.5倍PB**：价值股的估值上限，PB < 1.5通常有安全边际
- **3%股息率**：价值股的股息标准，高于3%有吸引力（vs 无风险利率）
- **60%资产负债率**：财务安全线，高于60%财务风险上升
- **0.8经营现金流/净利润**：财务质量标准，低于0.8可能存在利润质量问题

## 边界条件与降级策略

### 数据缺失/异常值处理

- **数据缺失**：
  - 财报未披露：使用业绩预告或业绩快报，标注"预估数据"
  - 估值数据缺失：使用前一交易日数据，标注"数据未更新"
  - 券商研报缺失：使用历史研报或行业平均，标注"研报缺失"

- **异常值处理**：
  - ROE > 50%：检查是否有非经常性损益或会计调整，标注"ROE异常"
  - 净利润增长率 > 200%：检查是否有并购、资产注入，标注"增长异常"
  - PE < 0（亏损）：使用PB或PS估值，标注"PE不适用"

### 降级策略

- **AKShare接口不稳定**：
  - 备用数据源1：Wind金融终端
  - 备用数据源2：同花顺iFinD
  - 备用数据源3：手动输入（从公司公告获取）
  - 降级标注："使用备用数据源，可能有延迟"

- **财报数据延迟**：
  - 使用业绩预告（提前1-2个月）
  - 使用业绩快报（提前2-4周）
  - 使用券商一致预期
  - 降级标注："使用预估数据，以正式财报为准"

- **券商研报缺失**：
  - 使用行业平均评级
  - 使用历史研报
  - 自行分析（基于财务数据和估值）
  - 降级标注："研报缺失，评级为自行判断"

## A股特殊注意

### 1. T+1交易制度影响

- **买入后次日才能卖出**：
  - 需要更谨慎的买入决策（无法当日止损）
  - 关注次日开盘情况（高开/低开）
  - 避免追涨杀跌（T+1限制灵活性）

### 2. 涨跌停限制影响

- **涨停板限制买入**：
  - 业绩超预期可能连续涨停（追不上）
  - 需要提前布局（财报披露前）
  - 涨停打开后可能已大涨（追高风险）

- **跌停板限制卖出**：
  - 业绩暴雷可能连续跌停（卖不出）
  - 需要严格风控（避免踩雷）
  - 关注风险提示公告

### 3. 停牌影响

- **重大事项停牌**：
  - 重组、控制权变更可能长期停牌（数月）
  - 停牌期间无法交易（流动性归零）
  - 复牌后可能大涨或大跌（不确定性）

### 4. 财报披露时间

- **季报披露**：4月、8月、10月、次年4月
  - 业绩超预期：提前布局（财报前）或追涨（财报后）
  - 业绩低于预期：及时止损（财报后）

- **业绩预告**：提前1-2个月披露
  - 业绩预告是重要的先行指标
  - 关注预告与实际财报的差异

### 5. 非经常性损益

- **A股公司常有非经常性损益**：
  - 资产出售、政府补助、投资收益
  - 需要关注扣非净利润（剔除非经常性损益）
  - 扣非净利润更能反映主营业务盈利能力

### 6. 会计调整

- **A股公司可能有会计调整**：
  - 收入确认、费用资本化、减值准备
  - 需要关注审计意见（标准无保留/保留/否定）
  - 关注会计政策变更公告

### 7. 行业特性

- **周期性行业**：
  - 盈利随行业景气度波动（钢铁、化工、煤炭）
  - 估值看PB（周期底部PB < 1）
  - 关注行业供需、价格、库存

- **成长性行业**：
  - 盈利持续增长（消费、医药、科技）
  - 估值看PE或PEG（成长股PE较高）
  - 关注市场空间、竞争格局、技术壁垒

- **价值股**：
  - 盈利稳定、高股息（银行、公用事业）
  - 估值看PB和股息率
  - 关注ROE、分红率、资产质量

### 8. 政策影响

- **A股是政策市**：
  - 行业政策（如新能源补贴、房地产调控）
  - 货币政策（降息降准、流动性）
  - 监管政策（如减持新规、退市新规）
  - 需要关注政策动向，评估政策影响

### 9. 市场风格

- **牛市**：成长股表现好，高估值被接受
- **熊市**：价值股表现好，低估值受青睐
- **震荡市**：均衡配置，关注业绩确定性
- 需要根据市场风格调整选股策略

### 10. 券商研报

- **A股券商研报质量参差不齐**：
  - 部分研报过于乐观（利益相关）
  - 部分研报滞后（股价已大涨）
  - 需要独立判断，不盲目跟随
  - 关注研报逻辑和数据，而非评级

## 回测说明（最小化）

- **回测设计**：
  - 每季度财报披露后进行股票筛选
  - 根据基本面、估值、技术面综合评分
  - 买入评分前20%的股票，持有至下一季度
  - 基准：沪深300、中证500

- **性能指标**：
  - 年化收益率
  - 夏普比率、最大回撤
  - 胜率（持有期收益 > 0的比例）
  - 超额收益（策略收益 - 基准收益）

- **证伪条件**：
  - 年化收益率 <= 基准收益率
  - 夏普比率 <= 1
  - 胜率 <= 60%

- **敏感性分析**：
  - 测试不同阈值（ROE 15% vs 20%）
  - 测试不同市场环境（牛市 vs 熊市）
  - 测试不同行业（周期 vs 成长 vs 价值）
  - 测试不同持有期（1季度 vs 2季度 vs 1年）

## 使用示例

### 示例1：优质成长股

**输入**：某消费股（市值500亿）
**数据**：
- ROE：22%（连续3年 > 20%）
- 净利润增长率：28%（3年CAGR 25%）
- 毛利率：45%（行业领先）
- 经营现金流/净利润：1.2（优质）
- PE：25（行业PE中位数30）
- PB：5.5
- 股价：突破60日均线，成交量放大

**输出**：
- 基本面评分：90（优秀）
- 估值评分：75（合理偏低）
- 技术面评分：80（强势）
- 综合评分：85（推荐买入）
- 目标价：+30%（未来12个月）
- **结论**：优质成长股，盈利能力强，估值合理，技术面强势，建议买入

### 示例2：价值股

**输入**：某银行股（市值3000亿）
**数据**：
- ROE：12%（稳定）
- 净利润增长率：5%（稳健）
- 资产负债率：90%（银行业正常）
- 不良贷款率：1.2%（行业平均）
- PE：5（行业PE中位数6）
- PB：0.6（< 1，低估）
- 股息率：4.5%（高股息）

**输出**：
- 基本面评分：70（良好）
- 估值评分：85（低估）
- 股息评分：90（高股息）
- 综合评分：80（推荐买入）
- 预期收益：+15%（未来12个月）+ 4.5%股息
- **结论**：价值股，估值低，高股息，适合稳健投资者，建议买入

### 示例3：业绩预增机会

**输入**：某科技股（市值200亿）
**数据**：
- 业绩预告：净利润预增50-70%
- PE：30（行业PE中位数35）
- 股价：突破60日均线
- 成交量：放大80%
- 主力净流入：+2亿

**输出**：
- 业绩预增评分：90（大幅预增）
- 估值评分：70（合理）
- 技术面评分：85（强势）
- 综合评分：82（推荐买入）
- 预期收益：+20%（未来1-3个月）
- **结论**：业绩预增催化剂，技术面强势，建议买入，关注财报披露

### 示例4：风险规避

**输入**：某制造业股（市值100亿）
**数据**：
- 净利润增长率：-35%（业绩暴雷）
- 扣非净利润：-50%（主营业务亏损）
- 经营现金流：-5亿（现金流恶化）
- PE：50（行业PE中位数20）
- 股价：跌破60日均线
- 主力净流出：-3亿

**输出**：
- 基本面评分：20（差）
- 估值评分：30（高估）
- 技术面评分：25（弱势）
- 综合评分：25（建议规避）
- 预期风险：-30%（未来3-6个月）
- **结论**：业绩暴雷，估值过高，技术面破位，建议规避或卖出
