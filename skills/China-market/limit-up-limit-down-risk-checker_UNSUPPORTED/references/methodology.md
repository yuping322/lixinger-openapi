# 方法论：涨跌停风险检查器（A股）

本技能用于**检查A股个股的涨跌停风险，评估流动性风险和交易风险**。涨跌停是A股特有的交易制度，对交易影响巨大。

## 数据口径

### 数据源与字段映射

涨跌停风险检查需要以下数据：
- **涨跌停数据**：AKShare (`stock_zt_pool_em`, `stock_dt_pool_em`)
  - 涨停股、跌停股、涨停原因
  - 日频数据，实时更新
- **行情数据**：AKShare (`stock_zh_a_hist`)
  - 涨跌幅、成交量、换手率
  - 日频数据，实时更新
- **买卖盘数据**：AKShare (`stock_bid_ask_em`)
  - 买一卖一价格、挂单量
  - 实时数据

关键字段：
- `涨跌停状态`：涨停、跌停、正常
- `涨跌停时间`：涨跌停封板时间
- `封单量`：涨跌停封单金额

### 时间窗口与频率

- **实时监控**：盘中实时监控
- **日频统计**：每日涨跌停统计
- **历史回溯**：历史涨跌停记录

## 核心指标

### 指标列表与计算公式

1. **涨跌停状态指标**：
   - `涨跌停状态 = 1{涨跌幅 >= 9.9%或 <= -9.9%}` (0/1)
   - `涨停时间 = 涨停封板时间` (时:分)
   - `跌停时间 = 跌停封板时间` (时:分)
   - `连续涨跌停天数` (天)

2. **封单指标**：
   - `封单量 = 涨跌停价位挂单量` (手)
   - `封单金额 = 封单量 × 涨跌停价` (万元)
   - `封单比 = 封单金额 / 流通市值 × 100%` (%)
   - `封单强度 = 封单金额 / 日均成交额` (倍)

3. **流动性风险指标**：
   - `涨停流动性风险 = 1 - 成交量 / 日均成交量` (0-1)
   - `跌停流动性风险 = 1` (无法卖出)
   - `开板概率 = f(封单强度, 市场情绪, 个股基本面)` (%)

4. **交易风险指标**：
   - `涨停买入风险 = 1{次日开板下跌}` (0/1)
   - `跌停卖出风险 = 1{无法卖出}` (0/1)
   - `涨跌停波动风险 = std(涨跌停后收益率)` (%)

5. **市场情绪指标**：
   - `市场涨停股数` (只)
   - `市场跌停股数` (只)
   - `涨跌停比 = 涨停股数 / 跌停股数` (倍)
   - `市场情绪 = f(涨跌停比, 涨停股数)` (0-100)

### 标准化方法

- 使用**封单强度**评估涨跌停牢固程度
- 使用**历史开板概率**评估开板风险
- 使用**市场情绪**评估涨跌停环境

## 信号与阈值

### 可测试规则（Insight Rules）

Rule 1 (强封涨停 + 早盘封板 → 持续性强):
IF {封单强度 >= 3倍 AND 涨停时间 <= 10:00 AND 市场情绪 >= 70}
THEN {次日继续涨停概率 >= 60%，持续性强}
CONFIDENCE {0.60; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股个股，涨停股}
FAILURE_MODE {市场整体转弱；个股利好出尽；封单被砸开}

Rule 2 (弱封涨停 + 尾盘封板 → 开板风险):
IF {封单强度 < 1倍 AND 涨停时间 >= 14:00 AND 封单量波动大}
THEN {次日开板概率 >= 70%，开板后可能回调}
CONFIDENCE {0.70; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股个股，涨停股}
FAILURE_MODE {市场情绪好（继续封板）；个股利好持续；封单增强}

Rule 3 (跌停 + 封单强 → 流动性风险):
IF {跌停 AND 封单强度 >= 2倍}
THEN {无法卖出，流动性风险极高，可能连续跌停}
CONFIDENCE {0.75; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股个股，跌停股}
FAILURE_MODE {跌停打开（可以卖出）；利空消化（反弹）；市场整体反弹}

Rule 4 (连续涨停 + 封单减弱 → 见顶信号):
IF {连续涨停天数 >= 5天 AND 封单强度下降 >= 50% AND 换手率上升}
THEN {涨停板见顶概率 >= 65%，建议获利了结}
CONFIDENCE {0.65; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股个股，连续涨停股}
FAILURE_MODE {利好持续（继续涨停）；市场情绪高涨；封单重新增强}

### 触发/解除/无效化条件

- **触发检查**：
  - 个股涨跌停
  - 封单强度 >= 2倍或 < 1倍
  - 连续涨跌停 >= 3天

- **解除检查**：
  - 涨跌停打开
  - 恢复正常交易
  - 涨跌停风险消除

- **无效化**：
  - 停牌
  - 退市
  - 涨跌停规则调整

### 阈值选择依据

- **3倍封单强度**：封单强度 >= 3倍属于强封板
- **1倍封单强度**：封单强度 < 1倍属于弱封板
- **10:00涨停时间**：10:00前涨停属于早盘封板
- **14:00涨停时间**：14:00后涨停属于尾盘封板
- **5天连续涨停**：连续涨停 >= 5天属于连板股

## 边界条件与降级策略

### 数据缺失/异常值处理

- **数据缺失**：
  - 封单数据缺失：使用成交量估算，标注"数据估算"
  - 涨停时间缺失：使用收盘数据，标注"时间未知"

- **异常值处理**：
  - 封单强度 > 10倍：检查是否小盘股或极端情况
  - 涨停时间异常：检查是否数据错误

### 降级策略

- **实时数据不可用**：
  - 使用收盘数据
  - 降级标注："使用收盘数据，实时性差"

- **封单数据不准确**：
  - 使用成交量代替
  - 降级标注："封单数据不准确"

## A股特殊注意

### 1. 涨跌停限制

- **主板±10%**：主板涨跌停限制±10%
- **科创板/创业板±20%**：科创板、创业板涨跌停限制±20%
- **ST股±5%**：ST股涨跌停限制±5%
- **新股首日**：主板新股首日涨幅限制44%

### 2. T+1交易

- **涨停无法当日卖出**：涨停买入当日无法卖出
- **跌停无法当日买入**：跌停卖出当日无法买入
- **次日风险**：涨跌停次日开盘风险大

### 3. 连续涨跌停

- **连续涨停常见**：A股连续涨停较常见
- **连续跌停风险**：连续跌停流动性风险极高
- **需要警惕连板**：连续涨跌停风险大

### 4. 涨停原因

- **利好涨停**：业绩、重组、政策利好
- **题材涨停**：概念炒作、热点题材
- **跟风涨停**：板块联动、跟风炒作
- **需要区分原因**：不同原因持续性不同

### 5. 封单强度

- **封单强度重要**：封单强度决定涨停牢固程度
- **封单波动**：封单量波动反映市场分歧
- **需要监控封单**：实时监控封单变化

### 6. 开板风险

- **开板后回调**：涨停开板后可能回调
- **开板时机**：开板时机影响交易决策
- **需要警惕开板**：开板是重要的交易信号

### 7. 市场情绪

- **市场情绪影响**：市场情绪影响涨跌停持续性
- **涨停股数**：涨停股数反映市场情绪
- **需要关注情绪**：市场情绪是重要参考

### 8. 流动性风险

- **涨停流动性差**：涨停股流动性差，难以买入
- **跌停无法卖出**：跌停股无法卖出，流动性风险极高
- **需要评估流动性**：涨跌停流动性风险大

### 9. 小盘股风险

- **小盘股易涨跌停**：小盘股容易涨跌停
- **小盘股封单弱**：小盘股封单相对较弱
- **需要区分市值**：不同市值涨跌停特征不同

### 10. 监管影响

- **监管打击炒作**：监管打击涨停板炒作
- **异常交易监控**：交易所监控异常涨跌停
- **需要关注监管**：监管影响涨跌停交易

## 回测说明（最小化）

- **回测设计**：
  - 统计历史涨跌停数据
  - 分析涨跌停持续性
  - 验证涨跌停信号有效性

- **性能指标**：
  - 涨停持续概率
  - 开板后收益
  - 跌停反弹概率

- **证伪条件**：
  - 涨停持续概率 < 50%
  - 开板后收益不显著

## 使用示例

### 示例1：强封涨停

**输入**：个股A，2024-03-15涨停
**数据**：
- 封单强度：3.5倍
- 涨停时间：09:45
- 市场情绪：75
**输出**：
- 信号：强封涨停（持续性强）
- **结论**：早盘强封涨停，次日继续涨停概率高

### 示例2：弱封涨停

**输入**：个股B，2024-04-20涨停
**数据**：
- 封单强度：0.8倍
- 涨停时间：14:30
- 封单量波动：±30%
**输出**：
- 信号：弱封涨停（开板风险）
- **结论**：尾盘弱封涨停，次日开板概率高

### 示例3：跌停流动性风险

**输入**：个股C，2024-05-10跌停
**数据**：
- 跌停
- 封单强度：2.5倍
- 卖一挂单：巨量
**输出**：
- 信号：跌停（流动性风险）
- **结论**：跌停封单强，无法卖出，流动性风险极高

### 示例4：连板见顶

**输入**：个股D，2024-06-15连续涨停
**数据**：
- 连续涨停天数：6天
- 封单强度下降：-60%
- 换手率：上升至15%
**输出**：
- 信号：连板见顶
- **结论**：连续涨停封单减弱，见顶概率高，建议获利了结
