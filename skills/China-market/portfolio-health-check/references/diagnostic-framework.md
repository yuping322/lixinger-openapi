## 集中度阈值

### 单只个股集中度

| 权重 | 严重程度 | 建议 |
|------|---------|------|
| < 5% | 安全 | 无需操作 |
| 5–10% | 关注 | 高确信度头寸可接受 |
| 10–15% | 警告 | 建议减仓；若持有需记录理由 |
| 15–25% | 高风险 | 强烈建议减持 |
| > 25% | 严重 | 显著的个股风险；需立即关注 |

### 行业集中度

| 行业权重 vs 基准 | 严重程度 |
|----------------|---------|
| ±5%以内 | 中性 |
| ±5–10% | 轻度偏移——有意为之可接受 |
| ±10–20% | 显著超配/低配 |
| > ±20% | 极端集中——标记为严重 |

### 前N大集中度

| 前N大权重 | 评估 |
|----------|------|
| 前5 < 30% | 分散良好 |
| 前5: 30–50% | 适度集中 |
| 前5: 50–70% | 集中——标记 |
| 前5 > 70% | 高度集中——严重 |
| HHI > 0.15 | 赫芬达尔指数显示集中 |

### 资产类别最低配比（按风险偏好）

| 资产类别 | 保守 | 稳健 | 积极 |
|---------|------|------|------|
| 权益类 | 20–35% | 40–60% | 60–80% |
| 固定收益 | 40–55% | 25–40% | 10–25% |
| 另类 | 5–10% | 5–10% | 5–15% |
| 现金类 | 10–15% | 5–10% | 0–5% |

## 相关性分析

### 聚集检测方法

1. 计算所有权益持仓过去3年的周收益率相关系数
2. 采用层次聚类法（平均链接）按相关距离聚类
3. 定义聚集：≥ 3个仓位平均两两相关系数 > 0.70
4. 计算聚集权重 = 聚集内所有仓位权重之和

### A股常见相关性聚集

| 聚集类型 | 常见示例 | 为何容易被忽视 |
|---------|---------|-------------|
| 白酒板块 | 贵州茅台+五粮液+泸州老窖+山西汾酒 | 分属不同子行业但高度同向 |
| 新能源产业链 | 宁德时代+比亚迪+隆基绿能+阳光电源 | 上下游看似分散实则共振 |
| 大金融 | 招商银行+中国平安+东方财富 | 银行+保险+券商看似多元 |
| 医药白马 | 恒瑞医药+药明康德+迈瑞医疗 | 细分领域不同但行情同步 |
| 互联网/科技 | 腾讯+美团+快手（通过港股通） | 平台经济监管共振 |

### 有效分散比率

```
EDR = (各持仓波动率之和) / 组合波动率
```

| EDR | 解读 |
|-----|------|
| > 1.5 | 分散效果良好 |
| 1.2–1.5 | 分散效果一般 |
| 1.0–1.2 | 分散效果较弱——持仓走势趋同 |
| ~1.0 | 几乎无分散效果 |

## 因子暴露框架

### 因子定义

| 因子 | 指标 | 暴露过度标准 |
|------|------|-------------|
| 市场（Beta） | 组合Beta | > 1.3 或 < 0.7 偏离目标 |
| 价值 | 加权PE、PB | PE < 10 或 > 40（极端值） |
| 成长 | 营收增速 | 加权平均 > 30%（激进成长倾斜） |
| 规模 | 加权平均市值 | 平均 < 100亿元（小盘偏重） |
| 动量 | 12-1月收益率 | 强烈偏向近期赢家 |
| 质量 | ROE、资产负债率 | 低ROE或高杠杆 vs 基准 |
| 低波动 | 已实现波动率 | 全部持仓低于平均波动（防御性聚集） |
| 股息率 | 股息率 | 平均 > 4%（收入倾斜过重） |

### 因子偏移评分

对每个因子，计算组合相对基准的Z分数：

| Z分数 | 偏移评估 |
|-------|---------|
| -0.5 至 +0.5 | 中性 |
| ±0.5 至 ±1.0 | 轻度偏移——可能有意为之 |
| ±1.0 至 ±2.0 | 显著偏移——应当有意为之 |
| > ±2.0 | 极端偏移——需审查 |

## 健康评分模型

### 评分构成（总分0–100）

| 评分维度 | 权重 | 评分标准 |
|---------|------|---------|
| 分散度 | 25 | 基于HHI；低集中度 = 高分 |
| 相关性效率 | 15 | 基于EDR；高EDR = 高分 |
| 因子均衡 | 15 | 各因子绝对Z分数的平均值；低 = 高分 |
| 风险收益效率 | 15 | 夏普/索提诺 vs 可达有效前沿 |
| 流动性 | 10 | 按日均成交量20%计可在1日内变现的比例 |
| 压力韧性 | 10 | 压力情景下的平均回撤 vs 基准 |
| 成本效率 | 10 | 加权费率（如为基金/ETF）vs 可选替代 |

### 健康评分解读

| 分数 | 评级 | 解读 |
|------|------|------|
| 85–100 | 优秀 | 构建良好，改进空间有限 |
| 70–84 | 良好 | 组合扎实，有优化机会 |
| 55–69 | 一般 | 存在明显问题；建议针对性改进 |
| 40–54 | 需关注 | 多项风险因素；建议较大调整 |
| < 40 | 较差 | 严重问题；建议重大调整 |

## 流动性评估

### 变现天数计算

```
变现天数 = 仓位市值 / (日均成交额 × 假设参与率)
```

默认参与率：日均成交额的20%。

### A股特殊流动性因素

| 因素 | 影响 |
|------|------|
| 涨跌停限制 | 跌停时无法卖出，严重影响极端行情下的流动性 |
| T+1交割 | 当日买入次日方可卖出 |
| 停牌制度 | 重大事项停牌可能导致数日甚至数周无法交易 |
| 北向资金波动 | 外资大幅流出可能加剧蓝筹股卖压 |
| 量化交易限制 | 监管对程序化交易的阶段性限制可能影响流动性 |

### 流动性分类

| 变现天数 | 类别 | 标记 |
|---------|------|------|
| < 1天 | 高流动性 | 无 |
| 1–3天 | 流动性良好 | 无 |
| 3–5天 | 一般 | 关注 |
| 5–10天 | 低流动性 | 警告 |
| > 10天 | 流动性不足 | 严重 |

## 常见组合病症

| 病症 | 症状 | 常见原因 |
|------|------|---------|
| **打新底仓依赖** | 大量持有沪深300成分股仅为打新配市值 | 打新策略的市值需求 |
| **追涨型组合** | 超配近期热门板块（如AI、新能源轮动） | 追逐热点、FOMO心理 |
| **散户化配置** | 持仓过多（>20只），单只仓位过小 | 缺乏集中度管理 |
| **赛道聚集** | 60%+仓位在同一赛道（如新能源全产业链） | 认为产业链分散等于行业分散 |
| **A股Home Bias** | 100% A股，0% 港股通/QDII | 熟悉度偏好 |
| **僵尸仓位** | 多个亏损深套的小仓位长期持有不卖 | 损失厌恶、锚定效应 |
| **高波低质** | 组合中大量高波动低ROE的概念股 | 投机心理 |
| **基金重叠** | 持有多只同类型基金但底层持仓高度重叠 | 以为买不同基金就是分散 |
| **忽视现金** | 满仓操作，0现金 | 不愿资金闲置 |
