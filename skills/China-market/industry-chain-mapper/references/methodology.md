# 方法论：产业链映射器（A股）

本技能用于**映射A股上市公司的产业链关系，识别上下游联动、产业链传导、供需关系变化**。产业链分析是深度研究的基础，产业链传导往往领先个股表现。

## 数据口径

### 数据源与字段映射

产业链映射需要以下数据：
- **产业链关系**：手动整理 + 公司公告 + 研报
  - 上游供应商、下游客户关系
  - 产业链位置（上游/中游/下游）
  - 产业链环节（原材料/零部件/组装/销售）
- **公司业务**：AKShare (`stock_individual_info_em`), 公司公告
  - 主营业务、产品结构、客户结构
  - 供应商集中度、客户集中度
- **行业数据**：国家统计局、行业协会
  - 行业产量、价格、库存、开工率
  - 上下游价格传导、供需关系
- **财务数据**：AKShare (`stock_financial_abstract`), 财报
  - 营收、毛利率、净利率、ROE
  - 应收账款、应付账款、存货周转

关键字段：
- `产业链位置`：上游/中游/下游
- `上游供应商`：主要供应商列表
- `下游客户`：主要客户列表
- `价格传导系数`：上游价格变化对下游的影响系数
- `供需缺口`：供给 - 需求（正为过剩，负为短缺）

### 时间窗口与频率

- **季频更新**：每季度财报后更新产业链关系
- **月频监控**：每月监控产业链价格、产量、库存变化
- **周频跟踪**：每周跟踪产业链热点、政策变化
- **日频预警**：每日监控产业链龙头股价格异动

建议窗口：
- 价格传导：1-3个月（上游价格变化传导到下游）
- 业绩传导：1-2个季度（上游业绩变化传导到下游）
- 政策影响：即时-3个月（政策对产业链的影响）

## 核心指标

### 指标列表与计算公式

1. **产业链位置指标**：
   - `产业链层级 = 上游/中游/下游`
   - `产业链环节 = 原材料/零部件/组装/销售`
   - `产业链集中度 = Top3供应商/客户占比` (%)
   - `产业链话语权 = 毛利率 / 行业平均毛利率`

2. **价格传导指标**：
   - `上游价格变化 = (当前价格 - 前期价格) / 前期价格` (%)
   - `价格传导系数 = 下游价格变化 / 上游价格变化`
   - `价格传导时滞 = 上游价格变化到下游价格变化的时间`（月）
   - `毛利率变化 = 当前毛利率 - 前期毛利率`（百分点）

3. **供需关系指标**：
   - `供需缺口 = 供给 - 需求`（万吨/万台等）
   - `库存天数 = 库存 / 日均销量`（天）
   - `开工率 = 实际产量 / 产能` (%)
   - `供需紧张度 = (需求 - 供给) / 供给` (%)

4. **业绩传导指标**：
   - `上游业绩变化 = (当前净利润 - 去年同期) / 去年同期` (%)
   - `业绩传导系数 = 下游业绩变化 / 上游业绩变化`
   - `业绩传导时滞 = 上游业绩变化到下游业绩变化的时间`（季度）
   - `ROE变化 = 当前ROE - 前期ROE`（百分点）

5. **产业链景气度指标**：
   - `产业链景气度 = (价格变化 + 产量变化 + 开工率变化) / 3`
   - `产业链景气度排名 = rank(产业链景气度)`
   - `产业链景气度变化 = 当前景气度 - 上期景气度`

### 标准化方法

- 使用**相对变化**计算价格/业绩传导（变化率而非绝对值）
- 使用**时滞分析**识别产业链传导规律（领先/滞后关系）
- 使用**相关性分析**验证产业链关系（上下游价格/业绩相关性）
- 使用**情景分析**预测产业链传导（假设上游变化X%，下游变化Y%）

## 信号与阈值

### 可测试规则（Insight Rules）

Rule 1 (上游价格上涨 + 下游需求旺盛 → 中游受益):
IF {上游价格3月涨幅 >= 20% AND 下游需求增速 >= 15% AND 中游库存天数 <= 30}
THEN {未来1-2个季度，中游公司毛利率上升概率 >= 65%，预期毛利率提升 >= 3百分点}
CONFIDENCE {0.65; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {制造业产业链（新能源、半导体、汽车等），中游环节公司}
FAILURE_MODE {中游无法转嫁成本（竞争激烈）；下游需求突然转弱；上游价格继续上涨（成本压力）；库存积压}

Rule 2 (上游业绩改善 → 下游业绩传导):
IF {上游净利润增速 >= 30% AND 业绩改善持续 >= 2季度 AND 下游与上游相关性 >= 0.7}
THEN {未来1-2个季度，下游公司业绩改善概率 >= 62%，预期净利润增速 >= 20%}
CONFIDENCE {0.62; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {产业链关系明确的行业（消费电子、新能源车、光伏等）}
FAILURE_MODE {下游需求不足（终端疲软）；产业链传导中断（政策/技术变化）；下游竞争加剧（价格战）}

Rule 3 (供需缺口扩大 → 价格上涨):
IF {供需紧张度 >= 20% AND 库存天数 <= 20 AND 开工率 >= 90%}
THEN {未来1-3个月，产品价格上涨概率 >= 68%，预期涨幅 >= 15%}
CONFIDENCE {0.68; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {大宗商品、化工品、有色金属等供需驱动的行业}
FAILURE_MODE {新增产能快速释放；需求突然转弱；进口增加（供给补充）；政策调控（限价）}

Rule 4 (产业链景气度下降 → 全链条承压):
IF {产业链景气度连续2季度下降 AND 上游价格下跌 >= 15% AND 下游需求增速 <= 0}
THEN {未来1-2个季度，产业链公司业绩下滑概率 >= 65%，预期净利润下降 >= 20%}
CONFIDENCE {0.65; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {周期性产业链（地产链、汽车链、化工链等）}
FAILURE_MODE {政策刺激（需求回暖）；产能出清（供给收缩）；成本下降（利润改善）；转型升级（新业务）}

### 触发/解除/无效化条件

- **触发监控**：
  - 上游价格变化 >= 15%或 <= -15%（显著变化）
  - 供需缺口 >= 20%或 <= -20%（供需失衡）
  - 产业链景气度变化 >= 10百分点（景气度转折）
  - 上游业绩增速 >= 30%或 <= -30%（业绩拐点）

- **解除监控**：
  - 价格变化恢复正常（-5% ~ +5%区间持续2个月）
  - 供需恢复平衡（-5% ~ +5%区间持续2个月）
  - 景气度稳定（变化 < 5百分点持续2个季度）

- **无效化**：
  - 产业链关系发生重大变化（技术革新、政策调整）
  - 数据缺失或不可靠（行业数据停止更新）
  - 公司业务转型（退出原产业链）
  - 不可抗力（疫情、战争、自然灾害）

### 阈值选择依据

- **20%价格涨幅**：上游价格上涨20%以上，对中下游有显著影响
- **15%需求增速**：下游需求增速15%以上，表明需求旺盛
- **30天库存**：库存天数30天以下，表明供需偏紧
- **20%供需缺口**：供需缺口20%以上，表明供需严重失衡
- **30%业绩增速**：业绩增速30%以上，表明景气度显著改善

## 边界条件与降级策略

### 数据缺失/异常值处理

- **数据缺失**：
  - 产业链关系缺失：使用行业研报、公司公告补充
  - 价格数据缺失：使用替代品价格、龙头公司价格
  - 产量数据缺失：使用行业协会数据、海关数据

- **异常值处理**：
  - 价格异常波动（> 50%）：检查是否政策调控、突发事件
  - 业绩异常变化（> 100%）：检查是否并购重组、资产处置
  - 供需数据异常：检查是否统计口径变化、数据错误

### 降级策略

- **产业链关系不明确**：
  - 使用行业分类代替产业链关系
  - 使用龙头公司代替产业链分析
  - 降级标注："产业链关系不明确，使用行业分析"

- **价格数据缺失**：
  - 使用PPI/CPI代替具体产品价格
  - 使用公司毛利率变化推算价格变化
  - 降级标注："价格数据缺失，使用代理指标"

- **供需数据缺失**：
  - 使用库存数据推算供需关系
  - 使用价格变化推算供需关系
  - 降级标注："供需数据缺失，使用间接指标"

## A股特殊注意

### 1. T+1交易制度影响

- **产业链传导需要时间**：产业链传导通常需要1-3个月，T+1限制影响较小
- **但短期交易受限**：发现产业链机会后，T+1限制无法当日买入卖出

### 2. 涨跌停限制影响

- **产业链龙头涨停**：产业链龙头涨停时，资金可能流向产业链其他公司
- **产业链联动涨停**：产业链公司可能连续涨停，无法买入

### 3. 停牌影响

- **产业链公司停牌**：产业链关键公司停牌，影响产业链分析
- **复牌后补涨/补跌**：停牌公司复牌后可能补涨/补跌

### 4. 政策驱动明显

- **产业政策影响大**：A股产业链受政策影响显著：
  - 新能源产业链：补贴政策、双碳政策
  - 半导体产业链：国产替代政策、科技自立
  - 地产产业链：房地产调控政策
  - 需要密切关注产业政策变化

### 5. 产业链数据滞后

- **财报数据滞后**：产业链业绩传导基于财报，滞后1-3个月
- **需要高频数据补充**：使用价格、产量、开工率等高频数据
- **需要草根调研**：实地调研产业链公司，获取一手信息

### 6. 产业链关系复杂

- **多层级产业链**：A股产业链通常有3-5层，关系复杂
- **交叉持股**：产业链公司之间可能交叉持股，影响独立性
- **需要简化分析**：聚焦核心产业链关系，避免过度复杂

### 7. 产业链集中度高

- **龙头主导**：部分产业链龙头公司市占率高，主导产业链
- **需要关注龙头**：龙头公司变化往往领先产业链变化
- **小公司跟随**：小公司往往跟随龙头，缺乏独立性

### 8. 产业链国产替代

- **国产替代加速**：半导体、新能源等产业链国产替代加速
- **进口依赖下降**：部分产业链进口依赖下降，国产化率提升
- **需要关注国产化进度**：国产化率提升对产业链影响大

### 9. 产业链全球化

- **全球产业链**：部分产业链（消费电子、汽车）全球化程度高
- **海外需求影响**：海外需求变化对国内产业链影响大
- **需要关注海外数据**：海外价格、需求、政策变化

### 10. 产业链周期性

- **周期性产业链**：地产链、汽车链、化工链等周期性明显
- **周期波动大**：产业链景气度波动大，需要把握周期
- **需要逆周期配置**：在周期底部配置，周期顶部减持

## 回测说明（最小化）

- **回测设计**：
  - 每季度财报后计算产业链信号
  - 次季度开盘买入产业链受益公司
  - 持有期：1-2个季度
  - 基准：行业指数、市场指数

- **性能指标**：
  - 信号胜率（信号后N季度业绩改善概率）
  - 平均超额收益（产业链公司收益 - 基准收益）
  - 信息比率、最大回撤

- **证伪条件**：
  - 信号胜率 <= 50%（不优于随机）
  - 平均超额收益 <= 0（不优于基准）
  - 信息比率 <= 0.5（风险调整后收益不佳）

## 使用示例

### 示例1：新能源车产业链

**输入**：新能源车产业链，2024Q1
**数据**：
- 上游（锂矿）：价格+25%，业绩+40%
- 中游（电池）：毛利率+2百分点，库存25天
- 下游（整车）：需求+18%，销量+20%
**输出**：
- 信号：上游价格上涨 + 下游需求旺盛（中游受益）
- **结论**：中游电池企业受益，毛利率改善，关注宁德时代、比亚迪

### 示例2：半导体产业链

**输入**：半导体产业链，2024Q2
**数据**：
- 上游（设备）：业绩+35%，持续2季度
- 中游（晶圆厂）：开工率85%，产能利用率提升
- 下游（芯片设计）：需求+12%
**输出**：
- 信号：上游业绩改善（下游业绩传导）
- **结论**：下游芯片设计公司业绩将改善，关注韦尔股份、兆易创新

### 示例3：化工产业链

**输入**：化工产业链，2024-03
**数据**：
- 供需紧张度：+25%（需求 > 供给）
- 库存天数：18天（偏低）
- 开工率：92%（高位）
- 价格：+8%（1月涨幅）
**输出**：
- 信号：供需缺口扩大（价格上涨）
- **结论**：化工品价格将继续上涨，关注万华化学、华鲁恒升

### 示例4：地产产业链

**输入**：地产产业链，2024Q3
**数据**：
- 产业链景气度：连续2季度下降
- 上游（水泥）：价格-18%
- 下游（地产）：需求增速-5%
- 中游（建材）：库存天数45天（偏高）
**输出**：
- 信号：产业链景气度下降（全链条承压）
- **结论**：地产产业链全链条承压，业绩下滑，谨慎配置
