# 方法论：A/H溢价监控（A股）

本技能用于**监控A股和H股（香港）同一公司的价格差异（溢价/折价），识别套利机会、市场情绪、资金流向等信号**。A/H溢价是反映两地市场情绪、流动性、投资者结构差异的重要指标，也是跨市场配置的参考依据。

## 数据口径

### 数据源与字段映射

A/H溢价监控需要以下数据：
- **A/H股价数据**：AKShare (`stock_a_lg_indicator`), Wind、东方财富
  - A股代码、A股价格、H股代码、H股价格、汇率
  - 日频数据，实时更新
- **A/H溢价指数**：AKShare (`stock_a_lg_indicator`), 恒生指数公司
  - 恒生AH股溢价指数（整体溢价水平）
  - 日频数据，实时更新
- **北向资金数据**：AKShare (`stock_hsgt_north_net_flow_in_em`)
  - 北向资金净流入、持股市值
  - 日频数据，T日晚间更新
- **汇率数据**：AKShare (`fx_spot_quote`)
  - 人民币兑港币汇率（CNY/HKD）
  - 日频数据，实时更新

关键字段：
- `A股价格`：A股市场价格（人民币）
- `H股价格`：H股市场价格（港币）
- `汇率`：人民币兑港币汇率（CNY/HKD）
- `H股价格(人民币) = H股价格 × 汇率`
- `A/H溢价率 = (A股价格 - H股价格(人民币)) / H股价格(人民币)` (%)
- `恒生AH股溢价指数`：整体溢价水平（100为平价）

### 时间窗口与频率

- **日频监控**：每日收盘后更新，监控溢价率变化
- **周频总结**：每周汇总溢价率变化、极端溢价/折价个股
- **月频深度分析**：结合北向资金、市场情绪，分析溢价趋势

建议窗口：
- 短期波动：5日、10日溢价率变化
- 中期趋势：60日、120日溢价率均值和标准差
- 长期配置：252日（年度）溢价率分布
- 极端监控：溢价率 > 50%（极高溢价）或 < -10%（折价）

## 核心指标

### 指标列表与计算公式

1. **溢价率指标**：
   - `A/H溢价率 = (A股价格 - H股价格(人民币)) / H股价格(人民币)` (%)
   - `溢价率百分位 = percentile_rank(当前溢价率, 252日历史)` (%)
   - `溢价率z-score = (当前溢价率 - 252日均值) / 252日标准差`
   - `恒生AH股溢价指数`：整体溢价水平（100为平价，> 100为溢价）

2. **溢价变化指标**：
   - `溢价率变化 = 当前溢价率 - 上期溢价率` (百分点)
   - `溢价率变化率 = (当前溢价率 - 上期溢价率) / |上期溢价率|` (%)
   - `溢价率趋势 = sign(20日溢价率变化)` (+1上升, -1下降)
   - `溢价率波动 = std(20日溢价率)` (%)

3. **相对表现指标**：
   - `A股相对涨跌幅 = A股N日涨跌幅 - H股N日涨跌幅` (百分点)
   - `A股相对强度 = A股价格 / H股价格(人民币)` (倍数)
   - `A股成交额 / H股成交额`：流动性比较
   - `A股换手率 / H股换手率`：活跃度比较

4. **市场情绪指标**：
   - `极端溢价个股数 = count(溢价率 > 50%)`
   - `折价个股数 = count(溢价率 < 0)`
   - `溢价率中位数`：整体市场溢价水平
   - `溢价率离散度 = std(所有AH股溢价率)` (%)

5. **套利指标**：
  - `套利空间 = |A/H溢价率|` (%)
   - `套利成本 = 交易费用 + 汇率成本 + 时间成本` (%)
   - `净套利空间 = 套利空间 - 套利成本` (%)
   - `套利可行性 = 1{净套利空间 > 2% AND 流动性充足}`

### 标准化方法

- 使用**百分位排名**对溢价率进行标准化（相对252日历史）
- 使用**z-score**检测极端溢价/折价（> 2倍标准差）
- 使用**中位数**衡量整体市场溢价水平（避免极端值影响）
- 使用**行业相对**比较溢价率（个股 vs 行业中位数）

## 信号与阈值

### 可测试规则（Insight Rules）

Rule 1 (极端溢价 + 溢价率下降 → A股承压):
IF {A/H溢价率 > 50% AND 溢价率百分位 > 90 AND 20日溢价率变化 < -5%}
THEN {未来10-20个交易日，A股相对H股下跌概率 >= 65%，溢价率回归，预期A股跑输H股 >= 3%}
CONFIDENCE {0.65; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A+H股，尤其是金融、能源等大盘股}
FAILURE_MODE {A股市场整体大涨（系统性上涨）；北向资金大幅流入（外资买入）；H股市场大跌（港股弱势）；公司基本面改善（业绩超预期）}

Rule 2 (溢价率快速收窄 + 北向流入 → A股相对强势):
IF {20日溢价率变化 < -10% AND 北向资金20日累计流入 > 200亿 AND 溢价率仍 > 20%}
THEN {未来10-20个交易日，A股相对H股上涨概率 >= 60%，溢价率继续收窄，预期A股跑赢H股 >= 2%}
CONFIDENCE {0.60; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A+H股，北向资金重仓股}
FAILURE_MODE {北向资金流入放缓或流出；A股市场整体下跌；H股市场大涨（港股强势）；溢价率已接近历史低位}

Rule 3 (折价 + H股强势 → 套利机会):
IF {A/H溢价率 < -5% AND H股20日涨幅 > A股20日涨幅 + 5% AND 流动性充足}
THEN {未来5-10个交易日，A股补涨概率 >= 62%，溢价率回归，预期A股跑赢H股 >= 3%}
CONFIDENCE {0.62; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A+H股，折价情况较少见，通常是套利机会}
FAILURE_MODE {A股市场整体弱势；H股继续强势（港股牛市）；公司基本面恶化；套利资金不足}

Rule 4 (恒生AH溢价指数高位 + 回落 → 市场情绪降温):
IF {恒生AH溢价指数 > 140 AND 20日变化 < -5 AND 北向资金流出}
THEN {未来1-2个月，A股市场情绪降温，预期A股相对H股下跌 >= 5%}
CONFIDENCE {0.63; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股市场整体，恒生AH溢价指数是重要的市场情绪指标}
FAILURE_MODE {A股市场基本面改善（经济数据好转）；政策利好（降息、降准）；北向资金重新流入；H股市场大跌}

### 触发/解除/无效化条件

- **触发监控**：
  - A/H溢价率 > 50%（极端溢价）
  - A/H溢价率 < -5%（折价）
  - 20日溢价率变化 > 10%或 < -10%（快速变化）
  - 恒生AH溢价指数 > 140或 < 110（极端水平）
  - 溢价率百分位 > 95或 < 5（历史极值）

- **解除监控**：
  - 溢价率回归正常（10-30%区间）
  - 溢价率稳定（20日变化 < 5%）
  - 恒生AH溢价指数回归正常（120-130区间）

- **无效化**：
  - 公司退市（A股或H股）
  - 公司私有化（H股退市）
  - 汇率大幅波动（> 5%）导致溢价率失真
  - 停牌（A股或H股）

### 阈值选择依据

- **50%极端溢价**：历史上溢价率超过50%通常是极端情况，回归概率高
- **-5%折价**：A股通常溢价，折价情况较少，可能是套利机会
- **10%溢价率变化**：20日内溢价率变化10%以上，表明市场情绪快速变化
- **140恒生AH溢价指数**：历史上指数超过140通常是高位，回落概率高
- **110恒生AH溢价指数**：历史上指数低于110通常是低位，反弹概率高
- **200亿北向流入**：20日累计流入200亿以上，表明外资持续买入

## 边界条件与降级策略

### 数据缺失/异常值处理

- **数据缺失**：
  - A股或H股价格缺失：使用前一交易日价格，标注"数据未更新"
  - 汇率缺失：使用前一交易日汇率，标注"汇率未更新"
  - 恒生AH溢价指数缺失：使用个股溢价率中位数估算

- **异常值处理**：
  - 溢价率 > 100%：检查是否有特殊事件（如A股炒作、H股停牌），标注"极端溢价"
  - 溢价率 < -20%：检查是否有特殊事件（如H股炒作、A股停牌），标注"极端折价"
  - 单日溢价率变化 > 20%：检查是否有涨跌停、停牌复牌，标注"异常波动"

### 降级策略

- **AKShare接口不稳定**：
  - 备用数据源1：Wind金融终端
  - 备用数据源2：东方财富AH股对比（`http://data.eastmoney.com/hsgt/`）
  - 备用数据源3：手动计算（从交易软件获取A/H股价格和汇率）
  - 降级标注："使用备用数据源，可能有延迟"

- **汇率数据缺失**：
  - 使用前一交易日汇率
  - 使用固定汇率（如0.9，人民币兑港币长期在0.85-0.95区间）
  - 降级标注："汇率为估计值"

- **恒生AH溢价指数缺失**：
  - 使用个股溢价率中位数
  - 使用加权平均溢价率（按市值加权）
  - 降级标注："指数为估算值"

## A股特殊注意

### 1. T+1交易制度差异

- **A股T+1，H股T+2**：A股当日买入次日可卖出，H股当日买入第三日可卖出：
  - 套利周期不同（A股更快）
  - 资金占用时间不同
  - 需要考虑时间成本

### 2. 涨跌停限制差异

- **A股±10%，H股无涨跌停**：A股有涨跌停限制，H股无限制：
  - A股涨停时无法买入（套利受限）
  - A股跌停时无法卖出（套利受限）
  - H股可自由交易（套利更灵活）
  - 溢价率可能因涨跌停而失真

### 3. 停牌影响

- **A股停牌H股继续交易**：A股停牌时H股仍可交易：
  - 溢价率失效（无法套利）
  - H股价格反映市场预期
  - A股复牌后可能大幅调整（补涨或补跌）

- **H股停牌A股继续交易**：H股停牌时A股仍可交易：
  - 溢价率失效（无法套利）
  - A股价格可能脱离H股
  - H股复牌后可能大幅调整

### 4. 交易时间差异

- **A股9:30-15:00，H股9:30-16:00**：交易时间不完全重合：
  - H股多1小时交易时间（15:00-16:00）
  - 这1小时内H股可能大幅波动
  - 需要使用同一时点价格计算溢价率（如14:30或15:00）

- **节假日差异**：A股和H股节假日不完全相同：
  - 香港特有假期（如佛诞日、圣诞节）
  - 内地特有假期（如国庆节）
  - 一方休市时溢价率失效

### 5. 投资者结构差异

- **A股散户主导，H股机构主导**：
  - A股情绪化交易更明显（溢价率波动大）
  - H股理性交易更多（溢价率相对稳定）
  - A股更容易炒作（溢价率飙升）
  - H股更注重基本面（溢价率合理）

### 6. 流动性差异

- **A股流动性通常更好**：
  - A股成交额通常 > H股成交额
  - A股换手率通常 > H股换手率
  - 套利时需要考虑流动性（H股可能难以成交）

### 7. 汇率影响

- **人民币贬值 → 溢价率上升**：
  - H股价格(人民币)下降
  - A/H溢价率上升（分母变小）
  - 需要剔除汇率影响（使用固定汇率计算）

- **人民币升值 → 溢价率下降**：
  - H股价格(人民币)上升
  - A/H溢价率下降（分母变大）

### 8. 北向资金影响

- **北向资金流入 → 溢价率下降**：
  - 外资买入A股（A股上涨）
  - 但同时也买入H股（H股上涨）
  - 通常H股涨幅更大（外资更熟悉港股）
  - 溢价率收窄

- **北向资金流出 → 溢价率上升**：
  - 外资卖出A股（A股下跌）
  - 但H股可能跌幅更大（外资主导港股）
  - 溢价率可能扩大或收窄（取决于谁跌得更多）

### 9. 套利限制

- **沪港通/深港通限制**：
  - 每日额度限制（520亿）
  - 部分股票不在互联互通范围（无法套利）
  - 需要开通港股通账户

- **套利成本**：
  - 交易费用（A股约0.1%，H股约0.2%）
  - 汇率成本（买卖价差约0.1-0.2%）
  - 时间成本（T+1或T+2）
  - 冲击成本（大额交易）
  - 总成本约1-2%，净套利空间需 > 2%才有利可图

### 10. 监管政策影响

- **互联互通扩容**：纳入更多标的，套利机会增加
- **额度调整**：额度扩大，套利更容易
- **交易规则变化**：如T+0试点，套利效率提高
- **汇率管制**：人民币国际化，汇率波动影响

## 回测说明（最小化）

- **回测设计**：
  - 每日收盘后计算溢价率
  - 根据溢价率信号买入/卖出A股或H股
  - 持有期：10日/20日
  - 基准：沪深300 vs 恒生中国企业指数

- **性能指标**：
  - 信号胜率（溢价率信号后相对表现符合预期的概率）
  - 平均超额收益（A股收益 - H股收益）
  - 套利成功率（净套利空间 > 0的比例）
  - 夏普比率、最大回撤

- **证伪条件**：
  - 信号胜率 <= 55%（不显著优于随机）
  - 平均超额收益 <= 0（不优于基准）
  - 套利成功率 < 60%（套利无效）

- **敏感性分析**：
  - 测试不同阈值（50% vs 60%极端溢价）
  - 测试不同市场环境（牛市 vs 熊市）
  - 测试不同行业（金融 vs 能源 vs 消费）
  - 测试不同汇率环境（人民币升值 vs 贬值）

## 使用示例

### 示例1：极端溢价回归

**输入**：某银行股（A+H）
**数据**：
- A股价格：6.5元
- H股价格：4.5港币
- 汇率：0.9（CNY/HKD）
- H股价格(人民币)：4.05元
- A/H溢价率：(6.5-4.05)/4.05 = 60.5%
- 溢价率百分位：95%
- 20日溢价率变化：-8%

**输出**：
- 信号：极端溢价回归
- 预期：A股相对H股下跌或H股相对A股上涨
- 策略：卖出A股或买入H股
- **结论**：溢价率过高且开始回落，建议配置H股或减持A股

### 示例2：折价套利

**输入**：某能源股（A+H）
**数据**：
- A股价格：8.0元
- H股价格：9.5港币
- 汇率：0.9（CNY/HKD）
- H股价格(人民币)：8.55元
- A/H溢价率：(8.0-8.55)/8.55 = -6.4%
- H股20日涨幅：12%
- A股20日涨幅：5%

**输出**：
- 信号：折价套利
- 预期：A股补涨
- 策略：买入A股
- **结论**：A股折价且H股强势，A股有补涨空间

### 示例3：市场情绪降温

**输入**：恒生AH溢价指数
**数据**：
- 当前指数：145
- 20日变化：-8
- 北向资金20日累计：-150亿（流出）

**输出**：
- 信号：市场情绪降温
- 预期：A股相对H股下跌
- 策略：减持A股或增持H股
- **结论**：整体溢价率高位回落，市场情绪降温，A股承压

### 示例4：溢价率收窄

**输入**：某消费股（A+H）
**数据**：
- A/H溢价率：25%（从40%下降）
- 20日溢价率变化：-15%
- 北向资金20日累计：+250亿（流入）
- A股20日涨幅：8%
- H股20日涨幅：12%

**输出**：
- 信号：溢价率快速收窄
- 预期：A股相对H股继续跑输或H股继续强势
- 策略：增持H股
- **结论**：外资持续买入，H股表现更强，溢价率继续收窄
