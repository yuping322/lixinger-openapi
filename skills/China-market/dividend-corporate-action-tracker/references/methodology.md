# 方法论：分红与公司行动跟踪器（A股）

本技能用于**跟踪A股上市公司的分红派息、送转股、配股、增发等公司行动，评估分红质量和股东回报**。分红是上市公司回馈股东的重要方式，也是价值投资的核心指标。

## 数据口径

### 数据源与字段映射

分红与公司行动跟踪需要以下数据：
- **分红数据**：AKShare (`stock_dividend_cninfo`), 东方财富
  - 分红方案、除权除息日、股权登记日、派息日
  - 每股分红、送股比例、转股比例
  - 实时数据，公告后更新
- **财务数据**：AKShare (`stock_financial_report_sina`)
  - 净利润、归母净利润、自由现金流
  - 季度/年度数据，财报发布后更新
- **股价数据**：AKShare (`stock_zh_a_hist`)
  - 股价、市值、成交量
  - 日频数据，实时更新
- **公司行动数据**：AKShare (`stock_share_change_cninfo`)
  - 配股、增发、回购、减持
  - 实时数据，公告后更新

关键字段：
- `分红率 = 每股分红 / 每股收益`（派息率）
- `股息率 = 每股分红 / 股价`（当前股息率）
- `分红连续性`：连续分红年数
- `分红增长率`：分红同比增长率

### 时间窗口与频率

- **年度分红**：每年4-6月公布年报分红方案
- **中期分红**：部分公司在8-9月公布中报分红
- **除权除息日**：分红方案实施日期
- **股权登记日**：确定分红股东名单的日期
- **派息日**：实际派发现金的日期

## 核心指标

### 指标列表与计算公式

1. **分红指标**：
   - `每股分红 = 分红总额 / 总股本` (元/股)
   - `股息率 = 每股分红 / 股价 × 100%` (%)
   - `分红率 = 分红总额 / 归母净利润 × 100%` (%)
   - `分红收益率 = (每股分红 + 股价变动) / 股价` (%)

2. **分红质量指标**：
   - `现金分红占比 = 现金分红 / (现金分红 + 送转股)` (%)
   - `分红连续性 = 连续分红年数` (年)
   - `分红增长率 = (本年分红 - 去年分红) / 去年分红 × 100%` (%)
   - `分红稳定性 = 1 - std(分红率) / mean(分红率)` (0-1)

3. **公司行动指标**：
   - `送转股比例 = (送股 + 转股) / 10股` (股/10股)
   - `配股比例 = 配股数 / 总股本 × 100%` (%)
   - `增发规模 = 增发股数 × 增发价格` (亿元)
   - `回购金额 = 回购股数 × 回购均价` (亿元)

4. **股东回报指标**：
   - `总股东回报 = (股价变动 + 分红) / 初始股价 × 100%` (%)
   - `年化股东回报 = (1 + 总回报)^(1/年数) - 1` (%)
   - `分红贡献度 = 分红收益 / 总回报 × 100%` (%)
   - `资本增值贡献度 = 股价增值 / 总回报 × 100%` (%)

5. **横截面指标**：
   - `行业分红率排名 = rank(分红率) within industry` (分位数)
   - `市场分红率排名 = rank(分红率) within market` (分位数)
   - `股息率排名 = rank(股息率) within market` (分位数)
   - `分红增长排名 = rank(分红增长率) within market` (分位数)

### 标准化方法

- 使用**分红率**（派息率）衡量分红慷慨度
- 使用**股息率**衡量当前分红吸引力
- 使用**分红连续性**和**稳定性**衡量分红质量
- 使用**行业排名**控制行业差异

## 信号与阈值

### 可测试规则（Insight Rules）

Rule 1 (高股息率 + 稳定分红 → 防御性配置):
IF {股息率 >= 4% AND 分红连续性 >= 5年 AND 分红率 >= 30% AND 分红率 <= 70%}
THEN {未来12个月，股价波动率 <= 市场平均 - 5%，下跌风险较低}
CONFIDENCE {0.68; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股个股，主板蓝筹股（市值 >= 100亿）信号更强}
FAILURE_MODE {公司业绩大幅下滑（分红不可持续）；市场风格转向成长股（高股息股跑输）；利率上升（债券吸引力增加）}

Rule 2 (分红率突然大幅提升 → 警惕业绩见顶):
IF {分红率 >= 80% AND 分红率增长 >= 30% AND 历史分红率 < 50%}
THEN {未来1-2年，业绩增长放缓概率 >= 60%，ROE可能下降}
CONFIDENCE {0.60; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股个股，成长股突然高分红需警惕}
FAILURE_MODE {公司现金充裕（主动提高分红）；一次性收益（非经常性损益）；股东要求（大股东资金需求）}

Rule 3 (高送转 + 低现金分红 → 警惕炒作):
IF {送转股比例 >= 10股送转5股 AND 现金分红占比 < 30% AND 股价涨幅 >= 20%（公告前1个月）}
THEN {除权后1-3个月，股价填权概率 < 40%，可能回调}
CONFIDENCE {0.55; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股个股，小盘股高送转炒作常见}
FAILURE_MODE {公司业绩高增长（支撑填权）；市场情绪好（题材炒作）；机构重仓（有资金支撑）}

Rule 4 (分红 + 回购 → 股东友好信号):
IF {股息率 >= 3% AND 回购金额 >= 市值 × 1% AND 分红连续性 >= 3年}
THEN {未来6-12个月，股价跑赢市场概率 >= 58%，超额收益 >= 3%}
CONFIDENCE {0.58; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股个股，股东友好型公司}
FAILURE_MODE {回购后减持（大股东套现）；业绩下滑（分红不可持续）；市场整体下跌（系统性风险）}

### 触发/解除/无效化条件

- **触发监控**：
  - 公司公告分红预案或分红方案
  - 股息率 >= 3%或分红率 >= 50%
  - 高送转方案（送转股比例 >= 10送5）
  - 公司公告回购或增发方案

- **解除监控**：
  - 分红方案实施完成（派息日后）
  - 除权除息日后30天（观察填权情况）
  - 分红方案取消或修改

- **无效化**：
  - 公司业绩大幅下滑（分红不可持续）
  - 公司退市或被收购
  - 分红方案被股东大会否决

### 阈值选择依据

- **4%股息率**：股息率 >= 4%在A股属于高股息，具有配置价值
- **30-70%分红率**：分红率在30-70%之间较为合理，过低说明分红不足，过高可能不可持续
- **5年连续分红**：连续分红5年以上说明分红稳定性好
- **80%分红率**：分红率 >= 80%可能说明公司缺乏增长机会或业绩见顶
- **10送5高送转**：送转股比例 >= 10送5在A股属于高送转，容易引发炒作

## 边界条件与降级策略

### 数据缺失/异常值处理

- **数据缺失**：
  - 分红数据缺失：使用公告数据补充，标注"手动补充"
  - 财务数据缺失：使用前期数据估算，标注"数据估算"
  - 股价数据缺失：使用前后交易日数据插值，标注"数据插值"

- **异常值处理**：
  - 分红率 > 100%：检查是否一次性收益或特殊分红
  - 股息率 > 10%：检查是否股价大幅下跌或特殊分红
  - 送转股比例 > 10送10：检查是否计算错误或极端高送转

### 降级策略

- **分红数据不完整**（< 3年）：
  - 使用当前年度分红数据
  - 无法计算分红连续性和稳定性
  - 降级标注："分红历史不足，无法评估稳定性"

- **财务数据缺失**：
  - 使用股息率代替分红率
  - 无法评估分红可持续性
  - 降级标注："财务数据缺失，无法评估分红质量"

- **除权除息日未到**：
  - 使用预案数据估算
  - 标注"分红方案未实施，数据为预案"
  - 降级标注："分红方案未实施，存在变更风险"

## A股特殊注意

### 1. T+1交易制度影响

- **除权除息日T+1交易**：除权除息日当天无法买入享受分红
- **股权登记日买入**：需在股权登记日收盘前买入才能享受分红
- **需要提前布局**：分红行情需要提前1-2周布局

### 2. 涨跌停限制影响

- **除权除息日涨跌停**：除权除息日股价可能涨跌停，影响交易
- **填权行情涨停**：高送转填权行情可能连续涨停
- **需要考虑流动性**：涨跌停限制影响分红套利策略

### 3. 停牌影响

- **分红方案公告停牌**：部分公司公告分红方案时临时停牌
- **除权除息日不停牌**：除权除息日正常交易
- **需要关注公告时间**：避免停牌期间无法交易

### 4. 税收影响

- **红利税**：持股1个月内20%，1个月-1年10%，1年以上免税
- **影响实际收益**：短期持股红利税较高，影响分红套利
- **需要考虑持股期限**：长期持股享受税收优惠

### 5. 高送转炒作

- **A股特色**：高送转是A股特有的炒作题材
- **小盘股为主**：小盘股高送转炒作更常见
- **填权概率低**：大部分高送转股票除权后难以填权
- **需要警惕炒作**：高送转炒作风险大，需谨慎参与

### 6. 分红连续性

- **监管要求**：证监会鼓励上市公司分红，部分行业有分红要求
- **再融资条件**：再融资需要满足分红条件（3年累计分红 >= 30%净利润）
- **需要关注政策**：分红政策影响公司分红决策

### 7. 大股东减持

- **分红后减持**：部分大股东分红后减持套现
- **需要关注减持公告**：分红后关注大股东减持动向
- **影响股价**：大股东减持可能导致股价下跌

### 8. 行业差异

- **周期行业**：周期行业分红波动大，景气高点分红率高
- **成长行业**：成长行业分红率低，资金用于再投资
- **公用事业**：公用事业分红率高且稳定
- **需要行业对比**：分红评估需要考虑行业特性

### 9. 市值效应

- **大盘股分红稳定**：大盘蓝筹股分红率高且稳定
- **小盘股分红不稳定**：小盘股分红波动大，可能不分红
- **需要控制市值**：分红策略需要考虑市值因素

### 10. 现金流质量

- **现金流充裕**：自由现金流充裕的公司分红可持续性强
- **现金流紧张**：现金流紧张的公司高分红不可持续
- **需要关注现金流**：评估分红质量需要关注现金流状况

## 回测说明（最小化）

- **回测设计**：
  - 构建高股息组合（股息率 >= 4%）
  - 每年调仓（分红方案公布后）
  - 计算组合收益和波动率

- **性能指标**：
  - 年化收益率
  - 夏普比率
  - 最大回撤
  - 相对市场超额收益

- **证伪条件**：
  - 高股息组合跑输市场（超额收益 < 0）
  - 夏普比率 < 市场（风险调整后收益差）

## 使用示例

### 示例1：高股息防御配置

**输入**：个股A，2024年分红方案
**数据**：
- 股息率：5.2%
- 分红连续性：8年
- 分红率：45%
- 市值：300亿（大盘蓝筹）
**输出**：
- 信号：高股息 + 稳定分红（防御性配置）
- **结论**：适合防御性配置，股价波动率低，下跌风险小

### 示例2：分红率突然提升

**输入**：个股B，2024年分红方案
**数据**：
- 分红率：85%（历史平均40%）
- 分红率增长：+50%
- ROE：12%（去年18%）
**输出**：
- 信号：分红率突然大幅提升（警惕业绩见顶）
- **结论**：公司可能缺乏增长机会，业绩见顶风险，谨慎参与

### 示例3：高送转炒作

**输入**：个股C，2024年高送转方案
**数据**：
- 送转股比例：10送8转
- 现金分红占比：15%
- 股价涨幅：+35%（公告前1个月）
- 市值：50亿（小盘股）
**输出**：
- 信号：高送转 + 低现金分红（警惕炒作）
- **结论**：高送转炒作风险大，除权后填权概率低，建议回避

### 示例4：分红 + 回购组合

**输入**：个股D，2024年分红 + 回购方案
**数据**：
- 股息率：3.5%
- 回购金额：5亿（市值200亿，占比2.5%）
- 分红连续性：6年
**输出**：
- 信号：分红 + 回购（股东友好信号）
- **结论**：公司股东友好，未来6-12个月股价跑赢市场概率高
