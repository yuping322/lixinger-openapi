# 方法论：行业板块分析器（A股）

本技能用于**分析A股行业板块的表现、资金流向、估值水平、景气度变化，识别行业轮动机会**。行业板块分析是自上而下投资的核心，行业轮动往往领先个股表现。

## 数据口径

### 数据源与字段映射

行业板块分析需要以下数据：
- **行业分类**：申万一级/二级/三级行业分类，AKShare (`stock_board_industry_name_em`)
  - 31个申万一级行业，约130个二级行业
  - 行业成分股列表，动态调整
- **行业行情**：AKShare (`stock_board_industry_hist_em`), 东方财富
  - 行业指数价格、涨跌幅、成交额、换手率
  - 日频数据，实时更新
- **行业资金流**：AKShare (`stock_sector_fund_flow_rank`), 东方财富
  - 行业主力资金净流入、流入强度
  - 日频数据，T日收盘后更新
- **行业估值**：根据成分股聚合计算
  - 行业PE、PB、PS、股息率
  - 周频/月频更新
- **行业景气度**：AKShare (`stock_industry_pe_ratio_cninfo`), 巨潮资讯
  - 行业盈利增速、ROE、毛利率
  - 季频数据，财报季更新

关键字段：
- `行业涨跌幅`：行业指数涨跌幅 (%)
- `行业相对强度`：行业涨跌幅 - 市场涨跌幅 (%)
- `行业资金流入`：行业主力资金净流入（亿元）
- `行业估值分位`：行业PE相对历史分位数 (%)
- `行业景气度`：行业盈利增速、ROE等综合指标

### 时间窗口与频率

- **日频监控**：每日收盘后更新行情和资金流
- **周频总结**：每周汇总行业表现、资金流向、轮动信号
- **月频分析**：每月分析行业估值、景气度、配置建议
- **季频深度**：每季度结合财报分析行业基本面变化

建议窗口：
- 短期动量：5日、10日、20日涨跌幅
- 中期趋势：60日、120日涨跌幅
- 长期配置：252日（年度）涨跌幅
- 估值分位：252日、1260日（5年）历史分位

## 核心指标

### 指标列表与计算公式

1. **行业表现指标**：
   - `行业涨跌幅 = (当前指数 - 前期指数) / 前期指数` (%)
   - `行业相对强度 = 行业涨跌幅 - 市场涨跌幅` (%)
   - `行业排名 = rank(行业涨跌幅)`（1-31，申万一级）
   - `行业动量 = (5日涨跌幅 + 20日涨跌幅) / 2`

2. **行业资金流指标**：
   - `行业资金净流入 = Σ(成分股主力净流入)`
   - `行业流入强度 = 行业资金净流入 / 行业成交额` (%)
   - `行业流入排名 = rank(行业资金净流入)`
   - `行业流入持续性 = 连续净流入天数`

3. **行业估值指标**：
   - `行业PE = Σ(成分股市值) / Σ(成分股净利润)`
   - `行业PB = Σ(成分股市值) / Σ(成分股净资产)`
   - `行业估值分位 = percentile_rank(行业PE, 252日历史)` (%)
   - `行业估值相对 = 行业PE / 市场PE`

4. **行业景气度指标**：
   - `行业盈利增速 = (当期净利润 - 去年同期) / 去年同期` (%)
   - `行业ROE = Σ(成分股净利润) / Σ(成分股净资产)` (%)
   - `行业景气度 = (盈利增速 + ROE) / 2`（简化）
   - `行业景气度变化 = 当期景气度 - 上期景气度`

5. **行业轮动指标**：
   - `行业轮动速度 = |当前排名 - 上期排名|`
   - `行业领先滞后 = 行业涨跌幅 vs 市场涨跌幅的时间差`
   - `行业轮动周期 = 行业从低点到高点的时间`

### 标准化方法

- 使用**百分位排名**对行业估值进行标准化（相对252日/1260日历史）
- 使用**相对强度**计算行业超额收益（行业涨跌幅 - 市场涨跌幅）
- 使用**z-score**检测极端表现（> 2倍标准差）
- 使用**移动平均**平滑短期波动（5日MA、20日MA）

## 信号与阈值

### 可测试规则（Insight Rules）

Rule 1 (行业相对强度 + 资金流入 → 行业领涨):
IF {20日行业相对强度 >= 5% AND 5日行业流入强度 >= 3% AND 行业排名 <= 5}
THEN {未来10-20个交易日，行业继续跑赢市场概率 >= 65%，预期超额收益 >= 3%}
CONFIDENCE {0.65; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {申万一级行业，流动性好的行业（金融、消费、科技）信号更强}
FAILURE_MODE {行业已大幅上涨（追高风险）；行业基本面转差；资金流入不持续；市场整体转弱（系统性风险）}

Rule 2 (行业估值低 + 景气度上升 → 配置机会):
IF {行业估值分位 <= 30% AND 行业景气度变化 > 0 AND 行业盈利增速 >= 10%}
THEN {未来60-120个交易日，行业上涨概率 >= 62%，预期涨幅 >= 15%}
CONFIDENCE {0.62; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {申万一级行业，周期性行业（有色、化工、机械）信号更强}
FAILURE_MODE {估值低是价值陷阱（基本面持续恶化）；景气度上升不持续；宏观环境转差；政策不利}

Rule 3 (行业轮动加速 → 短期机会):
IF {行业轮动速度 >= 10 AND 行业排名从 > 20 升至 <= 10 AND 5日涨跌幅 >= 5%}
THEN {未来5-10个交易日，行业继续上涨概率 >= 60%，预期涨幅 >= 3%}
CONFIDENCE {0.60; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {申万一级行业，题材性行业（传媒、计算机、军工）信号更强}
FAILURE_MODE {轮动是短期炒作（不持续）；行业已快速上涨（追高）；市场整体转弱；缺乏基本面支撑}

Rule 4 (行业估值高 + 资金流出 → 调整风险):
IF {行业估值分位 >= 80% AND 5日行业流入强度 <= -3% AND 行业景气度变化 < 0}
THEN {未来10-20个交易日，行业面临调整压力，预期下跌 >= 5%}
CONFIDENCE {0.63; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {申万一级行业，前期涨幅较大的行业}
FAILURE_MODE {估值高但景气度持续上升（戴维斯双击）；资金流出是获利了结（健康调整）；市场整体向好}

### 触发/解除/无效化条件

- **触发监控**：
  - 行业相对强度 >= 5%或 <= -5%（显著跑赢/跑输）
  - 行业排名变化 >= 10（轮动加速）
  - 行业估值分位 <= 20%或 >= 80%（极端估值）
  - 行业流入强度 >= 3%或 <= -3%（显著流入/流出）

- **解除监控**：
  - 行业相对强度回归正常（-2% ~ +2%区间持续5天）
  - 行业排名稳定（变化 < 3持续5天）
  - 行业估值回归中位（30%-70%区间）
  - 行业资金流恢复正常（-1% ~ +1%区间持续5天）

- **无效化**：
  - 数据缺失或延迟 > 2天（节假日除外）
  - 行业成分股大量停牌（> 30%）
  - 行业指数异常（计算错误）
  - 市场异常波动（熔断、涨跌停超过50%个股）

### 阈值选择依据

- **5%相对强度**：行业跑赢/跑输市场5%以上，是显著超额收益
- **3%流入强度**：行业资金流入占成交额3%以上，表明资金明显流入
- **30%/80%估值分位**：对应历史低估/高估区域
- **10%盈利增速**：行业盈利增速10%以上，表明景气度上升
- **10轮动速度**：行业排名变化10位以上，表明轮动加速

## 边界条件与降级策略

### 数据缺失/异常值处理

- **数据缺失**：
  - 节假日：使用前一交易日数据，标注"节假日无数据"
  - 系统故障：使用Wind/同花顺数据作为备份，标注"备用数据源"
  - 成分股停牌：从行业聚合中剔除，标注"剔除停牌股"

- **异常值处理**：
  - 行业涨跌幅 > 10%（单日）：检查是否数据错误或重大事件，标注"极端涨跌"
  - 行业PE < 0或 > 200：检查是否计算错误，标注"估值异常"
  - 行业流入强度 > 20%：检查是否数据错误，标注"极端流入"

### 降级策略

- **行业分类数据缺失**：
  - 备用分类1：中信一级/二级行业分类
  - 备用分类2：Wind一级/二级行业分类
  - 降级标注："使用备用行业分类，可能有差异"

- **行业估值数据缺失**：
  - 使用成分股市值加权平均PE/PB
  - 使用行业龙头股估值代替
  - 降级标注："估值数据不全，使用近似值"

- **行业景气度数据缺失**：
  - 使用行业价格走势代替景气度
  - 使用行业龙头股业绩代替
  - 降级标注："景气度数据不全，使用代理指标"

## A股特殊注意

### 1. T+1交易制度影响

- **当日买入次日才能卖出**：行业轮动时需要T+1日才能调仓，因此：
  - 行业轮动信号需要提前1天
  - 连续轮动比单日轮动更有操作性
  - 日内轮动无法参与（T+0限制）

### 2. 涨跌停限制影响

- **行业内个股涨跌停**：行业内个股涨跌停影响行业指数，导致：
  - 行业指数失真（涨停股无法买入）
  - 行业轮动延后（涨停打开后才能买入）
  - 需要关注行业内涨跌停个股占比

### 3. 停牌影响

- **行业内个股停牌**：行业内个股停牌影响行业指数和资金流，导致：
  - 行业指数失真（停牌股不计入）
  - 行业资金流失真（停牌股无交易）
  - 需要剔除停牌股重新计算

### 4. 政策驱动明显

- **政策对行业影响大**：A股是政策市，政策对行业影响显著：
  - 产业政策：新能源、半导体、医药等受政策影响大
  - 监管政策：地产、金融、教育等受监管影响大
  - 需要密切关注政策变化，及时调整行业配置

### 5. 行业轮动快

- **行业轮动周期短**：A股行业轮动快，题材炒作明显：
  - 行业轮动周期通常1-3个月
  - 题材性行业轮动更快（1-2周）
  - 需要及时跟踪行业轮动，避免错过机会

### 6. 行业集中度高

- **龙头股主导**：部分行业龙头股市值占比高，主导行业走势：
  - 金融：银行、保险龙头占比高
  - 消费：白酒、家电龙头占比高
  - 需要关注龙头股表现，避免被龙头股误导

### 7. 行业估值差异大

- **不同行业估值中枢不同**：
  - 成长性行业（科技、医药）：PE中枢30-50倍
  - 周期性行业（有色、化工）：PE中枢10-20倍
  - 稳定性行业（银行、公用事业）：PE中枢5-10倍
  - 需要使用行业历史分位数，避免跨行业比较

### 8. 行业景气度滞后

- **财报数据滞后**：行业景气度数据基于财报，滞后1-3个月：
  - 需要使用高频数据（价格、产量）补充
  - 需要关注行业龙头股业绩预告
  - 需要结合行业调研、草根调研

### 9. 行业分类调整

- **行业分类定期调整**：申万行业分类每年调整一次：
  - 成分股调整：个股从一个行业调整到另一个行业
  - 行业合并/拆分：行业分类结构调整
  - 需要关注行业分类调整，避免数据不连续

### 10. 北向资金偏好

- **外资行业偏好明显**：北向资金对行业有明显偏好：
  - 偏好行业：消费、医药、金融、科技龙头
  - 回避行业：周期、题材、小盘股
  - 需要结合北向资金行业配置，判断行业趋势

## 回测说明（最小化）

- **回测设计**：
  - 每日/每周收盘后计算信号
  - 次日/次周开盘买入行业ETF或成分股
  - 持有期：10日/20日/60日
  - 基准：沪深300、中证500

- **性能指标**：
  - 信号胜率（信号后N日行业跑赢市场概率）
  - 平均超额收益（行业收益 - 市场收益）
  - 信息比率、最大回撤

- **证伪条件**：
  - 信号胜率 <= 50%（不优于随机）
  - 平均超额收益 <= 0（不优于市场）
  - 信息比率 <= 0.5（风险调整后收益不佳）

## 使用示例

### 示例1：行业领涨

**输入**：新能源行业，2024-02-15收盘后
**数据**：
- 20日相对强度：+8.5%
- 5日流入强度：4.2%
- 行业排名：第3名
- 行业涨跌幅：+12.5%（20日）
**输出**：
- 信号：行业相对强度 + 资金流入（行业领涨）
- **结论**：新能源行业相对强度强，资金持续流入，短期继续看好

### 示例2：配置机会

**输入**：有色金属行业，2024-03-20收盘后
**数据**：
- 估值分位：25%（历史低位）
- 景气度变化：+5%（上升）
- 盈利增速：+15%
- 行业PE：18倍
**输出**：
- 信号：估值低 + 景气度上升（配置机会）
- **结论**：有色金属估值处于历史低位，景气度上升，中长期配置机会

### 示例3：行业轮动

**输入**：计算机行业，2024-04-10收盘后
**数据**：
- 轮动速度：12（排名从第25升至第13）
- 行业排名：第8名
- 5日涨跌幅：+7.2%
- 资金流入：+15亿
**输出**：
- 信号：行业轮动加速（短期机会）
- **结论**：计算机行业轮动加速，排名快速上升，短期关注

### 示例4：调整风险

**输入**：白酒行业，2024-05-15收盘后
**数据**：
- 估值分位：85%（历史高位）
- 5日流入强度：-4.5%（流出）
- 景气度变化：-3%（下降）
- 行业PE：45倍
**输出**：
- 信号：估值高 + 资金流出（调整风险）
- **结论**：白酒行业估值处于历史高位，资金流出，景气度下降，短期谨慎
