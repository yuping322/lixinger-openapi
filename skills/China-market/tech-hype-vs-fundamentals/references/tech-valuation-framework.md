## 估值体系

### 按公司阶段选择估值方法

| 阶段 | 特征 | 适用估值方法 |
|------|------|------------|
| 概念/早期 | 无收入或极少收入 | PS（极远期收入预测）、可比交易法、用户/客户数估值 |
| 高速增长期 | 收入快速增长但尚未盈利 | PS、EV/Revenue、订阅/客户生命周期价值 |
| 盈利初期 | 刚实现盈利，利润率不稳定 | PEG、PS、EV/EBITDA |
| 稳定盈利期 | 盈利稳定增长 | PE、PEG、DCF |
| 成熟期 | 增长放缓、高利润率 | PE、FCF收益率、分红 |

### 关键估值指标

| 指标 | 计算方式 | 适用场景 |
|------|---------|---------|
| PE（TTM） | 市值 / 近12月净利润 | 已盈利公司 |
| PE（预测） | 市值 / 一致预期净利润 | 盈利可预测公司 |
| PS（TTM） | 市值 / 近12月营收 | 亏损或低利润公司 |
| PEG | PE / 盈利增长率(%) | 比较不同增速公司 |
| EV/EBITDA | 企业价值 / 息税折旧摊销前利润 | 跨资本结构比较 |
| EV/Revenue | 企业价值 / 营收 | 亏损公司 |
| FCF收益率 | 自由现金流 / 市值 | 成熟科技公司 |

## 增长-估值映射

### 合理PS倍数参考（A股科技）

| 营收增速 | 毛利率 > 60% | 毛利率 40–60% | 毛利率 < 40% |
|---------|-------------|-------------|-------------|
| > 50% | 8–15x PS | 5–10x PS | 3–6x PS |
| 30–50% | 5–10x PS | 3–7x PS | 2–4x PS |
| 15–30% | 3–6x PS | 2–4x PS | 1–3x PS |
| < 15% | 2–4x PS | 1–3x PS | 0.5–2x PS |

*注：A股科技股因政策溢价、流动性溢价，估值中枢高于美股可比公司约20–40%。*

### 合理PE倍数参考

| 盈利增速 | 高确定性 | 中等确定性 | 低确定性 |
|---------|---------|----------|---------|
| > 50% | 40–60x | 30–50x | 20–40x |
| 30–50% | 25–40x | 20–35x | 15–25x |
| 15–30% | 18–30x | 15–25x | 10–20x |
| < 15% | 12–20x | 10–18x | 8–15x |

### PEG评估

| PEG | 评估 |
|-----|------|
| < 0.5 | 可能显著低估（需确认增长的可持续性） |
| 0.5–0.8 | 相对便宜 |
| 0.8–1.2 | 合理 |
| 1.2–2.0 | 略贵 |
| > 2.0 | 较贵，需很强的逻辑支撑 |

### "40法则"（Rule of 40）

```
40法则得分 = 营收增长率(%) + 营业利润率(%)
```

| 得分 | 评估 |
|------|------|
| > 60 | 顶级科技公司 |
| 40–60 | 优秀 |
| 25–40 | 良好 |
| 10–25 | 一般 |
| < 10 | 需关注 |

## 盈利质量分析

### 股份支付费用（SBC）调整

许多A股科技公司有大量员工股权激励。评估真实盈利时需加回SBC的影响：

```
SBC调整后净利润 = 报表净利润 − 股份支付费用
```

| SBC / 营收 | 评估 |
|-----------|------|
| < 5% | 合理 |
| 5–10% | 偏高，影响利润真实性 |
| 10–20% | 很高，Non-GAAP利润严重偏离 |
| > 20% | 过高，利润基本被稀释 |

### 研发资本化

| 资本化比例 | 评估 |
|-----------|------|
| 0%（全部费用化） | 保守，利润可靠 |
| < 30% | 合理范围 |
| 30–50% | 需关注理由 |
| > 50% | 激进，利润可能被美化 |

**检查方式**：
- 对比同行业公司的研发资本化比例
- 资本化比例大幅上升 → 利润质量下降信号
- 无形资产摊销 vs 新增资本化 → 净影响

### 收现比

```
收现比 = 经营活动现金流 / 净利润
```

| 收现比 | 评估 |
|--------|------|
| > 100% | 优秀，利润质量高 |
| 80–100% | 良好 |
| 50–80% | 一般，需了解原因 |
| < 50% | 差，大量利润未转化为现金 |

### 经营杠杆分析

```
经营杠杆 = 营业利润增速 / 营收增速
```

| 经营杠杆 | 含义 |
|---------|------|
| > 1.5 | 强经营杠杆 — 收入增长有效放大利润 |
| 1.0–1.5 | 适度杠杆 |
| 0.5–1.0 | 弱杠杆 — 费用增速接近收入增速 |
| < 0.5 | 无杠杆 — 增收不增利 |

## A股科技特殊因素

### 政策溢价

A股科技公司在特定政策支持方向上享有"国产替代溢价"：

| 领域 | 政策支持力度 | 估值溢价幅度 | 代表政策 |
|------|------------|-----------|---------|
| 半导体/芯片 | 极高 | +30–50% vs 国际可比 | 大基金、税收优惠 |
| 信创/国产软件 | 高 | +20–40% | 政府采购倾斜 |
| AI/大模型 | 高 | +20–40% | 数字经济发展战略 |
| 新能源设备 | 高（但边际递减） | +10–30% | 碳中和、补贴 |
| 军工电子 | 高 | +20–40% | 国防现代化 |
| 云计算/SaaS | 中 | +10–20% | 数字化转型 |
| 机器人 | 中（快速上升） | +20–40% | 制造业升级 |

### 概念股识别

**"纯概念"信号**（无实质收入支撑）：

1. 公司公告/互动易回复提及热门概念（AI、机器人、量子等），但：
   - 相关收入占比 < 5%
   - 无实际产品交付
   - 仅有"研发布局"或"战略合作"
2. 股价在概念炒作期间涨幅 > 50%，但期间业绩预告未上调
3. 龙虎榜显示游资（短线资金）主导交易

### 解禁压力评估

| 解禁规模 / 流通市值 | 评估 |
|-------------------|------|
| < 10% | 影响有限 |
| 10–30% | 中等压力 |
| 30–50% | 较大压力 |
| > 50% | 重大压力 — 严格关注解禁日期和大股东减持意愿 |

## 错误定价识别框架

### 高估定价特征

| 特征 | 严重程度 |
|------|---------|
| PS > 合理区间上限的2倍 | 🔴 严重 |
| 无盈利、无时间表，PE无意义 | 🔴 严重 |
| 核心概念收入占比 < 10% | 🔴 严重 |
| 研发资本化 > 50% 且上升中 | 🟡 中度 |
| SBC > 营收的15% | 🟡 中度 |
| 解禁规模 > 流通市值的30% | 🟡 中度 |
| 客户集中度 > 50%（前五大） | 🟡 中度 |
| 应收账款增速 > 营收增速 | 🟡 中度 |

### 低估定价特征

| 特征 | 支持强度 |
|------|---------|
| PEG < 0.8 且盈利增长可持续 | 🟢 强 |
| 40法则得分 > 40 但PS < 同行中位数 | 🟢 强 |
| 国产替代实质性进展（收入已兑现）但市场仍给折价 | 🟢 强 |
| FCF收益率 > 5%（科技公司中罕见） | 🟢 强 |
| 分析师覆盖 < 5家（信息不对称） | 🟡 中度 |
| 海外收入占比提升但未获重估 | 🟡 中度 |
| 机构持仓低位但基本面趋势向好 | 🟡 中度 |

### 错误定价解释框架

对于每只被识别为错误定价的股票，需回答：

**如果高估**：
1. 市场在定价什么预期？（例：假设AI收入3年内占比达50%）
2. 该预期实现的合理概率是多少？（例：根据公司产品成熟度和行业增速，概率约15%）
3. 如果预期不能实现，合理估值是多少？（例：回归传统IT公司估值，下行空间40%）

**如果低估**：
1. 市场在忽视什么？（例：公司半导体材料已通过客户验证，进入量产爬坡期）
2. 为什么忽视？（例：公司历史上是传统化工企业，新业务尚未被重新定价）
3. 催化剂是什么？（例：下一季报新业务收入占比首次超过20%，触发重估）

## 常见估值陷阱

### 高增长陷阱

- **低基数效应**：去年基数极低导致同比增速虚高
- **项目制收入波动**：单个大项目贡献的高增长不可持续
- **补贴驱动增长**：政府补贴推动的营收，补贴退坡后增速骤降
- **并表增长**：通过收购实现的增长，非内生能力体现

### A股科技估值特有问题

- **同概念不同公司差异极大**：同为"AI概念"，有的有实际技术和收入，有的只有概念。不能用同一估值框架
- **科创板公司多数亏损**：50%+科创板公司未实现盈利，传统PE完全失效
- **流动性溢价**：A股科技板块在增量资金充裕时享有明显流动性溢价，流动性收缩时回撤更猛
- **散户定价权**：热门科技题材受散户情绪影响大，基本面分析的定价效率有限（但长期终将回归）
