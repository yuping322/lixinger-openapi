# 方法论：可转债扫描器（A股）

本技能用于**扫描和分析A股可转债市场，识别低估值、高转股溢价率、强赎机会等投资机会，评估可转债的债性和股性**。可转债是A股重要的投资品种，兼具债券的安全性和股票的进攻性，是攻守兼备的投资工具。

## 数据口径

### 数据源与字段映射

可转债扫描需要以下数据：
- **可转债数据**：AKShare (`bond_zh_cov`), 集思录、东方财富
  - 转债代码、转债名称、转债价格、转股价、正股价格、转股价值、转股溢价率
  - 日频数据，实时更新
- **转债条款**：AKShare (`bond_cov_comparison`), 公司公告
  - 到期时间、票面利率、回售条款、强赎条款、下修条款
  - 发行时确定，条款变更时更新
- **正股数据**：AKShare (`stock_zh_a_hist`)
  - 正股价格、涨跌幅、成交量、市值
  - 日频数据，实时更新
- **财务数据**：AKShare (`stock_financial_abstract_em`)
  - 净资产、净利润、资产负债率、信用评级
  - 季度更新（财报披露时）

关键字段：
- `转债价格`：可转债的市场价格（元）
- `转股价`：转换为正股的价格（元/股）
- `正股价格`：对应正股的市场价格（元/股）
- `转股价值 = 正股价格 / 转股价 × 100`（元）
- `转股溢价率 = (转债价格 - 转股价值) / 转股价值` (%)
- `纯债价值`：按票面利率和到期时间折现的债券价值（元）
- `纯债溢价率 = (转债价格 - 纯债价值) / 纯债价值` (%)
- `双低指标 = 转债价格 + 转股溢价率`

### 时间窗口与频率

- **日频监控**：每日收盘后更新，监控转债价格、转股溢价率变化
- **周频总结**：每周汇总新发行转债、强赎预警、下修预期
- **月频深度分析**：结合财报数据，分析正股基本面

建议窗口：
- 短期交易：5日、10日转债价格和溢价率变化
- 中期持有：60日、120日正股走势和转债表现
- 长期配置：到期时间、票面利率、信用评级
- 强赎监控：连续N日正股价格 >= 转股价 × 130%

## 核心指标

### 指标列表与计算公式

1. **估值指标**：
   - `转股价值 = 正股价格 / 转股价 × 100`（元）
   - `转股溢价率 = (转债价格 - 转股价值) / 转股价值` (%)
   - `纯债价值 = Σ(票面利息 / (1+r)^t) + 100 / (1+r)^T`（元）
   - `纯债溢价率 = (转债价格 - 纯债价值) / 纯债价值` (%)
   - `双低指标 = 转债价格 + 转股溢价率`（越低越好）

2. **债性指标**：
   - `到期收益率 = (100 + 累计利息 - 转债价格) / 转债价格 / 剩余年限` (%)
   - `债底保护 = 纯债价值 / 转债价格` (%)
   - `信用评级`：AAA/AA+/AA/AA-/A+/A（越高越好）
   - `回售触发距离 = (回售触发价 - 正股价格) / 正股价格` (%)

3. **股性指标**：
   - `转股溢价率`：越低越好（< 10%为强股性）
   - `正股波动率 = std(正股日收益率)` over 60日 (%)
   - `正股涨跌幅`：20日、60日、120日涨跌幅 (%)
   - `Delta = ∂转债价格 / ∂正股价格`（转债对正股的敏感度）

4. **强赎指标**：
   - `强赎触发距离 = (转股价 × 130% - 正股价格) / 正股价格` (%)
   - `连续满足强赎天数 = count(正股价格 >= 转股价 × 130%)` over 连续交易日
   - `强赎概率 = f(连续满足天数, 正股走势, 公司意愿)` (0-100)
   - `强赎收益 = (130 - 转债价格) / 转债价格` (%)

5. **下修指标**：
   - `下修触发距离 = (转股价 × 80% - 正股价格) / 正股价格` (%)
   - `连续满足下修天数 = count(正股价格 < 转股价 × 80%)` over 连续交易日
   - `下修概率 = f(连续满足天数, 正股走势, 公司意愿, 股东利益)` (0-100)
   - `下修后转股溢价率 = (转债价格 - 正股价格 / 下修后转股价 × 100) / (正股价格 / 下修后转股价 × 100)` (%)

### 标准化方法

- 使用**百分位排名**对转股溢价率、双低指标进行标准化（相对全市场）
- 使用**评分**（0-100）量化强赎概率、下修概率
- 使用**等级分类**（强债性/平衡/强股性）对转债进行分类
- 使用**行业相对**比较正股基本面（个股 vs 行业中位数）

## 信号与阈值

### 可测试规则（Insight Rules）

Rule 1 (双低策略 → 低风险高收益):
IF {双低指标 < 120 AND 转债价格 < 110 AND 转股溢价率 < 20% AND 纯债价值 > 90}
THEN {未来3-6个月，转债价格上涨概率 >= 70%，预期收益 >= 5%，下跌风险有限}
CONFIDENCE {0.68; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股可转债市场，双低策略是经典策略}
FAILURE_MODE {正股大幅下跌（基本面恶化）；市场整体下跌（系统性风险）；转债被强赎（收益封顶）；信用风险（公司违约）}

Rule 2 (强赎预期 → 短期套利):
IF {连续满足强赎天数 >= 10 AND 正股走势强劲（20日涨幅 >= 15%）AND 转债价格 < 125}
THEN {未来1-2个月，转债价格上涨至130附近，预期收益 = (130 - 转债价格) / 转债价格}
CONFIDENCE {0.72; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股可转债市场，强赎是重要的套利机会}
FAILURE_MODE {正股回调（跌破强赎线）；公司延迟强赎（拖延时间）；转债价格已接近130（收益空间小）；市场整体下跌}

Rule 3 (下修预期 → 博弈机会):
IF {连续满足下修天数 >= 10 AND 正股价格 < 转股价 × 70% AND 公司有下修意愿（大股东持股高）}
THEN {未来1-3个月，转股价下修概率 >= 60%，下修后转债价格上涨 >= 10%}
CONFIDENCE {0.60; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股可转债市场，下修是重要的博弈机会，但不确定性高}
FAILURE_MODE {公司不下修（大股东不愿稀释股权）；下修方案被否决（股东大会）；正股继续下跌（下修后仍亏损）；市场整体下跌}

Rule 4 (高债性防守 → 熊市配置):
IF {纯债价值 > 95 AND 转债价格 < 105 AND 到期收益率 > 2% AND 信用评级 >= AA}
THEN {未来1-2年，转债价格下跌风险有限（债底保护），到期收益 >= 2%，适合熊市配置}
CONFIDENCE {0.75; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {A股可转债市场，熊市时债性强的转债是避险工具}
FAILURE_MODE {信用风险（公司违约、评级下调）；利率上升（债券价格下跌）；转债被强赎（收益封顶）}

### 触发/解除/无效化条件

- **触发监控**：
  - 双低指标 < 120（低估值机会）
  - 连续满足强赎天数 >= 10（强赎预期）
  - 连续满足下修天数 >= 10（下修预期）
  - 转债价格 < 100（破面值，安全边际高）
  - 转股溢价率 < 5%（强股性，跟随正股）
  - 新发行转债（打新机会）

- **解除监控**：
  - 转债被强赎（收益兑现）
  - 转股价下修完成（博弈结束）
  - 转债价格 > 130（收益空间小）
  - 转股溢价率 > 30%（估值过高）
  - 正股基本面恶化（信用风险上升）

- **无效化**：
  - 转债到期（自动赎回）
  - 转债提前赎回（公司行权）
  - 正股退市（转债失去价值）
  - 公司违约（信用风险实现）

### 阈值选择依据

- **120双低指标**：经典的双低策略阈值，历史表现良好
- **110转债价格**：低价转债，安全边际高，上涨空间大
- **20%转股溢价率**：合理的溢价水平，不过高也不过低
- **130%强赎触发**：法定的强赎条件（连续N日正股价格 >= 转股价 × 130%）
- **80%下修触发**：常见的下修条款（连续N日正股价格 < 转股价 × 80%）
- **90纯债价值**：债底保护较好，下跌风险有限
- **2%到期收益率**：合理的债券收益率，高于无风险利率

## 边界条件与降级策略

### 数据缺失/异常值处理

- **数据缺失**：
  - 转债价格缺失：使用前一交易日价格，标注"数据未更新"
  - 转股价缺失：使用公告披露的转股价，标注"使用公告数据"
  - 纯债价值缺失：使用简化公式估算，标注"估算值"

- **异常值处理**：
  - 转债价格 > 200：检查是否有特殊条款（如附加期权），若无则标注"价格异常"
  - 转股溢价率 > 100%：检查是否正股停牌或转债炒作，标注"溢价过高"
  - 转股溢价率 < -10%：检查是否有套利空间（转股卖出），标注"套利机会"

### 降级策略

- **AKShare接口不稳定**：
  - 备用数据源1：集思录（`https://www.jisilu.cn/data/cbnew/`）
  - 备用数据源2：东方财富可转债（`http://data.eastmoney.com/kzz/`）
  - 备用数据源3：手动输入（从交易软件获取）
  - 降级标注："使用备用数据源，可能有延迟"

- **转债条款缺失**：
  - 使用公司公告（募集说明书）
  - 使用集思录数据库
  - 使用行业平均条款（如强赎130%、下修80%）
  - 降级标注："条款为估计值，以公告为准"

- **纯债价值计算失败**：
  - 使用简化公式（忽略复杂条款）
  - 使用市场平均纯债价值（同评级、同期限）
  - 降级标注："纯债价值为估算值"

## A股特殊注意

### 1. T+1交易制度影响

- **转债T+0交易**：可转债实行T+0交易（当日买入当日可卖出），不同于股票的T+1：
  - 日内交易机会（高频套利）
  - 风险控制更灵活（可当日止损）
  - 波动可能更大（短线资金炒作）

### 2. 涨跌停限制影响

- **转债涨跌幅±20%**（2020年后）：比股票（±10%）更宽松：
  - 单日波动更大（风险和机会并存）
  - 强赎预期时可能大涨（单日+20%）
  - 正股跌停时转债可能大跌（单日-20%）

- **临时停牌机制**：转债涨跌幅达到±20%时临时停牌30分钟：
  - 冷静期（避免过度炒作）
  - 复牌后可能继续波动

### 3. 停牌影响

- **正股停牌转债继续交易**：正股停牌时转债仍可交易：
  - 转债价格反映市场预期（重组、业绩等）
  - 正股复牌后转债价格可能大幅调整
  - 需要关注正股停牌原因

### 4. 强赎条款差异

- **强赎触发条件**：通常为"连续N日（15-30日）正股价格 >= 转股价 × 130%"：
  - N值越小，强赎越容易触发
  - 部分转债有"任意N日"条款（更容易触发）
  - 需要仔细阅读募集说明书

- **强赎公告到执行**：通常有1个月的转股期：
  - 投资者可选择转股或卖出
  - 转债价格通常上涨至130附近（套利空间收窄）
  - 逾期未转股将被强制赎回（损失溢价）

### 5. 下修条款差异

- **下修触发条件**：通常为"连续N日（15-30日）正股价格 < 转股价 × 80%"：
  - 触发后公司有权下修（非强制）
  - 需要董事会决议和股东大会表决
  - 大股东持股高时下修概率更大（避免转债回售）

- **下修幅度**：通常下修至正股价格的90-100%：
  - 下修幅度越大，转债价值提升越多
  - 需要关注下修预案公告

### 6. 回售条款

- **回售触发条件**：通常为"连续N日（30日）正股价格 < 转股价 × 70%"：
  - 投资者有权以面值+利息回售给公司
  - 保护投资者利益（债底保护）
  - 公司为避免回售通常会下修转股价

### 7. 转股期

- **转股期**：通常为发行后6个月至到期前：
  - 转股期前无法转股（仅能交易转债）
  - 转股期内可随时转股
  - 转股后获得正股（T+1可卖出）

### 8. 可转债打新

- **打新规则**：可转债打新无需市值，中签后缴款：
  - 中签率通常较高（1-5%）
  - 上市首日通常上涨（10-30%）
  - 破发风险较低（债底保护）
  - 需要关注正股基本面和转债条款

### 9. 转债炒作

- **小盘转债炒作**：规模小（< 5亿）的转债容易被炒作：
  - 流动性差（日成交额 < 1000万）
  - 价格波动大（单日±20%）
  - 脱离正股走势（溢价率过高）
  - 需要警惕炒作风险

### 10. 信用风险

- **公司违约风险**：转债是公司债务，存在违约风险：
  - 关注信用评级（AAA > AA+ > AA > AA- > A+）
  - 关注财务指标（资产负债率、现金流）
  - 关注行业风险（周期性行业、产能过剩）
  - 违约时转债价格可能暴跌（损失本金）

## 回测说明（最小化）

- **回测设计**：
  - 每日收盘后计算双低指标、强赎概率、下修概率
  - 根据策略买入/卖出转债
  - 持有期：根据策略不同（双低3-6个月，强赎1-2个月）
  - 基准：中证转债指数

- **性能指标**：
  - 策略胜率（买入后上涨概率）
  - 平均收益率（买入后N日收益）
  - 最大回撤、夏普比率
  - 年化收益率

- **证伪条件**：
  - 策略胜率 <= 60%（不显著优于随机）
  - 平均收益率 <= 0（不盈利）
  - 夏普比率 <= 1（风险调整后收益不佳）

- **敏感性分析**：
  - 测试不同阈值（双低120 vs 130）
  - 测试不同市场环境（牛市 vs 熊市）
  - 测试不同规模（大盘 vs 小盘转债）
  - 测试不同策略（双低 vs 强赎 vs 下修）

## 使用示例

### 示例1：双低策略

**输入**：某转债（规模10亿）
**数据**：
- 转债价格：105元
- 转股溢价率：12%
- 双低指标：117
- 纯债价值：92元
- 正股基本面：良好

**输出**：
- 策略：双低策略
- 预期收益：5-10%（3-6个月）
- 风险评估：低（债底保护好）
- **结论**：符合双低策略，建议买入，安全边际高

### 示例2：强赎套利

**输入**：某转债（规模8亿）
**数据**：
- 转债价格：122元
- 正股价格：15元
- 转股价：10元
- 连续满足强赎天数：12天
- 正股20日涨幅：18%

**输出**：
- 策略：强赎套利
- 预期收益：(130-122)/122 = 6.6%
- 强赎概率：85%
- **结论**：强赎预期强烈，建议买入，目标价130元

### 示例3：下修博弈

**输入**：某转债（规模5亿）
**数据**：
- 转债价格：95元
- 正股价格：6元
- 转股价：10元
- 连续满足下修天数：15天
- 大股东持股比例：55%

**输出**：
- 策略：下修博弈
- 下修概率：70%
- 预期下修后转股价：6.5元
- 预期收益：10-15%
- **结论**：下修预期较强，可博弈，但风险较高

### 示例4：高债性防守

**输入**：某转债（规模20亿）
**数据**：
- 转债价格：102元
- 纯债价值：96元
- 到期收益率：2.5%
- 信用评级：AAA
- 剩余期限：3年

**输出**：
- 策略：高债性防守
- 债底保护：96/102 = 94%
- 下跌风险：有限（最多跌至96元）
- **结论**：适合熊市配置，安全性高，到期收益稳定
