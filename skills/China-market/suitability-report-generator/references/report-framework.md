## 风险分类

### 系统性（市场层面）风险

| 风险 | 描述 | 典型披露措辞 |
|------|------|-------------|
| 权益市场风险 | 整体市场下跌 | "权益投资面临市场风险，可能在市场下行时产生亏损" |
| 利率风险 | 利率变动→债券/股票重新定价 | "利率上升可能导致债券价格下跌，并可能对利率敏感型股票产生负面影响" |
| 通胀风险 | 购买力侵蚀 | "通货膨胀可能侵蚀投资回报的实际购买力" |
| 汇率风险 | 人民币汇率波动（QDII/港股通） | "海外投资面临人民币汇率波动风险" |
| 经济下行风险 | 经济收缩 | "经济下行可能影响企业盈利和股票价格" |
| 政策风险 | 监管政策变化 | "监管政策调整可能对相关行业和个股产生重大影响" |

### 非系统性（个体）风险

| 风险 | 描述 | 何时披露 |
|------|------|---------|
| 个股风险 | 单个公司经营失败 | 任何个股仓位 |
| 行业风险 | 特定行业低迷 | 行业集中度 > 20% |
| 信用/违约风险 | 债券发行人违约 | 任何信用债敞口 |
| 流动性风险 | 无法快速卖出 | 小盘股、微盘股、低成交量个股 |
| 集中度风险 | 过度集中于少数仓位 | 前5大 > 40% |
| 退市风险 | 因触及退市条件被终止上市 | ST/*ST个股、面值退市风险股 |
| 股权质押风险 | 大股东质押比例过高 | 控股股东质押超50% |

### A股特有风险

| 风险 | 描述 | 披露建议 |
|------|------|---------|
| 涨跌停风险 | 10%/20%涨跌幅限制导致无法止损 | 高波动个股 |
| T+1风险 | 当日买入次日方可卖出 | 短线策略 |
| 大股东减持风险 | 限售股解禁后抛售压力 | 近期有大额解禁的个股 |
| 注册制退市风险 | 注册制下退市常态化 | 基本面较弱个股 |
| 关联交易风险 | 控股股东通过关联交易侵占利益 | 民企、家族企业 |
| 北向资金波动风险 | 外资大幅流出影响核心资产 | 外资重仓股 |
| 政策调控风险 | 行业政策突变（如互联网反垄断、教培双减） | 政策敏感行业 |

### 行为风险

| 风险 | 描述 | 缓释措辞 |
|------|------|---------|
| 恐慌抛售 | 在下跌中卖出，锁定亏损 | "在市场下行时保持投资纪律对长期收益至关重要" |
| 追涨杀跌 | 在强势上涨后买入 | "过往业绩不代表未来表现" |
| 过度自信 | 获利后过度承担风险 | "投资结果具有不确定性，无论分析多严谨" |
| 频繁交易 | 过度交易导致成本侵蚀收益 | "定期再平衡有助于维持目标配置，但过度交易会增加成本" |

## 适当性评估框架

### 风险偏好映射

| 声明偏好 | 可接受 | 不可接受 | 适合的策略 |
|---------|--------|---------|----------|
| 保守 | 年度回撤-10%至-15% | > -20%回撤 | 高等级债券、货币基金、低波动蓝筹、可转债 |
| 稳健 | 年度回撤-15%至-25% | > -30%回撤 | 均衡配置、宽基指数、核心-卫星策略 |
| 积极 | 年度回撤-25%至-40% | > -50%回撤 | 成长型配置、行业主题、中小盘 |

### 投资期限匹配

| 投资期限 | 适合 | 不适合 |
|---------|------|--------|
| < 1年 | 货币基金、短期理财、国债逆回购 | 个股、长期限债基、另类 |
| 1–3年 | 纯债基金、均衡配置 | 高集中度权益、流动性差的产品 |
| 3–5年 | 偏股混合、指数基金、适度风险 | 极高波动策略 |
| 5年+ | 高权益配置、另类投资 | —（大多数策略可适用） |

### 财务状况评估

| 因素 | 绿灯 | 黄灯 | 红灯 |
|------|------|------|------|
| 应急储备 | 6个月+生活开支 | 3–6个月 | < 3个月 |
| 负债状况 | 无高息贷款 | 有可管理的贷款 | 大额房贷+消费贷 |
| 投资占净资产比 | < 30%流动净资产 | 30–60% | > 60% |
| 收入稳定性 | 稳定工资收入 | 浮动收入 | 收入有中断风险 |
| 社保/保险 | 五险一金+商业保险 | 有基本保障 | 保障缺失 |

### 适格投资者门槛

| 板块/产品 | 资产要求 | 经验要求 | 其他 |
|----------|---------|---------|------|
| 科创板 | 证券账户及资金≥50万元 | ≥24个月证券交易经验 | 需签署风险揭示书 |
| 北交所 | 证券账户及资金≥50万元 | ≥24个月证券交易经验 | — |
| 创业板 | 证券账户及资金≥10万元 | ≥24个月证券交易经验 | 2020年4月28日后新开户适用 |
| 融资融券 | 证券账户及资金≥50万元 | ≥6个月证券交易经验 | 需通过适当性评估 |
| 分级基金B | 证券账户及资金≥30万元 | — | 已停止新分级基金发行 |
| 私募基金 | 金融资产≥300万元 或 近3年年均收入≥50万元 | — | 单只最低投资100万元 |
| QDII基金 | 无门槛（公募QDII） | — | 额度可能受限 |
| 港股通 | 证券账户及资金≥50万元 | — | 沪港通/深港通 |

## 监管背景

### 中国投资者适当性管理制度

**核心法规**：
- 《证券期货投资者适当性管理办法》（2017年7月1日施行）
- 各交易所/行业协会配套细则

**关键要求**：
1. **投资者分类**：专业投资者 vs 普通投资者
2. **产品分级**：按风险等级（R1–R5）对产品进行分级
3. **匹配原则**：产品风险等级不得超过投资者风险承受能力
4. **告知义务**：充分揭示产品风险，获取投资者确认
5. **回访确认**：重大交易需进行回访确认
6. **信息留存**：保存适当性评估记录，备查不少于20年

### 投资者风险等级与产品风险等级匹配

| 投资者风险等级 | 可购买产品风险等级 |
|-------------|--------------|
| C1（保守型） | R1 |
| C2（稳健型） | R1、R2 |
| C3（平衡型） | R1、R2、R3 |
| C4（积极型） | R1、R2、R3、R4 |
| C5（激进型） | R1、R2、R3、R4、R5 |

### 关键文档措辞

使用以下框架确保合规：

- "根据[客户]提供的风险偏好、投资期限和财务状况信息……"
- "已识别并披露以下风险因素……"
- "考虑的替代方案包括[X]、[Y]和[Z]。选择当前方案的原因是……"
- "本建议符合[客户]声明的投资目标，即……"
- "本建议所依据的关键假设包括……"

## 文档标准

### 完整性检查清单

| 要素 | 必需 | 备注 |
|------|------|------|
| 客户身份信息 | 是 | 姓名/编号、日期 |
| 投资建议内容 | 是 | 具体推荐什么 |
| 投资理由 | 是 | 为何适合 |
| 风险披露 | 是 | 所有重大风险 |
| 适当性评估 | 是 | 正式评价 |
| 假设条件 | 是 | 关键输入和局限 |
| 免责声明 | 是 | 合规措辞 |
| 备选方案 | 建议 | 其他选项及未选原因 |
| 费用披露 | 建议 | 实施成本 |
| 复评日期 | 建议 | 何时重新评估 |

### 语气和措辞

- 使用主动语态："我们建议……"而非"建议如下……"
- 具体化："组合35%的电力设备行业敞口形成了行业集中风险"而非"存在一定风险"
- 尽可能量化："在2022年类似的市场调整情景下，组合预计下跌约-22%"
- 避免绝对措辞："预计"→"估计"或"预期"
- 同时包含正面和负面情景
