# 港股南向资金流向分析器

深度分析南向资金的流向模式、持仓变化和投资偏好，为港股投资提供资金面洞察。

## 核心功能

### 💰 资金流向监控
- 每日南向资金净流入/流出
- 沪深港通分别统计
- 历史资金流向趋势
- 资金流向与市场表现关联

### 📊 持仓分析
- 南向资金持仓明细
- 持仓市值变化
- 新增/减持股票
- 持仓集中度分析

### 🏢 行业偏好
- 各行业资金流入分布
- 行业持仓占比变化
- 热门行业轮动
- 行业估值与资金流向关系

### 🎯 个股关注
- 南向资金重点持仓个股
- 大额买入/卖出个股
- 新进南向资金个股
- 持仓比例变化大的个股

## 使用场景

### 投资决策
- 跟踪聪明资金动向
- 识别内地资金偏好
- 把握市场资金面变化
- 制定跟投策略

### 风险管理
- 监控资金流出风险
- 识别流动性风险
- 评估市场情绪变化
- 调整仓位配置

### 研究分析
- 南向资金行为研究
- 资金流向与收益关系
- A/H股联动分析
- 市场风格偏好研究

## 分析维度

### 时间维度
- **日内**: 实时资金流向监控
- **日度**: 每日资金流向统计
- **周度**: 周度资金流向趋势
- **月度**: 月度资金流向总结
- **年度**: 年度资金流向回顾

### 空间维度
- **整体市场**: 港股整体资金流向
- **行业板块**: 各行业资金分布
- **个股层面**: 具体个股资金流向
- **市值分层**: 大中小盘资金偏好

### 行为维度
- **流入模式**: 持续流入vs间歇流入
- **流出模式**: 获利了结vs风险规避
- **轮动模式**: 行业间资金轮动
- **集中度**: 资金集中vs分散投资

## 技术指标

### 资金流向指标
- **净流入率**: 净流入/总流入
- **流入持续性**: 连续流入天数
- **流入强度**: 单日流入/平均流入
- **流入动量**: 资金流向变化率

### 持仓指标
- **持仓集中度**: 前十大持仓占比
- **持仓变化率**: 持仓市值变化速度
- **换手率**: 买入/卖出与持仓比例
- **持仓稳定性**: 持仓时间长度

### 偏好指标
- **行业偏好度**: 行业持仓/基准权重
- **市值偏好**: 大中小盘持仓分布
- **估值偏好**: PE/PB分布特征
- **成长偏好**: 成长vs价值倾向

## 数据源

### 主要数据源
- **港交所**: 港股通官方数据
- **上交所**: 沪港通统计数据
- **深交所**: 深港通统计数据
- **各大券商**: 资金流向研究报告

### 数据更新频率
- **实时数据**: 交易时间内每5分钟更新
- **日度数据**: 每日收盘后更新
- **持仓数据**: 每月更新
- **历史数据**: 2014年11月互联互通开通以来

## 使用方法

### 基础查询

## 注意事项

1. **数据延迟**: 南向资金数据有T+1延迟
2. **汇率影响**: 资金以港币计价，需考虑汇率波动
3. **交易时间**: 港股通交易时间与港股一致
4. **额度限制**: 南向资金有每日额度限制
5. **节假日**: 内地和香港节假日影响交易
