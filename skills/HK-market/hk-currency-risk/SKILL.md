# 港股汇率风险监控器

专业的港股汇率风险管理工具，提供汇率风险监控、敞口分析和对冲策略建议。

- 汇率波动率分析
- 汇率趋势预测

- 对冲比例优化

## 汇率指标

### 基础汇率指标
- **即期汇率**: 当前市场汇率
- **远期汇率**: 未来交割汇率
- **汇率升贴水**: `远期汇率 - 即期汇率`
- **汇率波动率**: 汇率历史波动率

### 风险敞口指标
- **汇率敞口**: `外币资产 × 汇率敏感度`
- **净敞口**: `资产敞口 - 负债敞口`
- **集中度**: `单一货币敞口 / 总敞口`
- **对冲比例**: `对冲金额 / 总敞口`

### 风险价值指标
- **VaR**: 在险价值计算
- **CVaR**: 条件风险价值
- **期望短缺**: 平均损失超过VaR的部分
- **最大回撤**: 最大可能损失

## 使用方法

### 基础查询
```bash

# 获取风险敞口

# 获取对冲策略
```

## 注意事项

1. **汇率波动**: 汇率波动可能很大
2. **对冲成本**: 对冲需要考虑成本
3. **流动性**: 部分货币对流动性较差
4. **政策影响**: 央行政策影响汇率
5. **时区差异**: 不同市场时区影响
