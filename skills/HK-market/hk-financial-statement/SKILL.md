## 财务比率

### 盈利能力比率
- **毛利率**: `(营业收入 - 营业成本) / 营业收入`
- **营业利润率**: `营业利润 / 营业收入`
- **净利率**: `净利润 / 营业收入`
- **ROE**: `净利润 / 股东权益`
- **ROA**: `净利润 / 总资产`
- **ROIC**: `NOPAT / 投入资本`

### 偿债能力比率
- **流动比率**: `流动资产 / 流动负债`
- **速动比率**: `(流动资产 - 存货) / 流动负债`
- **资产负债率**: `总负债 / 总资产`
- **利息保障倍数**: `EBIT / 利息费用`
- **债务权益比**: `总负债 / 股东权益`

### 运营效率比率
- **总资产周转率**: `营业收入 / 总资产`
- **存货周转率**: `营业成本 / 存货`
- **应收账款周转率**: `营业收入 / 应收账款`
- **应付账款周转率**: `营业成本 / 应付账款`
- **现金转换周期**: `存货周转天数 + 应收账款周转天数 - 应付账款周转天数`

### 成长性比率
- **收入增长率**: `(本期收入 - 上期收入) / 上期收入`
- **利润增长率**: `(本期利润 - 上期利润) / 上期利润`
- **EPS增长率**: `(本期EPS - 上期EPS) / 上期EPS`
- **ROE增长率**: `(本期ROE - 上期ROE) / 上期ROE`

## 使用方法

### 基础查询
```bash
# 获取财务报表
python scripts/financial_statement.py --statement 00700

# 获取财务比率
python scripts/financial_statement.py --ratios 00700

# 获取健康度评估
python scripts/financial_statement.py --health 00700

# 获取投资价值
python scripts/financial_statement.py --value 00700
```

## 注意事项

1. **会计准则**: 港股采用IFRS会计准则
2. **报告周期**: 年报、中报、季报
3. **数据延迟**: 财务数据有披露延迟
4. **汇率影响**: 考虑汇率波动影响
5. **审计意见**: 关注审计意见变化
