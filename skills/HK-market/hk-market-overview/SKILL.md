# 港股市场概览分析器

提供港股市场全方位的实时分析，包括主要指数表现、板块轮动、市场情绪和流动性状况。

## 核心功能

### 📊 市场整体表现
- 恒生指数 (HSI) 实时表现
- 国企指数 (HSCEI) 走势分析
- 红筹指数 (HSCCI) 表现
- 科技指数 (HSTECH) 动态
- 市场成交量和活跃度

### 🏢 板块轮动分析
- 11个主要板块表现对比
- 板块资金流向分析
- 板块相对强度排名
- 板块轮动信号识别

### 💹 市场情绪指标
- 涨跌家数统计
- 市场宽度指标
- 恐慌贪婪指数
- VIX波动率指标

### 💰 流动性监控
- 市场总成交量
- 南向资金净流入
- 市场深度分析
- 流动性风险评估

### 投资决策支持

## 数据源

### 主要数据源
- **港交所**：实时行情数据
- **彭博**：指数和板块数据
- **路透**：资金流向数据
- **各大券商**：研究报告数据

### 数据更新频率
- **实时数据**：交易时间内每分钟更新
- **板块数据**：每5分钟更新
- **资金流向**：每日收盘后更新
- **技术指标**：实时计算

## 使用方法

### 基础查询
```bash
# 获取市场概览
python scripts/market_overview.py

# 获取特定指数详情
python scripts/market_overview.py --index HSI

# 获取板块表现
python scripts/market_overview.py --sectors

# 获取市场情绪
python scripts/market_overview.py --sentiment
```

### 高级分析
```bash
# 获取历史趋势
python scripts/market_overview.py --history 30

# 获取板块轮动信号
python scripts/market_overview.py --rotation

# 获取风险评估
python scripts/market_overview.py --risk
```

## 注意事项

1. **交易时间**：港股交易时间为9:30-12:00, 13:00-16:00
2. **数据延迟**：实时数据可能有1-2分钟延迟
3. **汇率影响**：港股以港币计价，需考虑汇率风险
4. **市场机制**：港股无涨跌停限制，波动可能较大
