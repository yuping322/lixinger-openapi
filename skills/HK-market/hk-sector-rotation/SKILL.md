# 港股板块轮动检测器

专业的港股板块轮动分析工具，提供行业轮动识别、资金流向分析和配置建议。

## 核心功能

### 🔄 轮动检测
- 板块相对强度分析
- 资金流向轮动
- 轮动信号识别
- 轮动周期分析

### 📊 行业表现
- 各行业涨跌幅排名
- 行业相对强度
- 行业资金流向
- 行业估值对比

### 💰 资金流向
- 行业资金流入流出
- 机构行业偏好
- 南向资金行业分布
- 资金轮动模式

### 🎯 配置建议
- 行业超配低配建议
- 轮动时机把握
- 风险分散建议
- 配置比例优化

## 使用场景

### 行业配置
- 识别强势行业
- 把握轮动机会
- 优化行业配置
- 分散投资风险

### 时机选择
- 判断轮动时机
- 识别轮动信号
- 把握入场时机
- 控制退出时机

### 风险管理
- 监控行业风险
- 识别轮动风险
- 控制集中度风险
- 动态调整配置

## 轮动指标

### 相对强度指标
- **相对强度**: `行业收益率 / 市场收益率`
- **相对强度排名**: 行业相对强度排名
- **相对强度动量**: 相对强度变化率
- **相对强度趋势**: 相对强度趋势方向

### 资金流向指标
- **资金净流入**: 行业资金净流入金额
- **资金流入占比**: 行业流入占总流入比例
- **资金流入持续性**: 连续流入天数
- **资金流入强度**: 单日流入强度

### 轮动信号指标
- **轮动信号强度**: 轮动信号的强度
- **轮动持续性**: 预计轮动持续时间
- **轮动确定性**: 轮动信号的确定性
- **轮动周期**: 轮动周期长度

## 技术分析

### 趋势分析
- **行业趋势**: 各行业趋势方向
- **轮动趋势**: 轮动整体趋势
- **相对趋势**: 行业相对趋势
- **趋势强度**: 趋势强度评估

### 模式识别
- **轮动模式**: 识别轮动模式
- **周期模式**: 轮动周期模式
- **资金模式**: 资金流向模式
- **背离模式**: 价格与资金背离

## 数据源

### 主要数据源
- **港交所**: 行业分类和交易数据
- **数据供应商**: 行业资金流向数据
- **券商研究**: 行业研究报告
- **资金流向**: 南向资金行业数据

### 数据更新频率
- **实时数据**: 交易时间内实时更新
- **日度数据**: 每日收盘后更新
- **资金数据**: 每日更新
- **历史数据**: 5年以上历史数据

## 使用方法

### 基础查询
```bash
# 获取轮动概览
python scripts/sector_rotation.py --overview

# 获取行业表现
python scripts/sector_rotation.py --performance

# 获取资金流向
python scripts/sector_rotation.py --flow

# 获取配置建议
python scripts/sector_rotation.py --allocation
```

### 高级分析
```bash
# 轮动信号检测
python scripts/sector_rotation.py --signals

# 轮动历史分析
python scripts/sector_rotation.py --history 30

```
