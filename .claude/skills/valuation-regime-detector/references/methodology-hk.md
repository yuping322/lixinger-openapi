# 方法论：估值制度检测器（港股）

本技能用于**检测港股市场的估值制度（高估、合理、低估），识别估值拐点**。港股估值受南向资金、A+H溢价、国际资金等多重因素影响。

## 数据口径

### 数据源与字段映射

估值制度检测需要以下数据：
- **估值数据**：理杏仁 API
  - 市盈率PE、市净率PB、股息率
  - 日频数据，实时更新
- **南向资金数据**：港股通资金流向
  - 净流入、净流出
  - 日频数据
- **A+H溢价数据**：AH股溢价指数
  - 溢价率、折价率
  - 日频数据
- **国际对比数据**：全球市场估值
  - 美股、欧洲、日本估值对比
  - 月频数据

关键字段：
- `PE = 总市值 / 归母净利润`
- `PB = 总市值 / 净资产`
- `估值分位数 = rank(PE) / 历史天数`

### 时间窗口与频率

- **日频数据**：PE、PB、股息率
- **历史回溯**：10年历史数据（恒生指数历史更长）
- **分位数计算**：滚动10年分位数

## 核心指标

### 指标列表与计算公式

1. **估值水平指标**：
   - `PE = 总市值 / 归母净利润` (倍)
   - `PB = 总市值 / 净资产` (倍)
   - `股息率 = 每股分红 / 股价 × 100%` (%)
   - `ERP = 1/PE - 10年期国债收益率` (%) (使用中国国债收益率作为参考)

2. **估值分位数指标**：
   - `PE分位数 = rank(PE) / 历史天数 × 100%` (%)
   - `PB分位数 = rank(PB) / 历史天数 × 100%` (%)
   - `综合估值分位数 = (PE分位数 + PB分位数) / 2` (%)

3. **估值制度指标**：
   - `估值制度 = f(PE分位数, PB分位数, 南向资金流向)` (高估/合理/低估)
   - `估值偏离度 = (当前PE - 历史中位数PE) / 历史中位数PE × 100%` (%)

4. **南向资金指标**：
   - `南向净流入 = 港股通买入 - 港股通卖出` (亿元)
   - `南向资金影响度 = 南向净流入 / 恒生指数成交量 × 100%` (%)

5. **A+H溢价指标**：
   - `AH溢价率 = (A股价格 - H股价格) / H股价格 × 100%` (%)
   - `溢价收敛度 = AH溢价率变化率` (%)

6. **估值拐点指标**：
   - `估值趋势 = sign(PE变化率)` (+1/-1)
   - `估值拐点 = 1{估值趋势反转}` (0/1)

### 标准化方法

- 使用**历史分位数**评估估值水平
- 使用**南向资金流向**评估资金驱动
- 使用**AH溢价**评估跨市场定价差异

## 信号与阈值

### 可测试规则（Insight Rules）

Rule 1 (低估值 + 南向资金流入 → 估值修复):
IF {PE分位数 <= 30% AND 南向净流入 >= 10亿 AND AH溢价收敛}
THEN {未来3-6个月，估值修复，市场上涨概率 >= 65%}
CONFIDENCE {0.65; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {港股市场，估值底部}
FAILURE_MODE {国际资金流出；汇率波动；地缘政治风险}

Rule 2 (高估值 + 南向资金流出 → 估值压缩):
IF {PE分位数 >= 70% AND 南向净流出 >= 5亿 AND AH溢价扩大}
THEN {未来3-6个月，估值压缩，市场回调概率 >= 60%}
CONFIDENCE {0.60; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {港股市场，估值顶部}
FAILURE_MODE {国际资金回流；A股大涨带动；政策利好}

Rule 3 (估值拐点 + 资金流向反转 → 趋势反转):
IF {估值趋势反转 AND 南向资金流向反转 AND 估值分位数处于极端（<20%或>80%）}
THEN {未来1-3个月，市场趋势反转概率 >= 62%}
CONFIDENCE {0.62; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {港股市场，估值拐点}
FAILURE_MODE {资金流向不稳定；外部冲击；汇率波动}

Rule 4 (AH溢价收敛 + 南向资金流入 → H股估值修复):
IF {AH溢价率 <= 20% AND 南向净流入 >= 10亿}
THEN {未来3-6个月，H股相对A股估值修复，AH溢价进一步收敛}
CONFIDENCE {0.58; 初步估计；需要历史验证}
APPLICABLE_UNIVERSE {港股市场，AH股对比}
FAILURE_MODE {A股下跌；南向资金减少；汇率波动}

### 触发/解除/无效化条件

- **触发监控**：
  - PE分位数 <= 30%或 >= 70%
  - 南向资金净流入/流出 >= 10亿/5亿
  - AH溢价率变化 >= 5%

- **解除监控**：
  - 估值回归合理区间（30-70%分位）
  - 南向资金流向稳定
  - AH溢价率稳定

- **无效化**：
  - 汇率大幅波动（HKD/USD）
  - 地缘政治事件
  - 国际资金大规模流动

### 阈值选择依据

- **30% PE分位数**：PE分位数 <= 30%属于低估
- **70% PE分位数**：PE分位数 >= 70%属于高估
- **10亿南向净流入**：显著资金流入信号
- **5亿南向净流出**：显著资金流出信号
- **20% AH溢价率**：AH溢价较低，H股相对便宜

## 边界条件与降级策略

### 数据缺失/异常值处理

- **数据缺失**：
  - 南向资金数据缺失：使用成交量变化代理，标注"使用成交量代理"
  - AH溢价数据缺失：仅分析港股估值，标注"AH溢价数据缺失"

- **异常值处理**：
  - PE < 0（亏损）：使用PB代替，标注"使用PB"
  - PE > 100：检查是否微利或数据错误

### 降级策略

- **历史数据不足**（< 5年）：
  - 使用可用数据计算分位数
  - 降级标注："历史数据不足，分位数可能有偏"

- **南向资金数据缺失**：
  - 仅使用估值分位数分析
  - 降级标注："南向资金数据缺失，仅使用估值分析"

## 港股特殊注意

### 1. 南向资金影响

- **南向资金主导**：港股受南向资金影响大
- **资金流向波动**：南向资金流向波动性较大
- **需要关注流向**：估值分析需结合南向资金流向

### 2. AH溢价

- **AH溢价显著**：同一公司在A股和H股价格差异大
- **溢价动态变化**：AH溢价随资金流向和市场环境变化
- **需要AH对比**：港股估值需与A股对比

### 3. 国际资金影响

- **国际资金参与**：港股国际资金占比高
- **美元流动性影响**：港股受美元流动性影响
- **需要国际对比**：港股估值需与国际市场对比

### 4. 汇率影响

- **港币汇率**：港币与美元挂钩，受汇率影响
- **人民币汇率**：南向资金受人民币汇率影响
- **需要汇率分析**：估值分析需考虑汇率因素

### 5. 无涨跌停限制

- **估值波动大**：港股无涨跌停，估值波动可能较大
- **短期调整剧烈**：估值调整可能比A股更剧烈
- **需要谨慎判断**：估值极端时需谨慎判断

### 6. 市场机制差异

- **交易机制不同**：港股交易机制与A股不同
- **做空机制**：港股有做空机制，影响估值
- **需要机制差异**：估值分析需考虑机制差异

### 7. 行业结构差异

- **行业结构不同**：港股行业结构偏向金融地产
- **科技股较少**：港股科技股占比较低
- **需要行业调整**：估值分析需考虑行业结构

### 8. 流动性差异

- **小盘股流动性差**：港股小盘股流动性较差
- **大盘股流动性好**：港股大盘股流动性好
- **需要流动性分析**：估值分析需考虑流动性

### 9. 估值中枢差异

- **估值中枢偏低**：港股估值中枢长期低于A股
- **股息率较高**：港股股息率较高
- **需要中枢调整**：估值分位数需考虑中枢差异

### 10. 地缘政治影响

- **地缘政治敏感**：港股受地缘政治影响大
- **政策不确定性**：政策环境不确定性较高
- **需要政治分析**：估值分析需考虑政治因素

## 回测说明（最小化）

- **回测设计**：
  - 计算历史估值分位数
  - 识别估值极端区域
  - 统计估值信号有效性（结合南向资金）

- **性能指标**：
  - 估值信号准确率
  - 估值信号后收益
  - 估值拐点识别准确率

- **证伪条件**：
  - 信号准确率 < 55%
  - 估值信号后收益不显著

## 使用示例

### 示例1：低估值修复（南向资金流入）

**输入**：2026-02-15估值数据
**数据**：
- PE分位数：25%
- 南向净流入：+15亿
- AH溢价率：18%
**输出**：
- 信号：低估值 + 南向资金流入（估值修复）
- **结论**：估值处于底部，南向资金流入，AH溢价收敛，未来3-6个月估值修复

### 示例2：高估值压缩（南向资金流出）

**输入**：2026-03-20估值数据
**数据**：
- PE分位数：75%
- 南向净流出：-8亿
- AH溢价率：35%
**输出**：
- 信号：高估值 + 南向资金流出（估值压缩）
- **结论**：估值偏高，南向资金流出，AH溢价扩大，未来3-6个月估值压缩风险

### 示例3：估值拐点（资金流向反转）

**输入**：2026-04-10估值数据
**数据**：
- 估值趋势：反转（由下降转上升）
- 南向资金流向：反转（由流出转流入）
- PE分位数：18%
**输出**：
- 信号：估值拐点 + 资金流向反转（趋势反转）
- **结论**：估值拐点出现，南向资金流向反转，市场趋势可能反转

### 示例4：AH溢价收敛

**输入**：2026-05-15估值数据
**数据**：
- AH溢价率：15%
- 南向净流入：+12亿
- H股PE分位数：30%
**输出**：
- 信号：AH溢价收敛 + 南向资金流入（H股估值修复）
- **结论**：AH溢价较低，南向资金流入，H股相对A股估值修复
