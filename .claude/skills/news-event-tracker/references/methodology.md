# 分析方法（methodology）

## A. 时间线框架

每条事件建议字段：
- date_event: 事件发生日期
- date_publish: 首次披露日期
- actor: 事件主体
- fact: 已证实事实
- impact: 影响描述
- confidence: 高 / 中 / 低

## B. 影响评估框架

1. 直接影响：收入、成本、供给、需求、估值、合规
2. 传导路径：上下游、竞品、行业、政策
3. 持续性：一次性 / 阶段性 / 结构性
4. 情景划分：基准 / 乐观 / 悲观

## C. 跟踪闭环

- 观察指标：价格、量能、政策进展、公告更新、财报验证
- 触发条件：指标阈值或关键事件触发二次更新
- 更新频率：突发（实时）、专题（日更/周更）
