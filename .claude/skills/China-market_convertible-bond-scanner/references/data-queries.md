# 数据获取指南

可转债扫描器使用 AkShare 接口获取数据。

---

## 本 Skill 使用的数据接口

### 1. 集思录可转债实时数据（bond_cb_jsl）

**用途**：获取可转债行情数据和基本信息，用于双低策略筛选

**接口信息**：
- 数据来源：集思录
- 更新频率：实时
- 返回字段：代码、转债名称、现价、涨跌幅、正股代码、正股名称、正股价、正股涨跌、正股PB、转股价、转股价值、转股溢价率、债券评级、回售触发价、强赎触发价、转债占比、到期时间、剩余年限、剩余规模、成交额、换手率、到期税前收益、双低

**调用示例**：
```python
import akshare as ak

bond_cb_jsl_df = ak.bond_cb_jsl(cookie="")
print(bond_cb_jsl_df)
```

**注意**：需要集思录 cookie 才能获取完整数据，否则只能返回前 30 条

---

### 2. 可转债强赎数据（bond_cb_redeem_jsl）

**用途**：获取可转债强赎信息，用于强赎预期分析

**接口信息**：
- 数据来源：集思录
- 更新频率：实时
- 返回字段：代码、名称、现价、正股代码、正股名称、规模、剩余规模、转股起始日、最后交易日、到期日、转股价、强赎触发比、强赎触发价、正股价、强赎价、强赎天计数、强赎条款、强赎状态

**调用示例**：
```python
import akshare as ak

bond_cb_redeem_jsl_df = ak.bond_cb_redeem_jsl()
print(bond_cb_redeem_jsl_df)
```

---

### 3. 可转债基本信息（bond_zh_cov）

**用途**：获取可转债发行信息，用于信用评级和发行规模分析

**接口信息**：
- 数据来源：东方财富网
- 更新频率：日频
- 返回字段：债券代码、债券简称、申购日期、申购代码、申购上限、正股代码、正股简称、正股价、转股价、转股价值、债现价、转股溢价率、原股东配售-股权登记日、原股东配售-每股配售额、发行规模、中签号发布日、中签率、上市时间、信用评级

**调用示例**：
```python
import akshare as ak

bond_zh_cov_df = ak.bond_zh_cov()
print(bond_zh_cov_df)
```

---

### 4. 可转债溢价率分析（bond_zh_cov_value_analysis）

**用途**：获取单只可转债的历史溢价率数据，用于深度分析

**接口信息**：
- 数据来源：东方财富网
- 更新频率：日频
- 返回字段：日期、收盘价、纯债价值、转股价值、纯债溢价率、转股溢价率

**调用示例**：
```python
import akshare as ak

bond_zh_cov_value_analysis_df = ak.bond_zh_cov_value_analysis(symbol="113527")
print(bond_zh_cov_value_analysis_df)
```

## 数据字段说明

| 字段 | 说明 | 计算公式/来源 |
|------|------|--------------|
| 转债价格 | 可转债的市场价格（元） | 直接获取 |
| 转股价 | 转换为正股的价格（元/股） | 直接获取 |
| 正股价格 | 对应正股的市场价格（元/股） | 直接获取 |
| 转股价值 | 正股价格 / 转股价 × 100 | 正股价格 / 转股价 × 100 |
| 转股溢价率 | (转债价格 - 转股价值) / 转股价值 (%) | (转债价格 - 转股价值) / 转股价值 × 100 |
| 纯债价值 | 按票面利率和到期时间折现的债券价值（元） | 需要计算或从接口获取 |
| 纯债溢价率 | (转债价格 - 纯债价值) / 纯债价值 (%) | (转债价格 - 纯债价值) / 纯债价值 × 100 |
| 双低指标 | 转债价格 + 转股溢价率 | 转债价格 + 转股溢价率 |
| 到期收益率 | (100 + 累计利息 - 转债价格) / 转债价格 / 剩余年限 (%) | 直接获取 |
| 强赎触发价 | 转股价 × 130% | 转股价 × 1.3 |
| 强赎天计数 | 连续满足强赎条件的天数 | 直接获取 |

---

## 策略筛选条件

### 双低策略
- 双低指标 < 120
- 转债价格 < 110
- 转股溢价率 < 20%
- 纯债价值 > 90

### 强赎预期
- 连续满足强赎天数 >= 10
- 正股走势强劲（20日涨幅 >= 15%）
- 转债价格 < 125

### 高债性防守
- 纯债价值 > 95
- 转债价格 < 105
- 到期收益率 > 2%
- 信用评级 >= AA

```bash
python3 plugins/query_data/lixinger-api-docs/scripts/query_tool.py \
  --suffix "cn/company/fundamental/non_financial" \
  --params '{"date": "2025-12-31", "stockCodes": ["600519", "000858", "300750"], "metricsList": ["pe_ttm", "pb", "dyr", "mc"]}' \
  --columns "date,stockCode,pe_ttm,pb,dyr,mc" \
  --limit 20
```
