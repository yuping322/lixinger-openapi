# 输出模板：市场宽度监控器

> 目标：判断指数表现背后的内部结构是否健康，识别宽度扩张、收缩与风险偏好切换。

## 市场宽度监控报告

### 1) 结论摘要（3–5条）

- 当前市场宽度状态为 `[扩张 / 中性 / 收缩 / 极弱]`。
- 指数走势与宽度表现呈 `[共振 / 背离]`，意味着市场结构 `[健康 / 隐患加大]`。
- 宽度改善/恶化主要由 `[上涨家数 / 新高新低 / 中小盘表现 / 行业轮动]` 驱动。
- 若当前状态延续，市场更可能进入 `[趋势确认 / 震荡分化 / 风险释放]`。
- 后续需重点观察 `[上涨比例 / 新高新低 / 跌停扩散 / 集中度 / 宽度动量]`。

### 3) 分析与解释

#### 3.1 宽度与指数关系

- 指数上涨是否由少数权重股拉动。
- 若宽度走弱而指数走强，说明结构性风险上升。
- 若宽度先于指数改善，可能是风险偏好回升的早期信号。

#### 3.2 内部结构

- 新高/新低结构能否支持当前趋势判断。
- 中位数个股表现是否改善，帮助判断真实赚钱效应。
- 跌停扩散比单纯下跌家数更能揭示压力。

#### 3.3 状态切换与应用

- 识别从收缩到扩张、从扩张到分化的切换点。
- 宽度指标更适合做风险管理还是做顺势确认。
- 建议与成交额、资金流、板块轮动联合使用。

### 4) 风险清单与监控指标

| 风险 | 触发条件 | 监控指标 | 应对建议 |
|---|---|---|---|
| 指数失真 | 权重拉升掩盖多数个股走弱 | 上涨比例、中位数涨跌幅 | 不高估市场强度 |
| 宽度假修复 | 一日反弹但持续性差 | 宽度动量、成交额 | 等待连续确认 |
| 跌停扩散 | 新低与跌停同步上升 | 新低家数、跌停家数 | 提高防守权重 |
| 过度集中 | 少数板块贡献过高 | 集中度、行业分布 | 警惕轮动脆弱性 |

### 5) 下一步（需要补充/核验）

- 补做 `5 / 10 / 20` 日宽度均线与拐点标记。
- 与市场总貌、板块轮动报告联动解释。
- 如需交易信号，增加宽度极值后的历史统计。
