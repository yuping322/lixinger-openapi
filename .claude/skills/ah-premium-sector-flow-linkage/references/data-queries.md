# 数据获取建议

> 本文件定义“应取哪些数据”，不强绑定单一数据供应商。

## 必要数据表

1. **AH价格与汇率**
   - 字段：交易日、A股收盘价、H股收盘价、汇率、复权因子
2. **沪深港通资金**
   - 字段：北向净买入、南向净买入、分市场（沪/深）拆分、成交额占比
3. **行业资金流**
   - 字段：行业主力净流入、超大单净额、行业换手率、行业涨跌幅
4. **估值指标**
   - 字段：PE(TTM)、PB(LF)、EV/EBITDA、行业估值分位

## 最小可用频率

- 日频为默认分析频率。
- 对交易执行，可增加30分钟或60分钟资金节奏监控。

## 数据对齐规则

- A/H交易日历不一致时，以共同交易日对齐。
- 汇率使用同一时点（收盘或固定时刻）并在报告中注明。
- 行业分类统一口径（申万/中信/GICS任选其一并保持一致）。

## 质量检查

- 缺失值比例 >10% 时，标注“样本可信度下降”。
- 若出现异常跳点，先检查拆股/复权/代码映射问题。
- 至少保留近3年历史用于分位和状态识别。
