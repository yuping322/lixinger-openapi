# AH溢价与行业资金结构联动交易

围绕“AH价差回归在什么市场环境下更有效”构建可执行分析流程。先判断市场状态，再识别触发条件，最后给出交易方案与失效预警。

## 工作流

### 1) 定义分析对象与窗口

先明确以下输入：
- 标的层级：指数/行业/个股（默认先行业后个股）。
- 时间窗口：短线（5–20日）/波段（1–3个月）/中期（3–6个月）。
- 交易方向：做“溢价收敛”还是“折价修复”。
- 约束条件：是否允许做空、最大回撤、换手与成交额门槛。

若用户未提供，给出默认设置并标注“可调整参数”。

### 2) 构建四因子诊断面板

依次完成四个维度，口径见 `references/methodology.md`：
- **AAH比价/分位**：识别价差是否达到历史可交易极值区。
- **沪深港通资金**：确认跨市场资金方向是否支持回归。
- **行业资金流**：判断行业内主导资金是“扩散”还是“抱团”。
- **估值因子**：检验价差是否伴随估值偏离而非基本面永久变化。

输出每个维度的结论：`支持 / 中性 / 反对`。

### 3) 判断市场环境（Regime）

将市场划分为四类状态：
1. **流动性宽松 + 风险偏好修复**（回归胜率通常最高）
2. **震荡均值市场**（回归有效但节奏慢）
3. **单边风险偏好上行**（价差可能继续扩张）
4. **流动性收缩/信用扰动**（回归易失效，先控风险）

用“政策/流动性 + 波动率 + 资金风格”三要素确认状态，不确定时给出两种情景并分别推演。

### 4) 生成交易信号与执行规则

按“触发-确认-执行-退出”结构输出：
- **触发**：价差进入历史极值分位。
- **确认**：至少2个资金维度同向确认（北向/南向/行业主力）。
- **执行**：分批建仓，给出仓位阶梯。
- **退出**：达到目标分位、资金背离、或波动超阈值即减仓/止损。

若无法形成可执行信号，明确写“当前仅观察，不交易”。

### 5) 输出结构化结论

按 `references/output-template.md` 输出以下模块：
- 市场环境判定
- 回归有效性评分（0–100）
- 证据链
- 交易方案
- 风险与失效条件
- 下一步跟踪清单

## 评分与决策规则

- 使用四因子加权评分：
  - AAH比价 30%
  - 沪深港通 25%
  - 行业资金流 25%
  - 估值偏离 20%
- 评分解释：
  - **≥70**：可执行回归交易（优先顺势确认后的逆向切入）
  - **50–69**：弱有效，轻仓试探 + 高频复核
  - **<50**：回归假设不足，避免强行交易

## 分析纪律

- 必须写明数据日期、频率和口径，不使用模糊表述。
- 必须区分“统计回归”与“基本面重定价”，防止误判。
- 必须给出失效条件（如政策冲击、汇率单边、流动性突变）。
- 不提供确定性承诺，所有结论使用概率化语言。
