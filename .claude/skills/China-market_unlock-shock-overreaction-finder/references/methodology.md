# 方法论：解禁冲击错杀识别框架

## 1) 核心判定逻辑

目标是回答两件事：
- **冲击是否被高估**：市场跌幅是否显著大于“真实抛压落地强度”。
- **修复多久完成**：价格从冲击低点回到事件前中枢需要的时间与条件。

## 2) 四维评分（0-100）

### A. 供给压力分（权重 30%）

建议指标：
- 解禁市值 / 日均成交额（冲击倍数）
- 解禁股份 / 自由流通股
- 解禁股东类型（财务投资者、产业股东、员工持股平台）

解释：
- 冲击倍数越高、财务投资者占比越高，短期供给压力越大。

### B. 承接质量分（权重 25%）

建议指标：
- 大宗交易折价率（越小越好）
- 解禁后10日内大宗成交连续性
- 大宗后次日到5日价格表现（验证是否“有承接”）

解释：
- 出现可持续成交且折价快速收敛，说明抛压被机构化吸收。

### C. 减持兑现分（权重 25%）

建议指标：
- 股东减持计划覆盖比例
- 实际减持进度 / 计划进度
- 减持节奏是否前置（解禁后1-2周集中兑现）

解释：
- 若公告很多但实际执行弱，市场常出现“过度预期抛压”。

### D. 资金修复分（权重 20%）

建议指标：
- 主力资金净流入占成交额比例
- 冲击后20日累计超额收益（对行业或指数）
- 缩量企稳 + 放量反弹结构是否出现

解释：
- 资金从净流出转净流入且价格结构改善，是修复启动信号。

## 3) 错杀评分与分层

- **错杀评分 = 承接质量分 + 资金修复分 - 供给压力分 - 减持兑现分（经标准化）**
- 建议分层：
  - A档（高确定）：错杀评分前20%，且存在至少2个正向触发器
  - B档（观察）：中间60%，等待右侧确认
  - C档（回避）：后20%，供给冲击仍在释放

## 4) 修复时钟估计（经验分档）

- **快修复（5-15交易日）**：承接质量高 + 减持执行弱 + 资金快速回流
- **中等修复（15-40交易日）**：供需再平衡需时间，资金回流慢
- **长尾修复（40+交易日）**：持续减持或基本面同步走弱

## 5) 失效条件

出现以下任一情况，降低评级或退出观察：
- 新增超预期减持主体
- 大宗折价持续扩大且二级市场无承接
- 资金流持续恶化并跌破关键支撑
