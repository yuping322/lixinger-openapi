# 输出模板：公告与信息披露监控器

> 目标：把离散公告整理为可执行的事件清单、影响评估和后续跟踪动作，避免只做公告罗列。

## 公告与信息披露监控报告

### 1) 结论摘要（3–5条）

- 本期最重要的公告事件为 `[业绩预告 / 股东减持 / 回购 / 诉讼 / 监管问询 / 重大合同]`。
- 对股价与基本面影响最大的事项是 `[列出 1–3 项]`，影响方向为 `[正面 / 中性 / 负面 / 待验证]`。
- 当前公告流显示公司处于 `[经营改善 / 资本运作活跃 / 治理扰动 / 风险暴露]` 阶段。
- 高优先级待跟踪事项包括 `[回复问询 / 审批进度 / 实施比例 / 业绩兑现]`。
- 若未来 `—` 日内没有进一步公告确认，当前结论需下调置信度。

### 3) 分析与解释

#### 3.1 事件分类与优先级

- 将公告分为经营、资本运作、治理、监管、诉讼、股东行为等类别。
- 说明哪些公告只是例行披露，哪些公告可能改变估值或风险认知。
- 对重复性公告，给出“信息增量是否显著”的判断。

#### 3.2 影响评估

- 对收入、利润、现金流、融资能力、控制权、市场预期分别打标签。
- 说明影响是即时性的，还是需要后续兑现才能成立。
- 若公告涉及区间描述或模糊表述，应明确不确定性来源。

#### 3.3 跟踪动作

- 需要等待的后续文件：回复函、补充公告、审议结果、实施进度等。
- 若事件带有时间窗口，给出关键日期与失效条件。
- 如需联动其他 skill，说明建议使用的后续分析路径。

### 4) 风险清单与监控指标

| 风险 | 触发条件 | 监控指标 | 应对建议 |
|---|---|---|---|
| 公告被市场误读 | 标题强但细节弱 | 公告全文、关键条款 | 以正文条款为准复核 |
| 兑现不足 | 利好公告后进展停滞 | 实施比例、后续披露 | 降低业绩兑现预期 |
| 治理风险升级 | 监管问询或诉讼继续扩散 | 问询函、处罚、诉讼进展 | 提升风险标签 |
| 股东行为压制估值 | 减持、质押、冻结增加 | 股东变动公告 | 联动治理与供给风险分析 |

### 5) 下一步（需要补充/核验）

- 建立按重要级别排序的事件待办列表。
- 对高影响公告补读正文与附件，避免只根据摘要结论。
- 对涉及财务影响的公告，补做金额、期限与兑现路径测算。

### 6) 缺失数据与降级说明

- 缺失数据：部分附件、交易所回复进度、管理层口径细节。
- 替代口径：使用公告正文、历史案例与公开问答口径。
- 对结论置信度影响：中至高，视公告完整性而定。

### 7) 免责声明

> **免责声明**：本分析仅供信息参考和教育目的，不构成任何投资建议。公告文本可能存在措辞模糊、实施条件约束或后续修订，不能将标题性信息直接等同于基本面变化。
