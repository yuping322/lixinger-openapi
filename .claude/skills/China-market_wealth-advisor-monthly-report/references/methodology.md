# 方法论：月度综合资产配置分析报告

本技能用于**生成月度综合资产配置分析报告，覆盖宏观经济、A股、债券、黄金外汇四大资产类别，基于确定性数据进行系统化分析并输出配置建议**。

## 核心工作原则（必须严格遵守）

### 1. 信息确定性原则

- 所有分析与结论必须基于【已公布的官方数据、已发生的政策事件、已收盘的市场价格】
- 严禁使用任何预测性或模糊措辞，如「可能」「预计」「大概率」「或将」
- 使用确定性表述：「数据显示」「已确认」「截至XX日」

### 2. 数据可验证原则

- 所有关键数据必须同时注明：
  - 具体日期
  - 明确来源（如：Wind、Bloomberg、同花顺、国家统计局、央行、FRED、CBOE、EPFR、ETF 官方披露等）
- 所有数据应可被主流金融终端或官方渠道回溯验证

### 3. 分析实用性原则

- 不做表层复述，必须穿透数据，提炼【驱动市场的核心逻辑】
- 所有资产配置建议，必须清晰说明：
  - 用到了哪些监测指标
  - 这些指标如何形成因果链条
  - 为什么会导向当前的配置结论

### 4. AI 增强分析方法

- 下述监测指标是"传感器"和"仪表盘"
- 需要在分析过程中【自然调用并交叉验证】，而非孤立罗列

## 分析框架

### 宏观经济分析

1. **经济周期定位**：基于 PMI、工业增加值、社融综合判断
2. **货币政策解读**：基于央行操作与利率决议
3. **收益率曲线分析**：10Y-2Y 利差反映市场隐含预期

### A股市场分析

1. **趋势判断**：指数位置 + 200 日均线 + 估值分位
2. **资金结构**：北向资金 + 融资融券 + ETF 资金流
3. **市场宽度**：涨跌家数比 + 新高新低比 → 内部健康度

### 债券市场分析

1. **利率趋势**：10Y 国债收益率变动方向与幅度
2. **信用风险**：信用利差水平与分位
3. **资金面**：DR007 + 债券 ETF 资金行为

### 黄金外汇分析

1. **黄金定价三因素**：实际利率 + 美元 + 风险情绪
2. **美元联动**：DXY 与人民币汇率
3. **情绪验证**：VIX + Put/Call Ratio

### 综合配置逻辑

- 所有配置建议必须由分析模块"推导得出"
- 清晰写出逻辑链条：「基于 A 指标 + B 指标 → 得出 C 结论 → 对应配置动作是……」
- 不做无依据的主观判断

## 信号与阈值

### 可测试规则（Insight Rules）

Rule 1 (经济复苏 + 信用扩张 → 权益占优):
IF {制造业 PMI > 50 连续 3 月 AND 社融同比增速 > 前值 AND 信贷脉冲向上}
THEN {权益资产相对占优，可增配 A 股}
CONFIDENCE {0.65; 基于历史周期规律}
APPLICABLE_UNIVERSE {大类资产配置}
FAILURE_MODE {外部冲击（贸易摩擦、地缘风险）；政策收紧预期提前}

Rule 2 (收益率曲线倒挂 + 信用利差走阔 → 防御配置):
IF {10Y-2Y 利差 < 0 AND 信用利差 > 历史 75 分位 AND VIX > 25}
THEN {增配债券与黄金，减配权益}
CONFIDENCE {0.60; 基于美国历史数据}
APPLICABLE_UNIVERSE {全球大类资产}
FAILURE_MODE {央行强力干预（非常规宽松）；通胀预期骤变}

Rule 3 (A股低估值 + 资金回流 → 战略配置窗口):
IF {沪深 300 PE 分位 < 30% AND 北向资金连续月净流入 AND 融资余额环比上升}
THEN {A 股处于战略配置区间}
CONFIDENCE {0.62; 基于 A 股历史估值周期}
APPLICABLE_UNIVERSE {A 股权益配置}
FAILURE_MODE {基本面持续恶化；政策不确定性加大}

Rule 4 (实际利率下行 + 美元走弱 → 黄金占优):
IF {TIPS 实际利率环比下行 AND DXY 月跌幅 > 1% AND VIX 上行}
THEN {黄金配置价值上升}
CONFIDENCE {0.63; 基于黄金定价框架}
APPLICABLE_UNIVERSE {避险资产配置}
FAILURE_MODE {美联储鹰派转向；风险偏好骤升}

### 阈值选择依据

- **PMI 50 荣枯线**：国际通用经济扩张/收缩分界
- **PE 30% 分位**：历史估值偏低区域
- **信用利差 75 分位**：信用风险偏高水平
- **VIX 25**：恐慌情绪升温阈值
- **DXY 月跌幅 1%**：美元明显走弱信号

## 边界条件与降级策略

### 数据缺失处理

- **单一数据源缺失**：使用替代数据源，标注"替代口径"
- **核心指标缺失**（如 PMI 未公布）：该模块标注"数据待更新"，不做结论
- **多数据缺失**（> 30%）：降级为定性分析，标注"数据不完整，结论参考度降低"

### 异常值处理

- 单月指标异动 > 3 倍标准差：检查是否极端事件，单独标注
- 数据修正：如遇历史数据修正，采用最新修正值

## A股特殊注意

- **T+1 交易制度**：影响短期交易策略的时效性
- **涨跌停限制**：影响极端行情下的统计分布
- **北向资金**：已成为 A 股重要定价因子
- **政策市特征**：政策对市场影响权重较大
- **季节性因素**：如"春季躁动"、年末资金回笼等
