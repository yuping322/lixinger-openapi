# 方法学：首日与中期收益优势条件

## 1. 标签定义

- 首日收益：
  - `R1_open = (上市首日开盘价 / 发行价 - 1)`
  - `R1_close = (上市首日收盘价 / 发行价 - 1)`
- 中期收益：
  - `R20 = (T+20收盘 / 首日收盘 - 1)`
  - `R60 = (T+60收盘 / 首日收盘 - 1)`
- 超额收益：`Alpha20/60 = 个股收益 - 对应市场指数收益`

默认按板块匹配基准，避免不同涨跌停制度造成偏差。

## 2. 条件因子池

### A. 供给层（注册进度→发行拥挤度）

- `supply_4w`: 上市日前4周同板块新股数量（越低越优）；
- `industry_cluster`: 同行业同月集中上市数量（越低越优）；
- `reg_speed`: 从受理到注册生效天数（过快可能透支，适中更优）。

### B. 需求层（申购热度→交易承接）

- `subscribe_heat`: 有效申购倍数/中签率反向指标（越热越优，但需防极端拥挤）；
- `turnover_d1`: 首日换手率（中高区间更优，极端高换手常对应短线透支）；
- `main_netin_d1_5`: 主力资金 T+1~T+5 累计净流入（持续为正更优）。

### C. 定价层（发行估值→上市后空间）

- `pe_pctile`: 发行PE相对同行分位（中低分位更利于中期）；
- `issue_discount`: 发行估值较可比公司折价（折价越明显越优）；
- `quality_growth`: ROE、营收增速、毛利率稳定性综合评分。

## 3. 条件组合模板（用于回答“什么条件更优”）

## 首日收益更优（经验模板）

常见强组合：
- `低供给` + `高申购热度` + `首日主力净流入为正`
- `行业景气上行` + `同周发行不拥挤` + `首日换手处于中高分位`

常见失效点：
- 同期新股密集发行导致资金分流；
- 首日换手极端+估值过热，冲高回落概率上升。

## 中期收益更优（20D/60D）

常见强组合：
- `发行估值不过热（PE分位中低）` + `上市后5日主力连续净流入` + `基本面质量高`
- `首日涨幅适中`（不过度透支）+ `行业景气兑现`

常见失效点：
- 首日涨幅过高且后续资金无法接力；
- 解禁预期、业绩不及预期、板块风险偏好回落。

## 4. 评分与阈值

将每个条件标准化为 0–100 分：
- 供给层 30%、需求层 40%、定价层 30%；
- 生成 `D1_score`（首日）与 `M20_score`（中期）；
- 按分位分组（Q1~Q5）比较胜率与收益中位数。

输出时优先报告：
1) Q5 vs Q1 的收益差；
2) 最近3个月是否仍有效；
3) 样本量是否足够（每组建议 >= 20）。
