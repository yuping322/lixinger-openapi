接口: fund_purchase_em

目标地址: http://fund.eastmoney.com/Fund_sgzt_bzdm.html#fcode,asc_1

限量: 单次返回当前时刻所有历史数据

|-----|-----|-----|
| -   | -   | -   |

| 手续费            | float64 | 注意单位: % |
