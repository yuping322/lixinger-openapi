接口: index_kq_fz

目标地址: http://www.kqindex.cn/flzs/jiage

描述: 指定 symbol 的柯桥纺织品指数的所有历史数据

输入参数

| 名称     | 类型  | 描述                                                |
|--------|-----|---------------------------------------------------|
| symbol | str | symbol="价格指数"; choice of {'价格指数', '景气指数', '外贸指数'} |

输出参数-价格指数

| 名称  | 类型      | 描述      |
|-----|---------|---------|
| 期次  | object  | -       |
| 指数  | float64 | -       |
| 涨跌幅 | float64 | 注意单位: % |

接口示例-价格指数

```python
import akshare as ak

index_kq_fz_df = ak.index_kq_fz(symbol="价格指数")
print(index_kq_fz_df)
```
