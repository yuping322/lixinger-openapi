接口: fund_fh_em

目标地址: http://fund.eastmoney.com/data/fundfenhong.html

描述: 天天基金网-基金数据-分红送配-基金分红

限量: 单次返回所有历史数据

输入参数

| 名称   | 类型  | 描述                       |
|------|-----|--------------------------|
| year | str | year="2025"; 最早支持 1999 年 |

输出参数

| 名称    | 类型      | 描述        |
|-------|---------|-----------|
| 序号    | int64   | -         |
| 基金代码  | object  | -         |
| 基金简称  | object  | -         |
| 权益登记日 | object  | -         |
| 除息日期  | object  | -         |
| 分红    | float64 | 注意单位: 元/份 |
| 分红发放日 | object  | -         |

接口示例

```python
import akshare as ak

fund_fh_em_df = ak.fund_fh_em(year="2025")
print(fund_fh_em_df)
```
