接口: index_global_hist_em

描述: 东方财富网-行情中心-全球指数-历史行情数据

| 名称     | 类型  | 描述                                               |
|--------|-----|--------------------------------------------------|
| symbol | str | symbol="美元指数"; 可以通过 ak.index_global_spot_em() 获取 |

输出参数

| 名称  | 类型      | 描述      |
|-----|---------|---------|
| 日期  | object  | -       |
| 代码  | object  | -       |
| 名称  | object  | -       |
| 今开  | float64 | -       |
| 最新价 | float64 | -       |
| 最高  | float64 | -       |
| 最低  | float64 | -       |
| 振幅  | float64 | 主要单位: % |

接口示例

```python
import akshare as ak
```
