接口: amac_fund_abs

目标地址: https://gs.amac.org.cn/amac-infodisc/res/fund/abs/index.html

描述: 中国证券投资基金业协会-信息公示-产品公示-资产支持专项计划

限量: 单次返回当前时刻所有历史数据

| 名称 | 类型 | 描述 |
|----|----|----|
| -  | -  | -  |

| 备案编号   | object | -  |
| 专项计划全称 | object | -  |
| 管理人    | object | -  |
| 托管人    | object | -  |
| 成立日期   | object | -  |
| 预期到期时间 | object | -  |
| 备案通过时间 | object | -  |

import akshare as ak
