接口: index_neaw_cx

目标地址: https://yun.ccxe.com.cn/indices/nei

限量: 该接口返回所有历史数据

输入参数

| 名称  | 类型  | 描述  |
|-----|-----|-----|
| -   | -   | -   |

输出参数

| 名称            | 类型      | 描述  |
|---------------|---------|-----|
| 日期            | object  | -   |
| 新经济行业入职平均工资水平 | float64 | -   |
| 变化值           | float64 | -   |

接口示例

```python
import akshare as ak

index_neaw_cx_df = ak.index_neaw_cx()
print(index_neaw_cx_df)
```
