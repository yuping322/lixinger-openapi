接口: macro_germany_trade_adjusted

目标地址: https://data.eastmoney.com/cjsj/foreign_1_3.html

描述: 东方财富-数据中心-经济数据一览-德国-贸易帐(季调后)

限量: 单次返回所有历史数据

输入参数

| 名称  | 类型  | 描述  |
|-----|-----|-----|
| -   | -   | -   |

输出参数

| 名称   | 类型      | 描述  |
|------|---------|-----|
| 时间   | object  | -   |
| 前值   | float64 | -   |
| 现值   | float64 | -   |
| 发布日期 | object  | -   |

接口示例

```python
import akshare as ak

macro_germany_trade_adjusted_df = ak.macro_germany_trade_adjusted()
```
