限量: 单次返回所有历史数据

|-----|-----|-----|
| -   | -   | -   |

|---------|---------|----------|
| 月份      | object  | -        |
| 当月      | float64 | 注意单位: 亿元 |
| 同比增长    | float64 | 注意单位: %  |
| 环比增长    | float64 | 注意单位: %  |

| 累计-同比增长 | float64 | 注意单位: %  |

import akshare as ak
