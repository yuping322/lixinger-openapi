# 方法论：事件研究分析器（A股）

本文件定义事件研究的标准分析口径。目标是衡量事件发生前后，标的相对基准或相对同业的异常表现，并明确事件影响的方向、强度、持续时间和失效条件。

## 数据口径

### 事件定义

事件研究必须先明确以下要素：

- `event_date`：事件公开发生日期或首次被市场广泛知晓的日期
- `event_type`：财报、回购、减持、政策、事故、诉讼、问询、并购等
- `event_scope`：单只股票 / 行业 / 指数 / 多标的事件
- `benchmark`：沪深300、申万行业指数或同业组合

### 时间窗口与频率

默认采用以下窗口：

- **估计窗**：`[-120, -21]` 个交易日
- **事件窗**：`[-1, +1]`、`[-3, +3]`、`[-5, +5]`
- **后续漂移窗**：`[+6, +20]`、`[+21, +60]`

如因上市时间较短、停牌较多等原因无法满足估计窗，必须降级说明。

### 数据字段

优先需要：

- 收盘价、复权收盘价
- 日收益率
- 成交额、换手率
- 波动率
- 基准收益率
- 事件公告时间与属性

## 核心指标

### 指标列表与计算公式

#### 1. 原始收益与基准收益

- `R_i,t = P_t / P_{t-1} - 1`
- `R_m,t = benchmark_t / benchmark_{t-1} - 1`

#### 2. 异常收益（AR）

优先使用市场调整法或市场模型：

- 简化法：`AR_t = R_i,t - R_m,t`
- 若估计窗足够：`AR_t = R_i,t - (α + βR_m,t)`

#### 3. 累计异常收益（CAR）

- `CAR[a,b] = Σ AR_t`, `t ∈ [a,b]`

建议重点观察：

- `CAR[-1,+1]`
- `CAR[-3,+3]`
- `CAR[-5,+5]`
- `CAR[+6,+20]`

#### 4. 成交与波动反应

- `成交额放大倍数 = 事件窗平均成交额 / 估计窗平均成交额`
- `换手率放大倍数 = 事件窗平均换手率 / 估计窗平均换手率`
- `波动率变化 = 事件窗波动率 - 估计窗波动率`

#### 5. 漂移与回吐

- `post_event_drift = CAR[+6,+20]`
- `reversal = sign(CAR[-1,+1]) != sign(CAR[+6,+20])`

### 标准化与分级

- `|CAR[-1,+1]| >= 5%`：强反应
- `2% <= |CAR[-1,+1]| < 5%`：中等反应
- `|CAR[-1,+1]| < 2%`：弱反应
- 成交额放大倍数 `>= 2.0`：说明事件被充分交易
- 若 CAR 与成交/换手同步放大，结论可信度更高

## 信号与阈值

### 事件影响分级

#### 显著正面

满足 2 项及以上：

- `CAR[-1,+1] >= 3%`
- `CAR[-3,+3] >= 5%`
- 成交额放大倍数 `>= 1.5`
- 后续 20 日未明显回吐

#### 显著负面

满足 2 项及以上：

- `CAR[-1,+1] <= -3%`
- `CAR[-3,+3] <= -5%`
- 成交额放大倍数 `>= 1.5`
- 后续 20 日继续走弱

#### 预期已定价 / 影响有限

- `|CAR[-1,+1]| < 2%`
- 成交额与换手无明显放大
- 事件前已连续上涨或下跌

### 解除条件

- 后续 20~60 个交易日 CAR 回归零附近
- 成交额与波动率恢复常态
- 事件被新催化覆盖，原始事件解释力下降

### 无效化条件

以下情况应降低结论强度或改用定性描述：

- 事件窗内长期停牌
- 连续涨停/跌停导致价格无法自由出清
- 同期存在更大级别共振事件（政策、业绩、黑天鹅）
- 标的流动性极差，价格代表性不足

### 阈值选择依据

- A股个股日常噪声较高，`2%` 可视为弱事件门槛，`5%` 以上通常对应更强冲击
- 成交额放大倍数 `1.5~2.0` 用于区分是否发生显著交易拥挤
- `[-1,+1]` 用于观察即时冲击，`[-3,+3]` 用于观察提前交易与滞后反应，`[+6,+20]` 用于观察漂移

## 边界条件与降级策略

### 数据缺失/异常处理

- 公告日期不清：以首次披露日为准，并注明可能存在信息泄漏误差
- 停牌：以复牌首日为主要冲击观察点，但需单独标注
- 连续涨跌停：CAR 可计算，但不能直接代表可交易性收益

### 降级策略

- 无法估计 α、β：退回市场调整法 `AR = R_i - R_m`
- 无行业基准：使用沪深300作为广义基准
- 无完整成交数据：仅做价格冲击与方向判断，降低置信度

## A股特殊注意

1. 公告往往出现在收盘后，事件日应区分“披露日”与“首个完整交易日”。
2. T+1 与涨跌停制度会放大表面反应，需区分“账面反应”与“可执行反应”。
3. 问询函、监管函、重组预案、停牌复牌等事件常伴随信息不连续，需单列说明。 
4. 行业与风格共振很常见，必须做基准或同业调整，不能只看绝对涨跌。
