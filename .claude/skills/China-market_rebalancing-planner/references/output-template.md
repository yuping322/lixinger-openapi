# 输出模板：再平衡计划器

> 目标：给出目标权重、触发规则、执行优先级与复盘方法，形成可落地的再平衡方案。

## 再平衡计划报告

### 1) 结论摘要（3–5条）

- 当前再平衡触发原因为 `[权重漂移 / 风格偏离 / 风险超限 / 现金需求 / 基准变化]`。
- 建议的核心动作是 `[减仓 / 加仓 / 替换 / 保持观察]`。
- 调整后组合将更接近目标状态 `[更分散 / 更防守 / 更进攻 / 更贴近基准]`。
- 本次执行的主要约束是 `[税费 / 冲击成本 / 流动性 / 单笔限额 / 风险预算]`。
- 后续需要重点观察 `[执行偏差 / 再平衡后暴露 / 市场风格变化]`。

### 3) 分析与解释

#### 3.1 触发规则

- 说明是时间驱动、阈值驱动还是风险事件驱动。
- 触发原因是否一次性，还是长期需要制度化处理。
- 若存在多重触发条件，说明优先级排序。

#### 3.2 调整逻辑

- 哪些仓位需要回到中性权重，哪些是主动观点表达。
- 再平衡后行业、风格、因子暴露如何变化。
- 是否需要保留一定偏离，以避免过度交易。

#### 3.3 执行安排

- 一次性执行还是分批执行。
- 如何兼顾流动性、成本与风险控制。
- 对低流动性标的给出单独执行限制。

### 4) 风险清单与监控指标

| 风险 | 触发条件 | 监控指标 | 应对建议 |
|---|---|---|---|
| 过度交易 | 小幅偏离也频繁调整 | 周转率、成本 | 设置最小阈值 |
| 风格切换失误 | 调整后风格突变 | 风格指数、相对收益 | 分批执行 |
| 冲击成本过高 | 低流动性仓位较多 | 成交额、滑点 | 延长执行窗口 |
| 目标不清晰 | 基准或风险预算不明确 | 目标权重、约束表 | 先明确规则再执行 |

### 5) 下一步（需要补充/核验）

- 核对每项交易的税费、费用与最小成交单位。
- 对高优先级调整建立执行前后的暴露对比表。
- 在执行后做偏差复盘，确认是否达到再平衡目的。

### 6) 缺失数据与降级说明

- 缺失数据：真实交易成本、税务主体差异、账户限制规则。
- 替代口径：使用历史成本、成交额与公开费率近似估计。
- 对结论置信度影响：中。
