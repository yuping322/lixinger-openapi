# 方法论：机构调研热度“真假景气”验证

## 1. 目标定义

回答核心问题：
**高机构调研热度是否能够稳定转化为后续业绩改善与股价超额收益？**

## 2. 因子框架

使用四类输入因子：

1. `ResearchHeat`：机构调研热度
2. `IndustryNarrative`：行业研报景气叙事一致性
3. `EPSRevision`：盈利预测上修
4. `FlowConfirm`：资金流确认

建议构建综合评分：

`RealityScore = 0.30*ResearchHeat + 0.20*IndustryNarrative + 0.30*EPSRevision + 0.20*FlowConfirm`

可按市场阶段调整权重（例如风险偏好高时提高Flow权重）。

## 3. 指标建议

### 3.1 ResearchHeat（调研热度）

- 近30天调研次数分位
- 调研机构家数分位
- 调研频次环比加速度
- 行业内排名变化

### 3.2 IndustryNarrative（行业叙事）

- 行业看多/看空报告占比
- 关键词趋势：订单、价格、库存、产能利用率、政策催化
- 研报目标价上调比例

### 3.3 EPSRevision（盈利上修）

- FY1/FY2一致预期净利润近30/60天变动
- 上修分析师占比（扩散度）
- 上修是否伴随营收预测上修（防止仅靠利润率假设）

### 3.4 FlowConfirm（资金确认）

- 北向近20日净流入强度
- 主力大单近20日净流入
- 主题ETF份额变化（若可得）
- 融资余额变化（拥挤度辅助）

## 4. 分组回测与验证

## 4.1 分组方式

- 每月或每周重平衡
- 按`ResearchHeat`分为Q1~Q5
- 再在Q5内按`EPSRevision`和`FlowConfirm`二次分组

## 4.2 验证指标

- 财务兑现：下一季营收同比、利润同比、业绩超预期比例
- 市场兑现：T+30、T+90、T+180超额收益
- 稳定性：滚动12期胜率与信息比率

## 4.3 显著性判断

建议报告：

- Q5-Q1的收益差及t统计量
- “Q5且上修且资金流入”与“仅Q5”收益差
- 子行业/市值分层后的稳健性

## 5. 真假景气判定规则（可直接落地）

满足以下条件可判定为**真景气候选**：

1. ResearchHeat >= 行业内80分位
2. EPSRevision 为正且位于全市场前40%
3. FlowConfirm 为正（至少2个资金子指标为正）
4. 行业叙事不过度分歧（看多报告占比 > 55%）

若仅满足(1)且(2)(3)不成立，判定为**伪景气风险**。

## 6. 常见误区

1. 只看调研次数，不看调研对象质量
2. 把短期上涨误认为基本面验证
3. 忽略估值透支导致的“好公司、差交易”
4. 样本期过短导致统计不稳定

## 7. 结论表达建议

输出时至少包含：

- “是否有效”的总体结论（是/否/阶段性）
- 哪类股票最容易把调研热度转化为业绩与股价
- 哪类情景最容易失效
- 未来1个季度的跟踪触发器
