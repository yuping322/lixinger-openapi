# 高股东回报策略方法论

目标是识别“能持续、能成长、能复利、能再评级”的红利样本，而不是简单找最高股息率。

## 1. 硬筛选层

先排除明显不适合继续研究的公司：
- `ST` / 退市 / 上市时间过短
- 股息率很高但估值与风险极端失衡
- 周期高点利润支撑的虚高分红
- 借债分红、牺牲经营分红或资本开支压力极大的样本

可作为候选池基线的条件：
- 有基本股息率门槛
- 估值不过分极端
- 上市时间足够长
- 具备基础流动性

## 2. 分红质量

核心不是“分了多少”，而是“拿什么分”。

重点检查：
- 利润分红率是否健康
- 现金流口径是否能覆盖现金分红
- 负债与资本开支是否在侵蚀未来分红能力
- 是否出现借债分红或牺牲经营分红的迹象

## 3. 分红稳定性

重点区分：
- 连续分红、波动小、经营韧性强的稳定收息型
- 周期高点分红很高、但下行期容易削减的高位陷阱型

优先关注：
- 连续分红年数
- 分红是否大起大落
- 盈利是否稳定
- 行业是否天然防御

## 4. 股东回报增强

高质量红利机会不一定来自当前最高股息率，也可能来自未来提升空间。

重点寻找：
- 当前股息率一般，但派息率有提升空间
- 央国企改革、市值管理、特别分红、回购等线索
- 盈利稳增且分红政策逐步清晰的公司

### 4.1 `shareholder-yield-enhancer` 执行顺序

默认按以下顺序判断：
1. 先确认 `dyr` 是否具备基础吸引力，避免完全脱离红利主线
2. 再看 `annualNetProfitDividendRatio` 是否已有提升、或具备提升空间
3. 再补查经营现金流 / 自由现金流是否覆盖分红
4. 最后验证回购、特别分红、市值管理、央国企股东回报约束等增强线索

只有当“分红基础 + 现金流支撑 + 增强线索”至少形成两层以上共振时，才把样本上调为 `分红成长` 或 `央国企分红重估`，否则只保留在观察名单。

### 4.2 `soe-rerating` 的分红重估面

本策略只承接以下部分：
- 派息率提升
- 分红政策改善
- 特别分红、回购、市值管理预期
- 央国企股东回报约束增强

不在本策略内直接断言治理改善已经成立。凡涉及：
- 股东结构改善
- 监管 / 问询 / 审计风险缓和
- 资产整合、激励约束、治理优化

都必须转交 `esg-screener` 做公开资料验证；没有公告或公开资料时，只能写成待验证线索。

## 5. 现金牛复利

现金牛样本的关键不是高息，而是持续创造现金并把现金用在对股东更友好的地方。

重点寻找：
- 经营现金流与自由现金流长期较稳
- 资本配置纪律较好，不过度冲动扩张
- 盈利兑现、分红与再投资之间平衡较优
- 被市场当成低波资产，但长期回报结构优于表面认知

### 5.1 `cash-cow-compounder` 执行顺序

默认按以下顺序判断：
1. 先看经营现金流与自由现金流是否持续稳定
2. 再看资本开支、分红、回购之间是否存在可持续平衡
3. 再看盈利兑现质量是否和现金创造能力匹配
4. 最后判断市场是否仍按“稳定防御资产”而非“长期复利资产”定价

如果需要解释为什么总回报结构更优，再追加 `quant-factor-screener` 补充 `价值 + 质量 + 成长 + 股东回报` 的共振说明；但最终分类仍回到本策略的 `现金牛复利` 或 `分红成长`。

## 6. 总回报框架

红利判断统一放在总回报里看：
- 股息收入
- 估值修复
- 业绩稳增
- 股东回报增强

将样本区分为：
- `稳定收息`
- `分红成长`
- `现金牛复利`
- `央国企分红重估`
- `高股息陷阱预警`

## 7. 异常发现层

每次执行必须写出异常信号，例如：
- 股息率高，但利润分红率和现金流覆盖同时转弱
- 当前股息率一般，但自由现金流与资本配置质量明显更强
- 央国企标的出现提高派息率或市值管理预期，市场还未充分计价
- 周期股股息率很高，但来自商品高点利润，不具持续性
- 经营稳定、现金流扎实，但长期被忽视为“无聊资产”

## 8. 跟踪信号模板

对重点候选至少跟踪以下 2-3 项：
- 下一次分红方案或派息率变化
- `annualNetProfitDividendRatio` 是否继续抬升或走稳
- 资产负债表是否恶化
- 经营现金流 / 自由现金流覆盖是否继续安全
- 回购、特别分红、市值管理是否落地
- 资本开支是否开始侵蚀分红与回购能力
