# 多因子策略方法论

目标是找出因子共振机会，并识别因子失效与因子冲突，而不是简单做一个综合分排行榜。

## 1. 因子设计原则

只保留当前仓库内较容易验证和复现的因子。

### 价值因子

优先使用：
- `d_pe_ttm`
- `pb_wo_gw`
- `pcf_ttm`
- `ev_ebitda_r`

用途：识别定价是否便宜。

### 质量因子

优先使用：
- `q.ps.wroe.t`
- `q.bs.tl.t` / `q.bs.ta.t`

用途：识别盈利能力与杠杆质量。

### 成长因子

优先使用：
- `q.ps.toi.t_y2y`
- `q.ps.np.t_y2y`
- `q.ps.gp_m.t`

用途：识别增长与盈利改善是否同步。

### 股东回报增强项

优先使用：
- `dyr`
- `annualNetProfitDividendRatio`
- `pcf_ttm`

用途：作为高共识主线的增强解释，不单独替代价值、质量、成长主因子。

### 动量与低波因子

基于 `candlestick` 价格序列计算：
- 中期价格趋势
- 回撤与波动水平
- 与基准的强弱关系

### 规模因子

优先使用：
- `mc`

用途：区分大市值稳健样本与小市值弹性样本。

## 2. 因子计算思路

### 行业内比较优先

默认优先在行业内比较，避免把行业偏好误写成因子优势。

### 先做单因子，再看组合

每个样本先判断单因子强弱，再判断：
- 是否出现 `价值 + 质量` 共振
- 是否出现 `质量 + 成长` 共振
- 是否出现 `价值 + 质量 + 成长` 三重共振
- 是否出现 `股东回报 + 质量` 增强
- 是否出现 `价值 + 动量` 冲突
- 是否出现 `动量强但基本面弱` 的高风险样本

### 因子分数只服务解释

分数是排序工具，不是最终结论。最终结论必须回到：
- 因子为何共振
- 冲突出现在哪里
- 当前风格是否支持这种暴露
- 它更像底仓型机会还是进攻型卫星

## 3. 异常发现层

每次执行必须主动指出异常，例如：
- 综合分一般，但 `价值 + 质量` 同时极强
- 动量领先，但成长和质量并未跟上
- 质量很好，但估值已经不再便宜
- 低波与高成长同时出现，说明防御与弹性兼具
- 股东回报一般，但价值、质量、成长三项共振更强

## 4. 机会分类层

### `高共振候选`

两类以上核心因子同时表现强，且互相不冲突。

### `风格受益候选`

当前市场环境明显偏好该类因子，短期得分受益于风格。

### `低估待启动候选`

价值、质量较好，但价格趋势尚未启动，适合观察。

### `因子冲突高风险候选`

某一因子特别强，但另一关键因子明显拖后腿，容易出现高分陷阱。

## 5. 宏观与风格解释

宏观解释只做轻量辅助：
- 利率环境偏低时，质量与红利更容易获得偏好
- 风险偏好提升时，成长、小盘、动量更容易受益
- 风格切换期，高动量样本回撤风险更大

不要把宏观解释写成确定性的择时预测。

## 6. 跟踪信号模板

对重点候选至少跟踪：
- 价值因子是否因估值修复而削弱
- 质量与成长是否继续兑现
- 股东回报增强项是否继续改善
- 价格趋势是否从待启动转为确认
- 利率与市场风格是否明显切换
