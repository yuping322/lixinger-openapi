# 输出模板：涨跌停与可交易性风险检查器

> 目标：评估涨跌停、停牌等制度约束对可交易性的影响，并明确风险标签与行动边界。

## 涨跌停与可交易性风险检查报告

### 1) 结论摘要（3–5条）

- 当前目标标的的可交易性状态为 `[正常 / 偏紧 / 高风险 / 极低可交易性]`。
- 主要风险来自 `[连续涨停买不到 / 连续跌停卖不出 / 停牌 / 封单过大 / 流动性差]`。
- 结合成交与封单结构，当前制度性约束对交易执行的影响为 `[轻微 / 明显 / 极强]`。
- 若未来出现 `[开板 / 放量换手 / 停牌复牌 / 监管干预]`，风险状态可能改变。
- 后续应重点监控 `[封单额、换手率、停牌状态、跌停打开次数、盘口深度]`。

### 3) 分析与解释

#### 3.1 制度约束类型

- 属于涨停买入困难、跌停卖出困难，还是停牌完全不可交易。
- 是情绪性封板还是基本面/公告驱动的持续性板。
- 在不同板块规则下，涨跌幅限制与风险边界需明确说明。

#### 3.2 可交易性评估

- 封单额与换手率共同决定次日是否可能开板。
- 连续跌停叠加低流动性时，应提高“无法退出”风险标签。
- 停牌风险要单独说明触发原因与预计持续时间。

#### 3.3 行动建议边界

- 对于研究结论，区分“逻辑正确”与“执行不可行”。
- 不应把无法交易的标的当作标准仓位管理对象。
- 可根据开板、换手恢复、停牌解除等条件重新评估。

### 4) 风险清单与监控指标

| 风险 | 触发条件 | 监控指标 | 应对建议 |
|---|---|---|---|
| 买不到 | 连续涨停封单高 | 封单额、换手率 | 不追高，等待换手 |
| 卖不出 | 连续跌停且缩量 | 跌停封单、成交额 | 提前识别流动性风险 |
| 停牌不确定 | 重大事项停牌 | 停牌公告、复牌安排 | 独立列示事件风险 |
| 误判可交易性 | 板上有量但承接弱 | 开板次数、盘口 | 结合分时复核 |
