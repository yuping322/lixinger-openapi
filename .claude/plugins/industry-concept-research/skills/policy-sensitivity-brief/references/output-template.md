# 输出模板：政策敏感度简报生成器

> 目标：把政策事件映射到行业、风格与标的敏感度，并给出情景推演与跟踪清单。

## 政策敏感度简报

### 1) 结论摘要（3–5条）

- 本次政策/宏观事件的市场属性为 `[增量利好 / 中性偏多 / 中性偏空 / 风险事件]`。
- 受影响最大的方向是 `[行业 / 风格 / 资产类别 / 主题]`。
- 主要传导路径为 `[需求拉动 / 成本变化 / 利率折现 / 风险偏好 / 监管约束]`。
- 当前市场是否已提前定价该政策变化：`[已反映 / 部分反映 / 尚未充分反映]`。
- 后续最关键的验证变量为 `[实施细则 / 数据兑现 / 资金反应 / 板块相对收益]`。

### 3) 分析与解释

#### 3.1 事件拆解

- 政策变化是方向性调整、节奏性微调还是执行性细则。
- 政策影响是短期情绪冲击还是中期盈利/估值重定价。
- 明确哪些内容是市场已知，哪些是新信息增量。

#### 3.2 传导路径

- 从政策到需求、价格、成本、融资、估值的传导链条是什么。
- 哪些行业处于最前端，哪些行业存在次级或滞后影响。
- 哪些风格（成长/价值/红利/周期）更敏感。

#### 3.3 情景推演

- 基准情景：政策按市场预期落地。
- 乐观情景：细则强化、执行超预期、数据快速验证。
- 悲观情景：落地不及预期、执行滞后或被其他变量对冲。

### 4) 风险清单与监控指标

| 风险 | 触发条件 | 监控指标 | 应对建议 |
|---|---|---|---|
| 政策误读 | 标题利好但细则有限 | 正文条款、实施细则 | 以条款为准复核 |
| 市场已充分定价 | 板块提前大涨 | 相对收益、估值分位 | 防止追高解读 |
| 兑现不足 | 数据未跟进 | 月度数据、业绩 | 下调敏感度判断 |
| 传导链断裂 | 受益环节未兑现 | 价格、订单、资金流 | 重新校准链条 |

### 5) 下一步（需要补充/核验）

- 建立“政策—行业—标的”映射清单。
- 对高敏感方向补做历史同类政策复盘。
- 若需交易视角，联动资金流、估值和情绪指标确认。

### 6) 缺失数据与降级说明

- 缺失数据：执行细则、地方配套、企业微观反馈。
- 替代口径：使用政策原文、历史案例、行业公开数据与市场反应。
- 对结论置信度影响：中。
