## 1. 项目概述

本项目是一个基于理杏仁开放平台 API 的专业金融分析技能包，提供 116 个开箱即用的金融分析技能，覆盖 A股、港股、美股三大市场。

### 核心特性

- **116 个专业分析技能**：66个A股 + 13个港股 + 37个美股
- **1 个数据查询工具**：支持 162 个理杏仁 API 接口
- **开箱即用**：无需虚拟环境，无需安装依赖
- **专业方法论**：每个技能包含完整的分析方法论和输出模板

## 3. 技能分类

### A股市场技能（66个）

- **基础分析类**（3个）：财务报表、同业对标、个股研究
- **风险监控类**（3个）：股权质押、股东风险、商誉风险
- **资金流向类**（3个）：市场资金流、北向资金、沪深港通
- **市场分析类**（6个）：市场概览、宽度监控、波动率、估值
- **事件驱动类**（6个）：龙虎榜、大宗交易、公告、内部人交易
- **估值选股类**（7个）：低估筛选、高股息、量化因子
- **组合管理类**（5个）：组合检查、再平衡、风险优化
- **行业板块类**（4个）：行业分析、轮动检测、产业链
- **特殊工具类**（29个）：可转债、IPO、ETF等

### 港股市场技能（13个）

- 市场分析类（4个）
- 资金流向类（3个）
- 风险监控类（3个）
- 基础分析类（3个）

### 美股市场技能（37个）

- 基础分析类（3个）
- 市场分析类（5个）
- 宏观利率类（3个）
- 事件驱动类（5个）
- 估值选股类（7个）
- 组合管理类（8个）
- 特殊工具类（6个）

## 5. 数据查询工具

### query_tool.py 特性

```bash
python3 skills/lixinger-data-query/scripts/query_tool.py \
  --suffix "cn.company" \
  --params '{"stockCodes": ["600519"]}' \
  --columns "stockCode,name,listDate"
```

**核心功能**：
- 162 个 API 接口
- 字段过滤（--columns）
- 数据筛选（--row-filter）
- 数组展开（--flatten）
- CSV 格式输出
- 独立运行，无需依赖

## 6. 使用工作流

```
1. 用户提出分析需求
   ↓
2. 识别需求类型（A股/港股/美股）
   ↓
3. 选择合适的分析技能
   ↓
4. 查看 SKILL.md 了解工作流程
   ↓
5. 查看 data-queries.md 了解数据需求
   ↓
6. 使用 query_tool.py 获取数据
   ↓
7. 按照 methodology.md 进行分析
   ↓
8. 按照 output-template.md 输出结果
```

## 7. 性能优化

### 数据查询优化

1. **字段过滤**：使用 `--columns` 只返回需要的字段
2. **数据限制**：使用 `--limit` 限制返回行数
3. **CSV 格式**：紧凑格式，节省 30-40% token

### 缓存策略建议

- 实时数据：缓存 1 小时
- 日线数据：缓存 24 小时
- 财务数据：缓存 7 天
- 宏观数据：缓存 24 小时
