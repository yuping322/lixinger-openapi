## 🚀 具体可用的命令示例

### 实时市场查询
```bash
# 市场概览
toolkit-cn --market --mode brief

# 查询特定股票
toolkit-cn --stock 600519 --mode full

# 板块分析
toolkit-cn --sector 食品饮料
```

### 数据筛选
```bash
# 筛选低估值股票
toolkit-cn --screen 'pe_ttm < 15 AND roe > 10'

# 筛选高成长股票
toolkit-cn --screen 'revenue_growth_yoy > 20'

# 龙虎榜股票筛选
toolkit-cn --screen 'lhb_date = today()'
```

### 龙虎榜专项分析
```bash
# 获取当日龙虎榜数据
python -c "
import akshare as ak
lhb_data = ak.stock_lhb_detail_em()
print(lhb_data.head(10))
"
```
